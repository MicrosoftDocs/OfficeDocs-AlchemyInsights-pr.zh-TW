# <a name="learn-how-to-configure-agenda-settings"></a>了解如何設定議程設定

議程設定位於網路上的 outlook**行事曆**設定。如需詳細資訊，請參閱[在網路上的 Outlook 行事曆設定](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f)。
  
系統管理員可以在 Exchange Online PowerShell 中使用**Set-calendarnotification**指令程式來設定議程設定。如需詳細資訊，請參閱[Set-calendarnotification](https://technet.microsoft.com/library/dd351284)。
