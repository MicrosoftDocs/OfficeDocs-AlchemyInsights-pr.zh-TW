# <a name="email-aliases-on-mobile-devices"></a>在行動裝置上的電子郵件別名

電子郵件別名為當成另一個信箱上設定電子郵件地址，讓郵件傳送給使用者的別名會傳遞至其信箱並將會在使用者的行動裝置上可讀取。需要進行其他設定。
  
但是，使用者無法回覆郵件或使用別名為新郵件傳送其從地址。所有外寄郵件會從其主要電子郵件地址。
