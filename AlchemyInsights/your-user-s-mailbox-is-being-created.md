# <a name="your-users-mailbox-is-being-created"></a>建立使用者信箱

可能很多達之前新的信箱已經準備好使用 24 小時。
  
如果使用者的信箱已 「 準備 」 超過一小時，嘗試移除使用者的 Exchange 授權，並重新新增其備份，或使用"我們呼叫您"] 選項來支援工程師與對話。
