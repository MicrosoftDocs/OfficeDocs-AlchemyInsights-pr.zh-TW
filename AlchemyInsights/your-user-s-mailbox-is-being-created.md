# <a name="user-mailbox-creation-is-taking-a-long-time"></a>建立使用者信箱花費的時間很長

最多可能需要24小時的時間，新信箱才可供使用。
  
如果使用者的信箱已 "準備" 超過一小時，請嘗試移除使用者的 Exchange 授權，然後重新加入，或[聯繫支援人員](https://go.microsoft.com/fwlink/p/?linkid=518322)。
