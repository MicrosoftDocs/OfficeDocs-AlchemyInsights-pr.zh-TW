# <a name="your-users-mailbox-is-being-created"></a>正在建立您的使用者信箱

最多可能需要24小時才能使用新的信箱。
  
如果使用者的信箱已 "準備" 超過一小時, 請嘗試將 Exchange 授權從使用者中移除並重新加入, 或使用「讓我們呼叫您」選項, 與支援工程師交談。
