# <a name="your-users-mailbox-is-being-created"></a>要建立您的使用者信箱

可能需要最多 24 小時才可供使用新的信箱。
  
如果使用者的信箱已被 「 準備 」 超過一小時，請嘗試從使用者移除 Exchange 授權，重新新增備份，或連絡技術支援工程師使用 「 讓我們撥話給您 」 選項。
