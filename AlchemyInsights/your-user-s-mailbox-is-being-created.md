# <a name="user-mailbox-creation-is-taking-a-long-time"></a>建立使用者信箱的時間較長的時間

可能需要最多 24 小時才可供使用新的信箱。
  
如果使用者的信箱已超過一小時的 「 準備 」，請嘗試移除使用者和重新新增回原處，或[連絡技術支援](https://docs.microsoft.com/office365/admin/contact-support-for-business-products?tabs=online)的 Exchange 授權。
