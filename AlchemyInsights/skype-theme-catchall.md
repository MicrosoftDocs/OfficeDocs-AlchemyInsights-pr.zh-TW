# <a name="skype-for-business-common-issues-and-resolutions"></a>商務用 Skype 常見問題與解決方案 

如需有關商務用 Skype 的特定問題，請簡短地重新表述您的問題以描述您的問題，包括錯誤、您所使用的功能等等。 

以下是一些經常要求的主題：

**商務用 Skype 登錄疑難排解：** 首先，請確定 [商務用 Skype 服務](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/servicehealth) 狀況良好。 若是，請查看 [疑難排解商務用 Skype Online 登入錯誤](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/troubleshooting-sign-in-errors-for-admins#check-for-common-causes-of-skype-for-business-online-sign-in-errors) ，以避免登入困難的常見原因。
 
**在商務用 Skype 中疑難排解音訊和影片：** 回顧 [商務用 Skype 中常見的音訊和影片問題的解決方案](https://support.office.com/article/Troubleshoot-audio-and-video-in-Skype-for-Business-62777bc6-c52b-47ae-84ba-a8905c3b71dc)。 

**針對商務用 Skype 中的連線問題進行疑難排解：** 回顧 [商務用 Skype 中常見連線問題的解決方案](https://support.office.com/article/troubleshoot-connection-issues-in-skype-for-business-ca302828-783f-425c-bbe2-356348583771)。

**與商務用 Skype 連絡人進行通訊 (同盟) ：** [設定商務用 skype 以聯繫外部商務用 skype 使用者](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users)。

**與 Skype 連絡人通訊：** [讓商務用 Skype 使用者新增 Skype 連絡人](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts)。

**將商務用 Skype 升級至 Microsoft 小組：** 請參閱 [Microsoft 團隊的快速入門升級](https://docs.microsoft.com/microsoftteams/upgrade-start-here) ，並深入瞭解在您採用組織中的團隊時，與 microsoft 小組 [共存](https://docs.microsoft.com/microsoftteams/coexistence-chat-calls-presence) 的相關資訊。
