# <a name="teams-dial-pad-configuration"></a>Teams 撥號鍵台設定

在 Teams 用戶端中，撥號鍵台可讓使用者存取公用交換電話網路（PSTN）功能。  

如需詳細資訊，請移至，[撥號鍵台設定](https://docs.microsoft.com/microsoftteams/dial-pad-configuration)。
