# <a name="enable-versioning-for-a-sharepoint-list-or-library"></a>啟用 SharePoint 清單或文件庫的版本設定


當您的 SharePoint 清單或文件庫中已啟用版本設定時，您可以儲存、 追蹤及每當他們變更還原清單中的項目和文件庫中的檔案。 版本設定、 結合 [其他設定，例如簽出，為您提供了許多之內容的網站上張貼及可以提供實際值，如果您曾經需要查看或還原項目或檔案的舊版本控制。

如需版本設定的詳細資訊，請造訪以下文章。

[版本設定如何在 SharePoint 清單或文件庫中運作](https://support.office.com/article/how-does-versioning-work-in-a-sharepoint-list-or-library-0f6cd105-974f-44a4-aadb-43ac5bdfd247)

[還原舊版本的 OneDrive 中的檔案](https://support.office.com/article/restore-a-previous-version-of-a-file-in-onedrive-159cad6d-d76e-4981-88ef-de6e96c93893?ui=en-US&amp;rs=en-US&amp;ad=US)

[檢視舊版 Office 檔案](https://support.office.com/article/view-previous-versions-of-office-files-5c1e076f-a9c9-41b8-8ace-f77b9642e2c2)
