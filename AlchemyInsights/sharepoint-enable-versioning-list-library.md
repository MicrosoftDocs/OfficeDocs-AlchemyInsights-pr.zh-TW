# <a name="enable-versioning-for-a-sharepoint-list-or-library"></a>啟用 SharePoint 清單或文件庫的版本設定


當您的 SharePoint 清單或文件庫中啟用版本設定時, 您可以在清單中儲存、追蹤及還原專案, 並在每次變更時儲存文件庫中的檔案。 版本設定, 與其他設定 (例如結帳) 相結合, 可讓您對網站上張貼的內容進行大量控制, 如果您需要查看或還原舊版本的專案或檔案, 就能提供實際值。

如需版本設定的詳細資訊, 請造訪下列文章。

- [版本設定在 SharePoint 清單或文件庫中的運作方式](https://support.office.com/article/how-does-versioning-work-in-a-sharepoint-list-or-library-0f6cd105-974f-44a4-aadb-43ac5bdfd247)

- [啟用及設定清單或文件庫的版本設定](https://support.office.com/article/enable-and-configure-versioning-for-a-list-or-library-1555d642-23ee-446a-990a-bcab618c7a37?ocmsassetID=HA102772148&amp;CTT=3&amp;CorrelationId=52441bb1-a619-4375-89d5-19d28769890f&amp;ui=en-US&amp;rs=en-US&amp;ad=US)

- [如何查看版本歷程記錄](https://support.office.com/article/View-the-version-history-of-an-item-or-file-in-a-list-or-library-53262060-5092-424D-A50B-C798B0EC32B1)

- [在 OneDrive 中還原舊版本的檔案](https://support.office.com/article/restore-a-previous-version-of-a-file-in-onedrive-159cad6d-d76e-4981-88ef-de6e96c93893?ui=en-US&amp;rs=en-US&amp;ad=US)

- [檢視舊版 Office 檔案](https://support.office.com/article/view-previous-versions-of-office-files-5c1e076f-a9c9-41b8-8ace-f77b9642e2c2)

- [版本限制](https://docs.microsoft.com/office365/servicedescriptions/sharepoint-online-service-description/sharepoint-online-limits)

**附注:** 如果您是 Office 365 客戶, 則在建立新的商務用 OneDrive 文件庫時, 預設會啟用版本設定, 而且會自動儲存檔的最後一個500版本。 這可協助您防止遺失重要的檔或資料。 如果您的商務用 OneDrive 網站或小組網站上的現有文件庫未啟用版本設定, 您可以隨時開啟版本設定。
