# <a name="change-your-microsoft-account-picture"></a>變更您的 Microsoft 帳戶圖片

1. 移至 [您的 Microsoft 帳戶資訊](https://go.microsoft.com/fwlink/p/?linkid=860841)。
2. 選取 [ **變更圖片**]。
3. 選取 [ **新增圖片**]。
4. 選擇您要使用的圖片，然後選取 [ **開啟**]。
5. 拖動或重設圖像框，然後選取 [ **儲存**]。

請允許最多24小時的設定檔相片的變更才會生效。
