# <a name="outlook-on-the-web-account-hacked"></a>網頁帳戶上的 Outlook 遭受駭客攻擊

如果您的帳戶受到攻擊，請依照 [如何保護和還原可疑的 Microsoft 365 帳戶和信箱中的電子郵件功能](https://docs.microsoft.com/microsoft-365/security/office-365-security/responding-to-a-compromised-email-account)的指示進行。

若要深入瞭解您可以採取的其他步驟，以防止您的帳戶遭到洩漏，請參閱 [在 Microsoft 365 中回應遭到破壞的電子郵件帳戶](https://docs.microsoft.com/microsoft-365/security/office-365-security/responding-to-a-compromised-email-account)。
