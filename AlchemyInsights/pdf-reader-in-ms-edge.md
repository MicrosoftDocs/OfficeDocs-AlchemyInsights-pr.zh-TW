# <a name="pdf-reader-in-microsoft-edge"></a>Microsoft Edge 中的 PDF 讀取器

PDF 檔案的普及程度，對企業公司可以採用的可靠、安全且功能強大的 PDF 讀取器有重要意義。 Microsoft Edge 內建的多功能 PDF 讀取器 (版本77和更新版本) 就是這類應用程式。 它可以在 Windows 和 macOS 中運作，也可以用來開啟和批註以本機、線上或內嵌于網頁中的 PDF 檔案。

若要深入瞭解，請參閱 [在 Microsoft Edge 中的 PDF 讀取器](https://go.microsoft.com/fwlink/?linkid=2140005)。
