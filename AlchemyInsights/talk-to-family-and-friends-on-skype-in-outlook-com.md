# <a name="talk-to-family-and-friends-on-skype-in-outlookcom"></a>在 Outlook.com 上與家人和朋友交談

Outlook.com 客戶可以透過 Skype 中的聊天、語音和影片功能與朋友和家人互動。 當您 [合併您的 Microsoft 與 Skype 帳戶](https://go.microsoft.com/fwlink/p/?linkid=2001101&amp;clcid=0x409)時，您會在 Outlook.com 中自動存取網頁上的 Skype。
  
深入瞭解如何 [在 Outlook.com 的 Skype 上交談至家人和朋友](https://go.microsoft.com/fwlink/p/?linkid=2001407&amp;clcid=0x409)。
