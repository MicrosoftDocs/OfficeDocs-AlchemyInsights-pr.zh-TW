# <a name="talk-to-family-and-friends-on-skype-in-outlookcom"></a>在 Outlook.com 中用 skype 交談系列和朋友

Outlook.com 客戶可以與朋友和家人與交談、 語音及視訊功能 skype 交談。 當您[合併您的 Microsoft 和 Skype 帳戶](https://go.microsoft.com/fwlink/p/?linkid=2001101&amp;clcid=0x409)，您自動上要能存取用 skype 在 Outlook.com。
  
深入了解如何[洽詢系列和朋友用 skype 在 Outlook.com 中](https://go.microsoft.com/fwlink/p/?linkid=2001407&amp;clcid=0x409)。
