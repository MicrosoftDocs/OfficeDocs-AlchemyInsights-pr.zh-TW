# <a name="create-a-new-folder"></a>建立新的資料夾

1. 在功能窗格的 [**資料夾**] 下, 選取 [**新增資料夾**]。

2. 輸入資料夾的名稱, 然後按 Enter 鍵。

若要深入瞭解, 請參閱[在 Outlook.com 中建立資料夾](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a)。
