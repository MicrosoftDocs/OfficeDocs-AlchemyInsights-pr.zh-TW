# <a name="create-a-new-folder"></a>建立新的資料夾

1. 在 [功能窗格中的**資料夾**，選取**新的資料夾**。 
    
2. 輸入該資料夾的名稱，然後按 Enter。
    
若要深入了解，請參閱 < <b0>Create Outlook.com 中的資料夾</b0>。
