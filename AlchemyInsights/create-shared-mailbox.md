# <a name="create-a-shared-mailbox"></a>建立共用信箱

共用的信箱可讓多人傳送及接收電子郵件別名，例如 sales@contoso.com，卻沒有指定另一個授權。 

請參閱[建立共用信箱](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)的秘訣與使用共用的信箱的詳細資訊。
