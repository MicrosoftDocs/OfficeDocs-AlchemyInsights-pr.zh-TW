# <a name="add-seats-to-a-prepaid-subscription"></a>新增基座至預付訂閱

1. 移至系統管理中心中的 [[訂閱]](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面，或選擇 **[計費]** \> **[訂閱]**。
    
2. 選擇您想要新增使用者的預付的訂閱。
    
3. 在 [**更多動作**] 功能表中，選擇 [**續約或新增授權權杖**。
    
4. 選取 [**新的及未使用的產品金鑰**] 選項。 （如果此訂閱已建立的產品金鑰購買零售位置，您可能會有 [若要使用其他付款方式新增使用者] 選項）。 
    
5. 選擇 [下一步]****。
    
6. 輸入產品金鑰，然後選擇 [**下一步**]。
    
7. 選擇 [新增更多使用者，選擇 [**下一步**，然後選擇 [**兌換**的選項。
