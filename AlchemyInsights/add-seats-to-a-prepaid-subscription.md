# <a name="add-seats-to-a-prepaid-subscription"></a>將座位新增至預付訂閱

1. 在系統管理中心中，移至 [訂閱][](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面，或選擇 [計費]**** \> [訂閱]****。

2. 選擇您想要新增使用者的預付訂閱。

3. 在 [**其他動作**] 功能表中, 選擇 [**以權杖更新] 或 [新增授權**]。

4. 選取 [**新增和未使用的產品金鑰**] 選項。 (如果此訂閱是使用在零售位置購買的產品金鑰來建立, 您可以選擇使用其他付款條件來新增使用者。)

5. 選擇 [下一步]****。

6. 輸入產品金鑰, 然後選擇 [**下一步]**。

7. 選擇 [新增更多使用者] 選項, 選擇 [**下一步]**, 然後選擇 [**兌換**]。
