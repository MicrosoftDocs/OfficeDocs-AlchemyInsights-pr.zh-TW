# <a name="create-users-in-hybrid-deployments"></a>在混合式部署中建立使用者

當您的組織是內部部署環境與 Office 365 之間的混合式部署時, 您必須在內部部署 Active Directory 中建立所有使用者物件。 您可以在適當的組織單位 (OU) 中建立使用者物件, 以同步處理至您的 Office 365 組織。
