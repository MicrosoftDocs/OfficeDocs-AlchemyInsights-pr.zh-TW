# <a name="create-users-in-hybrid-deployments"></a>混合式部署中建立使用者

當您的組織內部部署環境與 Office 365 之間的混合式部署時，您需要在內部部署 Active Directory 中建立使用者的所有物件。 讓他們正在同步處理至 Office 365 組織，您可以建立適當的組織單位 (OU) 中的使用者物件。
