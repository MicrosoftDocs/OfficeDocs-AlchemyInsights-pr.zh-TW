# <a name="cant-select-contacts-when-composing-email"></a>撰寫電子郵件時無法選取連絡人

撰寫新的電子郵件時, 有些客戶按一下 [**收件者**] 或 [**抄送**] 按鈕時, 看不到他們的連絡人清單。 若要選取連絡人, 請在 [**收件者**] 或**** [副本] 欄位中開始輸入, 並顯示您的相關連絡人。
