# <a name="cant-select-contacts-when-composing-email"></a>撰寫電子郵件時，不能選取連絡人

當撰寫新電子郵件並按一下 [**到**] 或 [**副本**] 按鈕，有些客戶可能不會看到其連絡人清單中的。 若要選取 [名稱]，開始輸入**收**件者或 **[副本]** 欄位中，並會顯示您相關的連絡人。
