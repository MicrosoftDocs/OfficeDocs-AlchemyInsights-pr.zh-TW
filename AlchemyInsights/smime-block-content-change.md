# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook 現在會預設會封鎖 S/MIME 中的外部內容

我們將在 Outlook for Windows 中引入新增的安全性度量，方法是在預設會封鎖 S/MIME 加密和簽署的郵件中的外部內容，避免以每個郵件為基礎的覆寫。

如需完整的詳細資訊，請參閱[Outlook For Windows 封鎖 S/MIME 郵件中的外部內容](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e)。
