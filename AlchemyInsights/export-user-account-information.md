# <a name="export-user-account-information"></a>匯出使用者帳戶資訊

若要匯出使用者帳戶資訊，在 Microsoft 365 系統管理中心中，包括有關授權、 密碼原則、 封鎖的帳戶、 AAD 同步處理的狀態，以及更多]，移至 [**使用者** > [作用中使用者](https://go.microsoft.com/fwlink/p/?linkid=834822)頁面，然後選取 [**匯出使用者**。
