# <a name="when-is-the-billing-statement-generated"></a>帳單報表產生的時間

每月您都會收到一封電子郵件, 告訴您您的新帳單聲明可在 Microsoft 365 系統管理中心中取得。 您的發票包含兩個頁面。 第1頁是發票摘要, 包含有關發票、訂單、到期金額、如何進行付款的一般資訊, 以及如何與支援人員聯繫。 第2頁包含每個訂閱的帳單活動詳細資料。
  
若要深入瞭解帳單概念, 請參閱[瞭解您的發票](https://support.office.com/article/0724b428-fb59-4962-8c37-6674166d7507)。
