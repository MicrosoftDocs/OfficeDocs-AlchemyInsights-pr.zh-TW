# <a name="disable-excel-live-preview"></a>停用 Excel 即時預覽

如果您會說: 「 沒有問題剪貼簿，但您仍可以貼上您在此活頁簿內的內容 」 的 Excel 中收到錯誤，請停用的即時預覽功能遵循下列步驟：

1. 在 Excel 中，按一下 [**檔案** > **選項**。
3. 在 [ **Excel 選項**] 對話方塊中，選取 [**一般**]。
4. 在**使用者介面選項**，清除 [**啟用即時預覽**。
5. 按一下 [確定]****。
6. 重新啟動 Excel。
