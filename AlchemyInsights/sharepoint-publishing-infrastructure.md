# <a name="sharepoint-publishing-infrastructure"></a>SharePoint 發佈基礎結構


SharePoint 包括一組傳統的功能，可讓您製作及發佈內部網路的豐富網頁。 啟動網站集合層級的 SharePoint 發佈基礎結構，以將其開啟。

如需詳細資訊，請造訪下列連結。

[啟用發佈功能](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[發佈功能](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

附註：

- 傳統發佈網站範本預設會包含發佈功能。

- 在新式網站上無法使用傳統發佈功能。 新式網站預設會提供較新的替代方式。
