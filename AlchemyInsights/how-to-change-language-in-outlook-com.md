# <a name="change-your-language"></a>變更您的語言

1. 在右上角，選擇 [**設定** \> **選項**。
    
2. 選取 [**選項**] 下的 [**一般** \> **區域和時區**。
    
3. 在 [**地區及時間的時區設定**] 中，選擇您的語言。
    
4. 選取**儲存**。
    
若要深入了解，請參閱[變更語言設定的 Outlook.com 和時間的時區](https://go.microsoft.com/fwlink/p/?linkid=873132)。
