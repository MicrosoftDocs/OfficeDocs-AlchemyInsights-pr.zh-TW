# <a name="ndr-with-error-code-554-522-mailbox-full"></a>NDR，錯誤代碼 554 5.2.2 信箱已滿

如果您收到 NDR，錯誤碼為 "554 5.2.2 信箱已滿"，當您將電子郵件傳送至擁有郵件功能的公用資料夾時：  

請使用 [本文中的](https://aka.ms/554522) 步驟，確定公用資料夾或公用資料夾信箱配額尚未到期。
