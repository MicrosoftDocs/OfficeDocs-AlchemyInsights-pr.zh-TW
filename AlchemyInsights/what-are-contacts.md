# <a name="what-are-contacts"></a>什麼是連絡人？

您可以透過多種方式在 Microsoft 365 中使用連絡人，其中大部分取決於您使用連絡人的方式。

- 若要深入了解 Microsoft 365 中的不同連絡人，請參閱[快速說明：管理連絡人的方法](https://docs.microsoft.com/microsoft-365/admin/misc/ways-to-manage-contacts?view=o365-worldwide)。

- 若要開始在 Outlook 中使用連絡人，請參閱[開始在 Outlook 中使用連絡人](https://support.office.com/article/using-contacts-people-in-outlook-on-the-web-1e3438c7-26b2-420c-87de-3cea9d31b5cb?WT.mc_id=365AdminCSH&ui=en-US&rs=en-US&ad=US)。
