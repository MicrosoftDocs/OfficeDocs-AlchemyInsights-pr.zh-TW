# <a name="outlook-categories"></a>Outlook 類別

類別只適用于您的行事曆中的專案。 如果您將色彩類別套用至會議邀請或您從其他人收到的會議，該色彩類別只會顯示在您的行事曆中。  如果您對共用信箱套用色彩類別、委派信箱、群組信箱、會議室信箱等等。色彩類別只會影響您的視圖。

如需詳細資訊，[請將色彩類別指派給行事曆約會、會議或事件](https://support.microsoft.com/office/750596d9-707d-4412-8c0e-7fdc0fc52527)。
