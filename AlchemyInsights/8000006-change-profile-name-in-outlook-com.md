# <a name="change-your-profile-name"></a>變更您的設定檔名稱

1. 移至[您的 Microsoft 帳戶資訊](https://go.microsoft.com/fwlink/p/?linkid=860841)。
    
2. 選取 [**編輯名稱**。 
    
3. 輸入名字和姓氏，然後選取 [**儲存**]。 
    
請允許最多 24 小時的值變更為您的設定檔名稱才會生效。
  
若要深入了解，請參閱[變更您的設定檔照片或在 Outlook.com 中的名稱](https://go.microsoft.com/fwlink/?linkid=873110)。
