# <a name="how-to-view-and-manage-apps"></a>如何查看和管理應用程式

您可以在 Microsoft 365 系統管理中心中查看您的應用程式。

1. 在系統管理中心，移至 **[帳單]** > [您的產品](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面。
2. 在 [ **產品** ] 索引標籤上，向下按 [ **應用程式** ] 區段。
3. 若要管理應用程式，請選取您要管理的應用程式。
