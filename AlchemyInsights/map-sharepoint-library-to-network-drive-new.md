# <a name="map-a-sharepoint-library-to-a-network-drive"></a>將 SharePoint 程式庫對應至網路磁片磁碟機

將文件庫對應為網路磁碟機是暫時作法，且僅支援 Internet Explorer。 您必須偶爾在 Internet Explorer 中開啟 SharePoint 網站，然後選取 [ **保留登入** ] 以防止會話過期。 相反地，請[使用新的 OneDrive 同步用戶端來同步處理 SharePoint](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)檔案，該用戶端會依 </a> [需求](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e)提供檔案。 不必使用本機儲存空間，即可在 OneDrive 中存取所有檔案。

如果您選擇要對應磁片磁碟機，而不是 [使用新的 OneDrive 同步處理用戶端](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)，請務必遵循下列文章中的步驟。 


**如何設定及疑難排解對應的網路磁片磁碟機**


請參閱 [疑難排解連接至 SharePoint Onlines 的對應網路磁碟機](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives)。

附注：如果您在 Windows 8 或 Windows 7 上使用 Internet Explorer 10，且在對應磁片磁碟機時**無法存取**「接收**存取權**」或「路徑」，請安裝[此修復程式](https://support.microsoft.com/help/2846960)以解決此問題。
