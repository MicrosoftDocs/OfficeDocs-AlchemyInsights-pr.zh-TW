# <a name="sending-email-as-the-shared-mailbox"></a>為共用信箱傳送電子郵件

[在 Outlook 2016 和 Outlook 2013 中開啟並使用共用信箱](https://support.office.com/en-us/article/open-and-use-a-shared-mailbox-in-outlook-2016-and-outlook-2013-d94a8e9e-21f1-4240-808b-de9c9c088afd)
  
[新增至 Outlook web 上共用的信箱](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)
