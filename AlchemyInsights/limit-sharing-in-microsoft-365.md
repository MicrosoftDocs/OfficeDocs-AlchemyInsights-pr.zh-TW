# <a name="limit-sharing-in-microsoft-365"></a>在 Microsoft 365 中限制共用

雖然您不能完全停用內部共用，或從網站移除 [共用] 按鈕，但有多種方式可以限制 Microsoft 365 中的共用，以符合組織的需求。 如需詳細資訊，請參閱[限制在 Microsoft 365 中共用](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)。
