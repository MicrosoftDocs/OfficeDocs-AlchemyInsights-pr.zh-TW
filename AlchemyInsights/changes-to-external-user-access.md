# <a name="changes-to-external-user-access"></a>外部使用者存取的變更

啟動 2018 年 3 月 23 日，外部使用者將不再能夠存取已與共用所有人，所有已驗證的使用者或所有表單使用者，即使他們已受邀或新增至授權群組的內容。 外部使用者只能存取與使用者或使用者所屬的群組已共用的內容。 如果您想要保留您現有的設定，請參閱：https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
