# <a name="changes-to-external-user-access"></a>外部使用者存取的變更

啟動 2018 年 3 月 23 外部使用者將無法再能夠存取即使他們已受邀或新增至授權群組，與所有人、 所有經過驗證的使用者或所有的表單使用者具有共用的內容。外部使用者將只能夠存取使用者或使用者所屬的群組已共用的內容。如果您想要保留現有的設定，請參閱：https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
