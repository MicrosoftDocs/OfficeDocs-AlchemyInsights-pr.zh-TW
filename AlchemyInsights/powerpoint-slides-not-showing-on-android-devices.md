# <a name="powerpoint-slides-not-showing-in-skype-for-business-on-android-devices"></a>Android 裝置上的商務用 Skype 無法顯示 PowerPoint 投影片

如果您在 Android 裝置上遇到無法顯示 PowerPoint 投影片的問題，這可能是因為 Chrome 瀏覽器中的錯誤所造成。

嘗試停用 Chrome，然後解除安裝。 解除安裝 Chrome 之後，重新啟用 Chrome 以取得原廠版本 (74.0.3729.136)，即可解決問題。
