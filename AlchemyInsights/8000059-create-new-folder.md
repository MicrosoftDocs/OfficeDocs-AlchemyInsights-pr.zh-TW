# <a name="create-a-new-folder"></a>建立新的資料夾

1. 在 [功能窗格中的**資料夾**，選取**新的資料夾**。 
    
2. 輸入資料夾的名稱，並按 Enter 鍵。
    
若要深入了解，請參閱 ＜ [Create Outlook.com 中的資料夾](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a)。
