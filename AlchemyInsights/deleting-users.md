# <a name="deleting-users"></a>刪除使用者

如果您想要立即封鎖使用者存取權與需要在保留其資訊，請參閱[有離開員工且需要封鎖電子郵件資料的存取權吗？](https://support.office.com/client/ba665d35-f4af-4bd0-b52d-841df7454d4b)
  
如果您只想要刪除的使用者並不擔心使用任何自己的資料，請參閱[刪除您組織中的使用者](https://support.office.com/article/d5155593-3bac-4d8d-9d8b-f4513a81479e)。這將會立即釋出其授權。
