# <a name="about-room-mailboxes"></a>會議室信箱的相關

會議室信箱是指派給像是會議室、視聽室或訓練室等實體地點的資源信箱。 如需有關會議室信箱的詳細資訊，請參閱[建立及管理會議室信箱](https://go.microsoft.com/fwlink/p/?linkid=717533)。
