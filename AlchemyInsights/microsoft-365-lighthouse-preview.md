# <a name="welcome-to-the-microsoft-365-lighthouse-private-preview"></a>歡迎使用 Microsoft 365 Lighthouse 私人預覽

您參與 Microsoft 365 Lighthouse 的私人預覽。 如果您想要提供關於 Microsoft 365 Lighthouse 的意見反應，請將詳細資料張貼至我們的 [使用者語音網站](https://aka.ms/M365Lighthouseuservoice)。 如果您在使用 Microsoft 365 Lighthouse 時遇到問題或面臨問題，請建立支援要求以取得進一步的協助。
