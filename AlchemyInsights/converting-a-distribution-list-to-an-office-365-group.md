# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>將通訊群組轉換成 Microsoft 365 群組

您可以將通訊群組轉換成符合特定準則的 Microsoft 365 群組。 若要轉換群組或疑難排解轉換問題，請參閱將 [通訊群組清單升級至 Outlook 中的 Microsoft 365 群組](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)。

我們也有可讓您執行的腳本，以協助您 [驗證要升級至 Microsoft 365 群組的通訊群組資格](https://aka.ms/DLToM365Group)。
