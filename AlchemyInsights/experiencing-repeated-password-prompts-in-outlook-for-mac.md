# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>遭遇到重複 Outlook for Mac 中的密碼提示吗？

我們可以協助提供更多特定的 Outlook for Mac 說明直接從應用程式。
  
請移至受影響的使用者的 Mac 開啟 [Outlook]，然後按一下**協助** \> **連絡支援服務**。
  
若要深入了解使用 Outlook for Mac 中應用程式客戶支援，請參閱本文： [Outlook 2016 for Mac 中的新連絡人支援功能](https://support.office.com/article/https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
