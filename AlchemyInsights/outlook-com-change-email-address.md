# <a name="change-your-email-address"></a>變更您的電子郵件地址

若要變更您的 Outlook.com 電子郵件地址，您必須加入另一個電子郵件地址，並將其設為主要電子郵件別名。
  
依照[Add 或 remove Outlook.com 中的電子郵件別名](https://support.office.com/article/459b1989-356d-40fa-a689-8f285b13f1f2?wt.mc_id=Office_Outlook_com_Alchemy)中的指示進行。
