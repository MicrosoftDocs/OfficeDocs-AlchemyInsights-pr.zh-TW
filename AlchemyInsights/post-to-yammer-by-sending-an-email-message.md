# <a name="post-to-yammer-by-sending-an-email-message"></a>傳送電子郵件訊息張貼至 Yammer

[最新版本的 Outlook](https://support.microsoft.com/office/work-with-yammer-from-outlook-fd695485-225b-410f-b24a-17f971b46b25) 可在 Yammer 中提供電子郵件的最佳體驗。 這包括完全互動交談的體驗，包括：對郵件進行訊息、查看 Gif，以及能夠像您在網頁上使用 Yammer 一樣回復。 其他的電子郵件類型超過個別交談執行緒（如探索電子郵件）之後，將會成為互動式的未來。

其他用戶端可以透過 [電子郵件傳送至 Yammer](https://support.microsoft.com/office/new-yammer-post-to-yammer-by-sending-an-email-message-830e6825-56f6-4169-a6b9-1b3ca0cdad4d) ，以開始新的交談，或回復現有的交談。 這適用于所有具有正確安全性設定的電子郵件系統和用戶端。 歐盟地區的客戶僅限張貼至 [Microsoft 365 所連接](https://docs.microsoft.com/yammer/manage-yammer-groups/yammer-and-office-365-groups)的社區。
