# <a name="cant-read-email-in-google-chrome-with-adblock"></a>無法讀取 Google Chrome 搭配 AdBlock 電子郵件

某些使用者讀取電子郵件時遇到問題，或是使用 Google Chrome AdBlock 副檔名時撰寫新電子郵件。請嘗試下列解決方法：
  
- 使用[Microsoft 的邊緣](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409)或其他網頁瀏覽器。 
    
- 排除 AdBlock 設定 Outlook.com。在右上角的 Google Chrome、 選取**AdBlock** ，然後選取 [**不要在這個網域上的頁面上執行**。若要一次載入 Outlook.com 瀏覽器的重新整理。
