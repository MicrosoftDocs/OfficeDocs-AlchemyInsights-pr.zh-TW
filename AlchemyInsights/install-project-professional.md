# <a name="install-project-professional"></a>安裝 Project Professional

請參閱了解如何安裝訂閱版本的 Project Online，與非訂閱版本的專案 2019，2016年或 2013年的[安裝專案](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281?ui=en-US&rs=en-US&ad=US)。 

Project 2010 或 2007年，請遵循[Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)中的步驟 或 [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657)。 

專案售出做為獨立的應用程式，因此，如果您不需要它尚未， [購買或試用專案](https://products.office.com/project)。
