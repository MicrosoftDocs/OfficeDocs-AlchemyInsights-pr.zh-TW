# <a name="what-are-groups"></a>何謂群組？

Microsoft 365 群組可讓您選擇一組您想要共同作業的人員，並輕鬆地設定共用的資源集合。 您不需要擔心手動指派許可權給所有這些資源，因為新增成員至群組會自動將其所需的許可權提供給群組所提供的工具。 此外，「群組」是我們用於使用通訊群組清單或共用信箱進行哪些新的和改善的經驗。  如需詳細資訊，請參閱 [瞭解 Microsoft 365 群組](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2) 。
