# <a name="set-up-microsoft-information-protection"></a>設定 Microsoft 資訊保護

概括瞭解可套用於資訊保護戰略的功能，以便您確信您的敏感資訊得到了保護。 使用四階段生命週期方法來探索、分類、保護和監視敏感資訊。 「Microsoft 資訊保護設定」指南提供了完成每個階段的指導。
