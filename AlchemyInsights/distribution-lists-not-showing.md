# <a name="distribution-group-not-showing-in-groups-list"></a>不顯示群組] 清單中的通訊群組

可能很最多 60 分鐘才新的通訊群組清單會顯示在 Exchange 系統管理中心管理中心。
  
如果您要新增成員或正進行其他的編輯，請在[Exchange 系統管理中心](https://support.office.com/article/https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx)使用 「 群組 」 功能。
