# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>設定共用信箱的疑難排解 Outlook 行動裝置

您無法從行動裝置已安裝的 Outlook 應用程式存取共用的信箱。您是否希望此功能吗？讓 Outlook 工程小組知道。[以下投票](https://go.microsoft.com/fwlink/?linked=862116)！
  
您可以僅使用共用的信箱所開啟的行動裝置瀏覽器中登入 Office 365，然後再進入網路上的 Outlook。指示，請參閱[新增 outlook web 上共用信箱](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)。從網路上的 Outlook 您將能夠存取共用的信箱。
  
深入了解：[取得 iOS 及 Android outlook 的應用程式說明](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
