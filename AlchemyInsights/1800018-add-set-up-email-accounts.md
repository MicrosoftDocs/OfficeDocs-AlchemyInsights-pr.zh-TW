# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>設定在 Outlook 中的 Exchange 電子郵件帳戶的問題

如果您設定在 Outlook 中的 Exchange 帳戶的問題，您可以在受影響的電腦上執行[Outlook 帳戶設定工具](https://aka.ms/SaRA-OutlookSetupProfile)。它會執行各種檢查並提供帳戶安裝問題可能的解決方案。 
  
如果您正在仍然有問題，請參閱[Outlook 無法設定新的設定檔所使用的 Office 365 中的 Exchange Online 信箱的 Exchange 自動探索](https://support.microsoft.com/help/2404385/outlook-can-t-set-up-a-new-profile-by-using-exchange-autodiscover-for)。
