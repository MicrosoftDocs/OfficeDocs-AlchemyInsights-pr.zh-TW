# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>取得帳戶已停用在網路上的 Outlook 中的錯誤吗？

如果您要取得**您的帳戶已停用**的錯誤與`X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException`當您嘗試開啟 （前身為 Outlook Web App） 在 web 上的 Outlook，貴組織管理可能已停用在存取 outlook web 上。您的系統可修正下列錯誤所[啟用或停用 Outlook Web App 信箱的](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)主題中的步驟。
