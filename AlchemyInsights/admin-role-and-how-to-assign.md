# <a name="admin-roles"></a>系統管理員角色

- 在系統管理中心選取 [[角色](https://admin.microsoft.com/Adminportal/Home#/roles)]，以指派或深入瞭解系統管理員角色。

- 選取角色名稱，然後按一下 [指派角色]****，或按一下 [比較角色]**** 以深入瞭解。
