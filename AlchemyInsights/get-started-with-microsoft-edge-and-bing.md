# <a name="get-started-with-microsoft-edge--bing"></a>開始使用 Microsoft Edge + Bing

開始使用 Microsoft Edge + Bing 時，不需要額外進行其他設定。 若要尋找與工作相關的資訊，只要在 Microsoft Edge 的網址列中輸入您的查詢。

若要深入瞭解，請參閱 [使用 Microsoft Edge + Bing 取得更多內容](https://go.microsoft.com/fwlink/?linkid=2152963)。
