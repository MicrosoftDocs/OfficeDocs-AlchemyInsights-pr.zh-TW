# <a name="delete-a-sharepoint-team-site"></a>刪除 SharePoint 小組網站

1. 移至網站。
    
2. 按一下右上角的 [**設定**] 圖示。 
    
3. 按一下 [**網站**設定]。 在**站台資訊**或**網站動作]** 下，按一下 [**刪除此網站**]，然後按一下 [**刪除**。
