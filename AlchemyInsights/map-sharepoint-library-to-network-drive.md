# <a name="map-a-sharepoint-library-to-a-network-drive"></a>網路磁碟機對應的 SharePoint 文件庫

若要使用網站文件庫在檔案總管] 中，我們建議使用 OneDrive sync 用戶端，而不對應到網路磁碟機的文件庫。磁碟機對應是只是暫時性的解決方案。使用 sync 用戶端檔案可供使用者運用即使它們離線。Sync 用戶端也提供較佳的效能、 不必下載這些使用[檔案隨選](https://support.office.com/article/Learn-about-OneDrive-Files-On-Demand-0E6860D3-D9F3-4971-B321-7092438FB38E)允許存取檔案並不需要使用 Internet Explorer 的安裝程式。 
  
- [在 Windows 中開始使用新版 OneDrive 同步處理用戶端](https://go.microsoft.com/fwlink/?linkid=866427)
    
- [在檔案總管中使用網站文件庫檔案的方法](https://go.microsoft.com/fwlink/?linkid=866291)
    
- [設定及疑難排解對應的網路磁碟機](https://support.microsoft.com/kb/2616712)
