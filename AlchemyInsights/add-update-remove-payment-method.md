# <a name="add-update-or-remove-payment-method"></a>新增、更新或移除付款方式

您可以使用信用卡或轉帳卡或銀行帳戶 (直接扣款) 購買 Microsoft 的產品和服務。

- 若要將銀行帳戶新增為付款方式，請參閱[新增付款方式](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide#add-a-payment-method)。

- 若要更新銀行帳戶的資訊，請參閱 [更新目前的付款方式](https://docs.microsoft.com/en-us/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide#update-an-existing-payment-method)。

- 若要至或由銀行帳戶變更付款方式，請參閱[變更付款方式](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide#change-a-payment-method)。

- 若要將銀行帳戶移除為付款方式，請參閱[移除付款方式](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide#remove-a-payment-method)。 

如需詳細資訊，請參閱[新增、更新或移除付款方式](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/add-update-or-remove-credit-card-or-bank-account?view=o365-worldwide) (部分機器翻譯)。
