# <a name="unable-to-delete-photos-from-files"></a>無法從檔案刪除相片

若要刪除的檔案或相片：
  
1. 移至您**的收件匣**] 或 [**寄件備份**資料夾，並刪除它們附加至電子郵件訊息。 
    
2. 從您**已刪除的項目**移除的項目。 
    
如需詳細資訊，請閱讀[如何刪除檔案或 Outlook.com 的相片](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx)。
