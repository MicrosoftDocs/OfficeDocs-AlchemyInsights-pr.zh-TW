# <a name="unable-to-delete-photos-from-files"></a>無法刪除檔案中的相片

若要刪除檔案或相片:
  
1. 移至 [**收件**匣] 或 [**寄**件備份] 資料夾, 並刪除其所附加的電子郵件。

2. 從**刪除的專案**中移除專案。

如需詳細資訊, 請參閱[如何刪除 Outlook.com 中的檔案或相片](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx)。
