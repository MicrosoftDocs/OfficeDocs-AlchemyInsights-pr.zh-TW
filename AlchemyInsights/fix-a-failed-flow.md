# <a name="fix-a-flow-that-failed"></a>修正失敗流程

1. 移至[Microsoft Flow](https://flow.microsoft.com/)，然後按一下右上角的 [通知] 圖示。 （或是在行動裝置 app 中，選取 [活動] 索引標籤）。選取的流程，以查看其詳細資料失敗。
    
2. 尋找與紅色的驚嘆號圖示以會看見錯誤訊息流程的步驟。
    
3. 根據錯誤訊息中，您應該能夠編輯流程並修正問題。 
    
[常見的流程失敗](https://go.microsoft.com/fwlink/?linkid=872110)
