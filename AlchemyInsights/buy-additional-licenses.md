# <a name="buy-additional-licenses"></a>購買額外的授權

1. 在系統管理中心中，移至 [訂閱][](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面，或選擇 [計費]**** \> [訂閱]****。

2. 在 [**訂閱**] 頁面上, 選擇您想要新增授權的訂閱, 然後選擇 [**新增/移除授權**]。

    [如果看不到 [新增/移除] 連結, 該怎麼辦？](https://support.office.com/article/36081d8d-b3fa-4948-8c34-e217bba825e1#bkmk_no_link)

3. 在 [**授權總數**] 方塊中, 輸入此訂閱所需的授權總數, 然後選擇 [**提交** \> ] [**關閉**]。 For example, if you have 100 licenses and you need to add 5 more, enter 105.

    > [!NOTE]
    > 如果您是以信用卡或銀行帳戶支付訂閱費用，在收到訂單確認後，就可立刻使用所購買的所有新授權。 如果您的訂閱付款方式是使用發票，您可能需要等待信用查核，然後才能使用新授權。
