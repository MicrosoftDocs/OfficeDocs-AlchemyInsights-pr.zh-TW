# <a name="buy-additional-licenses"></a>購買額外的授權

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務][](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面。

2. 尋找您要新增授權的訂閱，然後選取 [**新增/移除授權**]。

    [如果看不到 [新增/移除] 連結，該怎麼辦？](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/buy-licenses#what-if-i-dont-see-the-addremove-licenses-link)

3. 在 [**授權總數**] 方塊中，輸入此訂閱所需的授權總數，然後選擇 [**提交變更**]。 For example, if you have 100 licenses and you need to add 5 more, enter 105.

    > [!NOTE]
    > 如果您是以信用卡或銀行帳戶支付訂閱費用，在收到訂單確認後，就可立刻使用所購買的所有新授權。 如果您的訂閱付款方式是使用發票，您可能需要等待信用查核，然後才能使用新授權。

若要深入瞭解，請參閱[購買商務用 Microsoft 365 訂閱的授權](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/buy-licenses)。
