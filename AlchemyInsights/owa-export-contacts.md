# <a name="export-contacts"></a>匯出連絡人

1. 在頁面底部，選取 [移至 [人員] 頁面上的 [人員] 圖示。
2. 在工具列上，選取 [**管理** > **匯出連絡人**。 
3. 選擇 [所有連絡人] 或 [只有連絡人都匯出特定資料夾而言，然後選取 [**都匯出**]。
4. 在頁面底部，選取 [預設的下載項目] 資料夾中的 [**儲存**] 以儲存 「 contacts.csv 」。 或者，選取 [**儲存**] 旁的箭號，然後選取 [**另存為**。

若要深入了解，請參閱[從 Outlook 連絡人匯出](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f#ID0EAACAAA=Outlook_on_the_web)。
