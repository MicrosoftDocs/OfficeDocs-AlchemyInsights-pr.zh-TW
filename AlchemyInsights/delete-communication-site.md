# <a name="delete-a-communication-site"></a>刪除通訊網站

1. 移至網站。
    
2. 按一下右上角的 [設定] 圖示。
    
3. 按一下 [**網站設定**]。 在**站台資訊**] 下按一下 [**刪除此網站**]，然後按一下 [**刪除**。
