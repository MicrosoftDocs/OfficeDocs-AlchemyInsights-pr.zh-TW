# <a name="setting-up-a-hybrid-environment"></a>設定混合式環境

若要讓您的使用者在其資料儲存在內部部署伺服器或 Office 365 時無縫地一起運作，您可能需要設定混合式環境。 若要深入瞭解您的選項，請參閱[Office 365 與內部部署環境的整合](https://docs.microsoft.com/office365/enterprise/office-365-integration)。
