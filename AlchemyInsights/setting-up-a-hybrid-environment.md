# <a name="setting-up-a-hybrid-environment"></a>設定混合式環境

若要讓使用者可以順暢地搭配使用，無論文如果其資料儲存在內部部署伺服器上或在 Office 365 中，您可能需要設定混合式環境。 若要取得您選項的概觀，請參閱[Office 365 與內部部署環境的整合](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx)。
