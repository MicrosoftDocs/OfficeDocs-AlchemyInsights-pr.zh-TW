# <a name="turn-off-text-predictions"></a>關閉文字預測

1. 選取 [ **設定] > 查看所有 Outlook 設定 > 郵件 > 撰寫及回復**。
2. 在 [ **文字預測**] 底下，取消選取 [ **輸入時的建議片語文字**]。

如需詳細資訊，請參閱 [使用 Outlook 網頁版中的智慧技術和 Outlook.com](https://support.microsoft.com/office/use-intelligent-technology-in-outlook-on-the-web-and-outlook-com-24b30683-8340-4b69-b8ac-4193ec528a70)。
