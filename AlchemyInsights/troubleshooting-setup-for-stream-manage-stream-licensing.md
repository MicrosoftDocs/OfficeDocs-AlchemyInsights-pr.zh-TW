# <a name="managing-microsoft-stream-licenses"></a>管理 Microsoft Stream 授權

**重要：** 您必須是租用戶系統管理員角色，才能執行本主題所述的作業。

您必須是全域系統管理員或使用者管理系統管理員。如需詳細資訊，請參閱[關於系統管理員角色](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles)。

您可以指派[授權給使用 Office 365 PowerShell 的使用者帳戶](https://go.microsoft.com/fwlink/p/?linkid=850410)。
