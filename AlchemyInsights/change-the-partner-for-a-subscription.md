# <a name="change-the-partner-for-a-subscription"></a>變更訂閱的合作夥伴

您可以在 [Microsoft 365 系統管理中心]**** 中變更訂閱的合作夥伴。

1. 在系統管理中心，移至 [帳單] > [產品與服務]**[](https://go.microsoft.com/fwlink/p/?linkid=842054)** 頁面。 
2. 如果您有多個訂閱，請選取您要編輯的訂閱名稱。 
3. 在 [合作夥伴 ID]**** 下，按一下 [編輯記錄可查夥伴]****。
4. 輸入您新增之合作夥伴的新 Microsoft 合作夥伴 ID****、按一下 [檢查 ID]****，然後按一下 [提交]****。 您可以詢問合作夥伴，以取得對方的 Microsoft 合作夥伴識別碼。

如需詳細資訊，請參閱[新增、變更或刪除訂閱顧問合作夥伴](https://docs.microsoft.com/microsoft-365/admin/misc/add-partner)。
