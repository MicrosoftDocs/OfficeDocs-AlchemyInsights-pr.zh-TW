# <a name="create-users-in-hybrid-deployments"></a>混合部署中建立使用者

當您的組織之間的內部部署環境與 Office 365 的混合部署時，您需要在內部部署 Active Directory 中建立使用者的所有物件。建立使用者物件中的適當的組織單位 (OU)，讓他們正在同步處理至 Office 365 組織。
