# <a name="error-when-an-office-365-trial-user-sends-email-client-host-blocked-using-spamhaus"></a>Office 365 試驗使用者傳送電子郵件時的錯誤： 使用 Spamhaus 封鎖的用戶端主機

它會顯示您要從新的 Office 365 試用版的組織傳送電子郵件。若要協助防止不當使用，我們會封鎖先前已遭洩露已知的 IP 位址。開始使用付費授權之後不再強制執行此特定檢查。您可以透過從網際網路服務提供者 (ISP)，並要求新的 IP 位址或移除封鎖從[Spamhaus](https://go.microsoft.com/fwlink/p/?linkid=123245)網站取得暫時浮雕。
