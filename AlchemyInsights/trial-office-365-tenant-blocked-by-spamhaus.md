# <a name="error-when-a-microsoft-365-trial-user-sends-email-client-host-blocked-using-spamhaus"></a>Microsoft 365 試用使用者傳送電子郵件時發生錯誤：已使用 Spamhaus 封鎖用戶端主機

您似乎從新的 Microsoft 365 試用組織傳送電子郵件。 為了協助防止濫用，我們封鎖已知先前已遭破壞的 IP 位址。 開始使用付費授權之後，就不再強制執行此特定檢查。 您可以從網際網路服務提供者（ISP）尋求新的 IP 位址，或從[Spamhaus](https://go.microsoft.com/fwlink/p/?linkid=123245)網站移除該塊，以取得暫時的起伏。
