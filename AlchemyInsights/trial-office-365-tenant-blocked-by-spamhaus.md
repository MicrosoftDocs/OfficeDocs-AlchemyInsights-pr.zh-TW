# <a name="error-when-an-office-365-trial-user-sends-email-client-host-blocked-using-spamhaus"></a>Office 365 試用版使用者傳送電子郵件時發生錯誤: 已使用 Spamhaus 封鎖用戶端主機

您似乎是從新的 Office 365 試用組織傳送電子郵件。 為了防止濫用, 我們封鎖已知先前被攻破的 IP 位址。 開始使用付費授權後, 就不再強制執行這種特定檢查。 您可以從網際網路服務提供者 (ISP) 取得新的 IP 位址, 或是從[Spamhaus](https://go.microsoft.com/fwlink/p/?linkid=123245)網站移除區塊, 以取得暫時的止裂止。
