# <a name="enable-microsoft-stream"></a>啟用 Microsoft Stream

貴組織的員工在帳戶取得授權之前，無法使用 Stream 服務。 您可以將 Stream 授權指派給每位使用者，這樣他們就不需要手動註冊服務。

- [為組織中所有人啟用 Microsoft Stream](https://docs.microsoft.com/stream/assign-user-licenses)。
