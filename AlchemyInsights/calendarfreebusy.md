# <a name="issues-seeing-calendar-freebusy-information"></a>查看行事曆空閒/忙碌資訊的問題

如果您在其他行事曆中看不到空閒/忙碌資訊，或其他人在您的行事曆中看不到空閒/忙碌資訊，請使用[適用於 Office 365 的 Microsoft 支援及修復小幫手](https://diagnostics.office.com/)。
