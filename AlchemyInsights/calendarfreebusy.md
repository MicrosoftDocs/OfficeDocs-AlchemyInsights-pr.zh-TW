# <a name="issues-seeing-calendar-freebusy-information"></a>查看行事曆空閒/忙碌資訊的問題

如果您在其他 Outlook calendar 中看不到 [行事曆空閒/忙碌資訊]，或是其他人無法在您的行事曆中看到空閒/忙碌資訊，請使用 microsoft [支援和修復的 microsoft 365](https://diagnostics.office.com/)。
