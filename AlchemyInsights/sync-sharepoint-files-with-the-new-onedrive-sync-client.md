# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>將 SharePoint 檔案與新的 OneDrive 同步處理用戶端同步

1. 在傳統的 SharePoint 系統管理中心，選取 [**設定**]。
    
2. 請確定 「 OneDrive 同步處理] 按鈕 」 設為 「 顯示同步處理] 按鈕 」。
    
3. 「 同步處理用戶端 for SharePoint，「 選取 [**啟動新的用戶端**]。
    
    > [!NOTE]
    > 如果沒有出現這個選項，您的組織已設定要使用新版 OneDrive 同步處理用戶端來同步處理 SharePoint 檔案。 
  
4. 按一下 [確定]****。
    
如需詳細資訊，請參閱[啟用同步處理 SharePoint 檔案與新的 OneDrive 同步處理用戶端的使用者](https://go.microsoft.com/fwlink/?linkid=866433)。 授與使用者的相關指示，請參閱 <<c0>同步處理 SharePoint 檔案與新的 OneDrive 同步處理用戶端。
