# <a name="enable-or-disable-forms-pro"></a>啟用或停用表單專業人員

如果您想要啟用 Microsoft Forms 專業人員，最好是[Dynamics 365 企業版](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant)授權的一部分。 或者，系統管理員可以購買[Microsoft Forms 專業版的授權](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant)。 您也可以從使用者[移除 Microsoft Forms 專業版的授權](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1)。
