# <a name="get-started-with-using-microsoft-search-usage-reports"></a>開始使用 Microsoft 搜尋使用方式報告

Microsoft 搜尋使用方式報告提供搜尋在組織中運作方式的深入瞭解。 這些真知灼見會協助您尋找內容，並讓您的組織中的使用者搜尋更實用且 delightful 的經驗。

若要深入瞭解，請參閱 [Microsoft 搜尋使用方式報告](https://go.microsoft.com/fwlink/?linkid=2152048)。
