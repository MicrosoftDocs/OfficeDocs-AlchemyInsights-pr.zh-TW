# <a name="banned-sending-ip"></a>禁止傳送 IP

傳送郵件給 Microsoft 365 時，寄件者可能會遇到此錯誤。 寄件者必須造訪https://sender.office.com並依照指示執行。  如果他們沒有從入口網站接收驗證電子郵件，請讓他們檢查 [垃圾郵件] 資料夾和/或嘗試使用不同的電子郵件地址。
