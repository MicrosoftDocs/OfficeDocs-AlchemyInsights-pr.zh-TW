# <a name="edit-a-read-only-file"></a>編輯唯讀檔案

您可能會發現，當您開啟檔案時，這些檔案會以唯讀方式開啟。 在某些情況下，這是為了增加安全性，例如當您從網際網路開啟檔案時，以及其他時間，這可能是因為可變更的設定。 在本文中，[我的檔案 [為何會以唯讀方式開啟](https://support.office.com/article/Why-did-my-file-open-read-only-3ab4b792-da50-4b38-8628-14c64e1f1d15) ] 中，我們會概述一些檔案開啟唯讀的案例，以及您可以採取哪些步驟來變更該檔案。

以下說明如何開啟 office 檔案進行編輯。

請按一下 [ **開啟] 功能表** (三個點) ]，然後按一下 [ **編輯**]，而不是按一下檔標題。

附註：

- 如果您在按一下標題時，已開啟 Office 中的檔進行編輯，您可以設定文件庫執行該 (，只要您具有變更文件庫設定) 的許可權。

- 如果您有文件庫具有結帳、必要欄，或只有在 SharePoint 文件庫的版本設定中可以編輯的使用者，這些專案就會以唯讀方式進行同步處理。

- 即使您啟用「以讀/寫方式開啟 Office 檔」原則設定，Office 檔還是會在受保護的檢視中開啟。 如需詳細資訊，請參閱 [本文。](https://support.microsoft.com/help/983047/an-office-document-opens-in-protected-view-even-though-you-enable-the)
