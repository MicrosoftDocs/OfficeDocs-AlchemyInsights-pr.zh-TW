# <a name="restore-deleted-files-or-folders-in-onedrive"></a>還原已刪除的檔案或 OneDrive 中的資料夾

如果您不小心刪除檔案或資料夾，在 OneDrive 中的，您可以稍後 OneDrive 資源回收筒中復原。

* [還原已刪除的檔案或 OneDrive 中的資料夾](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)

如果您的 OneDrive 檔案大量取得已刪除、 覆寫、 損毀，或感染惡意程式碼，您可以還原整個 OneDrive 至先前的時間。

* [還原您的 OneDrive](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
