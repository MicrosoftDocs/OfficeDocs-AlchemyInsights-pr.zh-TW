# <a name="public-folder-access-from-mobile-devices"></a>移動裝置的公用資料夾存取權

存取公用資料夾無法使用行動裝置，請使用 OWA (網頁版 Outlook) 或 Outlook for Windows desktop 或 Outlook for Mac 用戶端存取公用資料夾。

如果您想要具有與公用資料夾類似的功能，且在行動裝置上具有可存取的內容，請參閱 [瞭解 Microsoft 365 群組](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) 以取得更好的替代方法。
