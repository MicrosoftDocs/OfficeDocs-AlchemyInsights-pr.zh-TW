# <a name="enable-seamless-sso"></a>啟用無縫 SSO

Azure Active Directory (Azure AD) 無縫單一登入 (無縫 SSO) 會自動登入的使用者在連線至公司網路其公司桌上型電腦上時。
  
[請遵循本篇文章以啟用無縫 SSO 的步驟](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
