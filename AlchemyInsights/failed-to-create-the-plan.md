# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>嘗試在 Microsoft 小組中建立 Planner 索引標籤時發生「無法建立方案」錯誤

如果您在嘗試在 Microsoft 小組中建立 Planner 標籤時發生「無法建立方案」錯誤，請遵循下列步驟，確定系統管理員是群組的成員：

1. 在系統管理中心中，移至 [**群組**] [  >  [群組](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups)]。 
2. 選取群組，並在 [成員]**** 索引標籤中選取 [檢視所有並管理成員]****。
3. 新增成員，然後選取 [ **儲存**]。
