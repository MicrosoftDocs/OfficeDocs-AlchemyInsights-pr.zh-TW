# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>「 無法建立計劃 」 的錯誤時，嘗試建立計劃] 索引標籤中 Microsoft Teams

如果您嘗試建立計劃] 索引標籤中 Microsoft Teams 時收到 「 無法建立計劃 」 的錯誤，請確定系統管理員群組的成員遵循下列步驟：

1. 在系統管理中心，移至 [**群組** > [群組](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups)。 
2. 選取群組，，然後在 [**成員**] 索引標籤上選取 [**檢視所有及管理成員**。
3. 新增成員並選取 [**儲存**]。
