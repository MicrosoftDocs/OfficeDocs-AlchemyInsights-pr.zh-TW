# <a name="change-your-password"></a>變更您的密碼

Outlook.com 密碼是 Microsoft 帳戶密碼相同。
  
若要變更您的密碼，請前往[https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409)並選取 [**變更密碼**。 
  
如果您忘記您的密碼，請移至[Microsoft 帳戶的密碼重設] 頁面](https://go.microsoft.com/fwlink/p/?linkid=841909)。
  
若要深入了解，請參閱[變更您在 Outlook.com 中的密碼](https://go.microsoft.com/fwlink/?linkid=873109)。
