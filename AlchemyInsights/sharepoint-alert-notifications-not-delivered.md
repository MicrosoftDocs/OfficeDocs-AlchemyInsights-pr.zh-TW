# <a name="sharepoint-alert-notifications-not-delivered"></a>未傳送 SharePoint 警示通知 

<p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">判斷<strong>是否所有警示</strong>都無法傳送，還是只有從<strong>特定檔案或文件庫</strong>的個別警示無法傳送。</span></p> <ol> <li><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">如果從多個檔案或文件庫的<strong style="mso-bidi-font-weight: normal;">所有警示</strong>都無法傳送，請查看<a href="https://admin.microsoft.com/AdminPortal/Home#/servicehealth">服務健康狀況儀表板</a>來檢查 SharePoint 或 Exchange 是否有發生任何諮詢或事件。 此問題可能與 SharePoint 警示功能有關，或透過 Exchange 電子郵件的延遲。 另外也請注意是否有其他電子郵件傳遞，如果沒有，問題可能與 Exchange 延遲相關。 <br /><br /></span></li> <li style="line-height: normal; font-size: 11pt; font-style: normal; font-weight: 400;">
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">如果從特定檔案或文件庫的</span><strong><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">個別警示</span></strong><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">無法傳送，請嘗試刪除與重新建立。請參閱</span><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> <a href="https://support.office.com/zh-TW/article/manage-view-or-delete-sharepoint-alerts-99dfb19c-9a90-4a8c-aba1-aa8c8afb0de2#ID0EAADAAA=Online">管理、檢視或刪除 SharePoint 警示 (機器翻譯) </a>以重新建立警示。 </span></li> </ol> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;">
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">如需有關警示通知，請參閱 <span style="color: #333333;"><a href="https://support.office.com/zh-TW/article/create-an-alert-to-get-notified-when-a-file-or-folder-changes-in-sharepoint-e5a79e7b-a146-46da-a9ef-d65409ba8918">建立警示以在 SharePoint 中的檔案或資料夾變更時獲得通知 (機器翻譯)</a>。</span></span></p> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;">
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">如需有關疑難排解警示的詳細資訊，請參閱<a href="https://support.office.com/zh-TW/article/users-don-t-receive-sharepoint-online-alert-notifications-14fc22dd-e873-482c-844d-f67ad41313f1">使用者無法&rsquo;接收 Sharepoint Online 警示通知</a>。 </span></p> <p>&nbsp;</p>
