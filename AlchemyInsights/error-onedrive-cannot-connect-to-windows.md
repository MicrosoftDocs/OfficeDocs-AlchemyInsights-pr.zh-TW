# <a name="error-onedrive-cannot-connect-to-windows"></a>錯誤： OneDrive 無法連線至 Windows

若要解決錯誤訊息，搜尋 cldflt.sys 檔案正在執行，並不是，如果加以啟用。 
  
如需如何啟用 cldflt.sys 檔案的詳細資訊，請參閱[OneDrive 無法連線至 Windows"存取 OneDrive 中的檔案時的錯誤。](https://go.microsoft.com/fwlink/?Linkid=2031032)
