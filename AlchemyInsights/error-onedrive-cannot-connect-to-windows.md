# <a name="error-onedrive-cannot-connect-to-windows"></a>錯誤： OneDrive 無法連接到 Windows

若要解決錯誤訊息，請搜尋 cldflt 檔案以查看其是否在執行中，如果不是，請加以啟用。 
  
如需如何啟用 cldflt 檔案的其他資訊，請參閱[當存取 OneDrive 中的檔案時，OneDrive 無法連線至 Windows」錯誤。](https://go.microsoft.com/fwlink/?Linkid=2031032)
