# <a name="error-onedrive-cannot-connect-to-windows"></a>錯誤： OneDrive 無法連接到 Windows

若要解決此錯誤訊息，請搜尋 cldflt.sys 檔案以查看其是否正在執行，如果沒有，請加以啟用。 
  
如需如何啟用 cldflt.sys 檔案的其他資訊，請 [OneDrive 參閱當存取 OneDrive 中的檔案時，無法連接至 Windows」錯誤。](https://go.microsoft.com/fwlink/?Linkid=2031032)
