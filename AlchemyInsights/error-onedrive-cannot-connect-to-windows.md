# <a name="error-onedrive-cannot-connect-to-windows"></a>錯誤： OneDrive 無法連線到 Windows

若要解決此錯誤訊息，搜尋 cldflt.sys 檔案，以查看它正在執行，且如果它不會啟用它。 
  
如需有關如何啟用 cldflt.sys 檔案的詳細資訊，請參閱[OneDrive 無法連線到 Windows 」 存取 OneDrive 中的檔案時的錯誤。](https://go.microsoft.com/fwlink/?Linkid=2031032)
