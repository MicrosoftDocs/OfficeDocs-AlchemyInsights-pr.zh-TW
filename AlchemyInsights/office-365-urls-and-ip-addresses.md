# <a name="office-365-urls-and-ip-address-ranges"></a>Office 365 URL 和 IP 位址範圍

Office 365 需要特定稱為*端點*的 IP 位址透過網際網路連線。
如需一定要可以到達的端點的清單，請參閱[Office 365 Url 和 IP 位址範圍](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Office 365 IP 位址和 URL Web 服務可協助您更能識別並區分 Office 365 網路流量，讓您更容易評估、設定及掌握變更。 請參閱[Office 365 IP 位址和 URL web 服務](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service)。
