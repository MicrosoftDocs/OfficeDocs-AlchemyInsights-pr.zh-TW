# <a name="supported-version-of-the-sharepoint-migration-tool"></a>支援的 SharePoint 移轉工具版本



目前 SharePoint 移轉工具僅支援從 SharePoint 2013 的移轉。 版本 2010年和 2016年不支援這一次。
  
如需詳細資訊，請參閱 < <b0>SharePoint 移轉工具簡介</b0>。
