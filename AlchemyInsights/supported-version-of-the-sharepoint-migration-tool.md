# <a name="supported-version-of-the-sharepoint-migration-tool"></a>支援的版本的 SharePoint 移轉工具



目前 SharePoint 移轉工具僅支援從 SharePoint 2013 的移轉。不支援版本 2010年和 2016年這一次。
  
如需詳細資訊，請參閱[簡介 （英文） SharePoint 移轉工具](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)。
