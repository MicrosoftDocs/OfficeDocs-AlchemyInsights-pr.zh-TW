# <a name="how-to-share-sharepoint-files-or-folders"></a>如何共用 SharePoint 檔案或資料夾

您在 SharePoint 網站儲存的檔案可通常是供所有人具有網站權限，但您可能想要與否則不需要存取網站的人員共用特定的檔案或資料夾。 當您共用的檔案及資料夾時，您可以決定是否要讓人員編輯，或只是檢視它們。

[共用 SharePoint 檔案或資料夾](https://support.office.com/article/share-sharepoint-files-or-folders-1fe37332-0f9a-4719-970e-d2578da4941c?ui=en-US&rs=en-US&ad=US)
