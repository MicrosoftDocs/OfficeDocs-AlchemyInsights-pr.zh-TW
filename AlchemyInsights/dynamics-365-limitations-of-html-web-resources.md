# <a name="webpage-html-web-resources-limitations"></a>網頁 (HTML) 網站的資源限制

* HTML 網頁資源不能包含任何必須在伺服器執行的程式碼。 ASP.NET 網頁無法上傳做為 HTML web 資源。

* HTML 網頁資源只能接受有限的數量的查詢字串參數。 更多資訊：[將 HTML 網頁的資源參數傳遞](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
