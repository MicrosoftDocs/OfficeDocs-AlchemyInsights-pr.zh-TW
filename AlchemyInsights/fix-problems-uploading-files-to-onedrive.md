# <a name="fix-problems-uploading-files-to-onedrive"></a>修正問題將檔案上載至 OneDrive

如果檔案不會將上傳，請嘗試這個：
  
- 重新命名檔案並再試一次。不允許某些檔案名稱及檔案名稱中的字元。 
    
- 如果您正在同步處理檔案，將它移移 OneDrive 資料夾 （例如，將它移到您的桌面），而且當將 OneDrive 同步狀態顯示為 「 最新、"將它移回。
