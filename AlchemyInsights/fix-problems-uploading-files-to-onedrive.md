# <a name="fix-problems-uploading-files-to-onedrive"></a>修正問題將檔案上傳至 OneDrive

如果檔案不會將上傳，請嘗試這一項：
  
- 重新命名檔案，然後再試一次。 不允許某些檔案名稱和檔案名稱中的字元。 
    
- 如果您正在同步處理檔案，將其移出 OneDrive 資料夾 （例如，將其移到您的桌面），以及當您的 OneDrive 同步處理的狀態會顯示為 「 最新狀態，「 移回。
