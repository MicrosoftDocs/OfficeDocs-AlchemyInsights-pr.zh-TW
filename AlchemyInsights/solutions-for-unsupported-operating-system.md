"不受支援的作業系統"

# <a name="solutions-for-unsupported-operating-system"></a>"不受支援的作業系統"的解決方案

當您嘗試安裝 Office 365/2016年時，如果您的裝置不符合[系統需求](https://products.office.com/office-system-requirements)您可能會看到下列錯誤訊息。
  
 **"不受支援的作業系統"**
  
Office 365/2016年需要：
  
PC： Windows 10、 Windows 8.1、 Windows 8、 Windows 7 Service Pack 1、 Windows Server 2016、 Windows Server 2012 R2、 Windows Server 2012 或 Windows Server 2008 R2
  
或
  
Mac OS X 10.10
  
若嘗試在行動裝置或 Chromebook 上安裝 Office 時收到此訊息，請參閱[Set up Office 應用程式和行動裝置上的電子郵件](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx)。
