# <a name="entity-type-not-showing-in-search-results"></a>未顯示在搜尋結果中的實體類型

若要顯示在 Dynamics 365 整合介面中的已分類搜尋] 或 [相關性搜尋中的實體類型，它必須先新增至應用程式模組。

附註： 搜尋的相關性搜尋中的附件時，附件實體必須新增至應用程式模組即使父記錄會顯示在結果中。
