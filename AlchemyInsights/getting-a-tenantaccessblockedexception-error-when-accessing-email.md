# <a name="getting-a-tenantaccessblockedexception-error-when-accessing-email"></a>存取電子郵件時，請取得 TenantAccessBlockedException 錯誤？

有幾個不同的原因為何您可能會收到此錯誤。 例如：

- 確認您的訂閱是適用於 live @**帳單** \> **訂閱**] 頁面上，在[Microsoft 365 系統管理中心](https://portal.office.com/adminportal/home#/subscriptions)。

- 檢查是否有任何逾期中的 **[帳單**通知\>**帳單**] 頁面上，在[Microsoft 365 系統管理中心](https://portal.office.com/adminportal/home#/billoverview)。

- 帳務和訂閱帳戶最近已更新，那麼就會花時間，變更才會傳播。 再試一次在幾分鐘，以及如果問題仍然存在，請與我們連絡。
