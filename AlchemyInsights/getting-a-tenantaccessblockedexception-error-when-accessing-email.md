# <a name="getting-a-tenantaccessblockedexception-error-when-accessing-email"></a>存取電子郵件時取得 TenantAccessBlockedException 錯誤？

有幾個不同的原因可能會收到此錯誤。 例如：

- 請確認您的訂閱在 Microsoft 365 系統管理中心的 [ **帳單** \> **[產品](https://portal.office.com/adminportal/home#/subscriptions)** ] 頁面上有效。

- 在 Microsoft 365 系統管理中心的 [ **帳單**帳單 \> **[& 付款](https://portal.office.com/adminportal/home#/billoverview)** ] 頁面上檢查是否有任何過去到期的通知。

- 如果您帳戶的帳單和訂閱最近更新最近更新，可能需要一些時間才能傳播變更。 請在幾分鐘後再試一次，如果問題仍然存在，請聯繫我們。
