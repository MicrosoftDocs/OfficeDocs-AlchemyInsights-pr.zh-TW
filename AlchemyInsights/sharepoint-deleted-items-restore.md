# <a name="restore-items-deleted-from-sharepoint"></a>還原 SharePoint 中刪除的項目

當您刪除項目或網站從 Sharepoint，它不&rsquo;t 立即移除。 已刪除的項目移至資源回收筒一段時間。 在這段時間，您可以還原已刪除的項目至其原始位置。 如需詳細資訊，請瀏覽下列連結。</p> <p><strong>從資源回收筒還原已刪除的檔案、 資料夾及網站。</strong></p> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b?ui=en-US&amp;rs=en-US&amp;ad=US">還原 SharePoint 網站資源回收筒中的項目</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/Restore-deleted-files-or-folders-in-OneDrive-949ada80-0026-4db3-a953-c99083e6a84f">還原已刪除的檔案或 OneDrive 中的資料夾</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/sharepoint/restore-deleted-site-collection">還原已刪除的網站集合 （包括群組、 通訊以及其他網站）</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/en-us/onedrive/restore-deleted-onedrive">還原已刪除的 OneDrive 網站</a></li> </ul> <p>大量資源回收筒] 動作，系統管理員可能會考慮使用<a href="https://docs.microsoft.com/en-us/powershell/sharepoint/sharepoint-pnp/sharepoint-pnp-cmdlets?view=sharepoint-ps">Sharepoint Online PNP。</a></p> <p><strong>檔案還原功能</strong></p> <p>如果您的 OneDrive 或 Sharepoint 檔案大量取得刪除、 覆寫、 損毀，或感染惡意程式碼，您可以還原您整個 OneDrive 或 Sharepoint 文件庫至前一次&nbsp;使用檔案還原功能。</p> <ul> <li><a href="https://support.office.com/en-us/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15">還原 OneDrive 文件庫</a></li> <li><a href="https://support.office.com/en-us/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a?ui=en-US&amp;rs=en-US&amp;ad=US">還原的文件庫</a></li> </ul> <p><strong>附註：</strong>SharePoint Online 會保留所有內容的備份實際刪除超過 14 其他天。 如果無法透過資源回收筒還原的內容或檔案還原時，系統管理員可以連絡 Microsoft 支援服務要求還原 14 天的期間內的任何時間。&nbsp;從備份還原，才可完成的網站集合或子網站，不適用於特定的檔案、 清單或文件庫。</p>
