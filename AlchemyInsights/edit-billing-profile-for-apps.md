# <a name="edit-billing-profile-for-apps"></a>編輯應用程式的帳單設定檔

## <a name="to-change-the-billing-profile-on-apps"></a>若要變更應用程式上的帳單設定檔

1. 移至**帳單** > [產品 & 服務](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面上，然後選擇 [**應用程式**] 索引標籤。

2. 尋找並選取您想要管理的應用程式。  

3. 旁邊**帳單設定檔**，選取 [**編輯**]，然後選擇您想要與應用程式建立關聯的帳單設定檔。

## <a name="edit-billing-profiles"></a>編輯帳單的設定檔

1. 移至**帳單** > [帳單 & 付款](https://go.microsoft.com/fwlink/p/?linkid=848039)] 頁面上，然後選擇 [**計費設定檔**。

2. 從編輯清單中選擇帳單的設定檔。
