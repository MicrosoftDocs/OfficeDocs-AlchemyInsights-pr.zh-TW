# <a name="were-here-to-help"></a>我們隨時為您效勞

請按一下下方的**連絡支援**，以向我們提出支票證。 支援專員將盡快回覆，以協助您解決問題。

如果您的訂閱由 Microsoft 合作夥伴管理，請與您的合作夥伴合作，以開啟支援票證。
