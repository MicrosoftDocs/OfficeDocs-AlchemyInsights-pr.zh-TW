# <a name="create-a-folder"></a>建立資料夾

1. 在 [資料夾] 窗格底部，選取**新的資料夾**。
2. 在新的 [資料夾] 方塊中，輸入資料夾名稱，然後按 Enter 鍵。

若要深入了解，請參閱 < <b0>Create Outlook.com 中的資料夾</b0>。
