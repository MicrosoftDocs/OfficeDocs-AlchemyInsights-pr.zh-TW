# <a name="create-a-folder"></a>建立資料夾

1. 在資料夾窗格的底部, 選取 [**新增資料夾**]。 您可能需要向下移動資料夾窗格以查看 [**新增資料夾**] 按鈕。
2. 在文字方塊中, 輸入資料夾的名稱, 然後按 Enter。

若要深入瞭解, 請參閱[在 Outlook.com 中建立資料夾](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)。
