# <a name="insufficient-user-permissions"></a>使用者權限不足

不符合[必要條件](https://docs.microsoft.com/configmgr/tenant-attach/device-sync-actions#prerequisites)時，就會發生此問題。 請確認使用者在 [設定管理員] 的 [集合]**** 物件類別底下，具有 [通知資源]**** 權限。
