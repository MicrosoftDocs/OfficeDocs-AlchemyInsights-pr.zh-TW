# <a name="cancel-project-subscription"></a>取消專案訂閱

系統管理員可在 Microsoft 365 系統管理中心取消其組織的專案訂閱。

1. 移至 [ **帳單** \> [產品](https://go.microsoft.com/fwlink/p/?linkid=842054) ] 頁面。
2. 在 [ **產品** ] 索引標籤上，尋找您要取消的專案訂閱。 選取 [ **更多動作** (三點) ]，然後選取 [ **取消訂閱**]。
3. 在 [ **取消訂閱** ] 窗格中，選擇您要取消的原因。 （選用）提供任何意見反應。
4. 選取 **[儲存]**。

**退款：** 在下一個計費週期內，任何按比例支付的信用都會傳回給您。

若要深入瞭解，請參閱 [取消您的訂閱](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription)。
