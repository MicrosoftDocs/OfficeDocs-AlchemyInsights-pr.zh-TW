# <a name="cancel-project-subscription"></a>取消專案訂閱

系統管理員可以從**Microsoft 365 系統管理中心**取消其組織的專案訂閱。 

1. 選取 [**帳單**] 選項。

2. 按一下 [[產品 & 服務](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/subscriptions)]，以查看組織的所有訂閱。

3. 找出專案計劃以取消。

4. 按一下 [**取消訂閱**]。
