# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>設定轉送至多個電子郵件地址

如果您需要設定電子郵件轉寄給使用者群組, 請建立通訊群組清單、將多個使用者新增至該清單, 然後設定轉寄至該清單。 首先, 流覽至 [[管理群組](https://portal.office.com/adminportal/home#/groups)]
  
1. 建立*通訊群組清單*

2. 將多個使用者新增至其中

3. 然後設定信箱轉寄至該群組
