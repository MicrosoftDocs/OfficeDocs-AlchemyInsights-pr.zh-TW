# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>設定轉寄至多個電子郵件地址

如果您需要設定電子郵件轉寄給一群使用者，建立通訊群組清單、 新增多位使用者至它，然後設定 [轉寄至該清單。 首先，瀏覽至 [[管理群組](https://portal.office.com/adminportal/home#/groups)
  
1. 建立*通訊群組清單* 
    
2. 多個使用者加入
    
3. 然後設定信箱轉寄給該群組
