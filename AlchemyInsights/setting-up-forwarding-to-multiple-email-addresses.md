# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>設定轉接至多個電子郵件地址

首先，瀏覽至 [[管理群組](https://portal.office.com/adminportal/home#/groups)
  
1. 建立*通訊群組清單* 
    
2. 加入多位使用者
    
3. 然後設定信箱轉寄給該群組
