# <a name="in-app-support-in-outlook-for-windows"></a>適用于 Windows 的 Outlook 中的應用程式內支援

作為租使用者系統管理員，您可以在適用于 Windows 的 Outlook 中收到問題的應用程式支援。 在 Outlook 中，**選取 [** 說明] 以  >  **支援人員**。 請輸入您的問題，且 Outlook for Windows 支援小組成員將會回復您的郵件。
