# <a name="sharing-options-appear-dim-or-are-not-available"></a>共用選項會變暗或無法使用

如果嘗試從 SharePoint Online 或 OneDrive 共用時，有些共用選項或無法使用，請參閱 [當您從線上 SharePoint 線上或 OneDrive 時，共用選項會變灰](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive)。
