# <a name="changing-public-folder-permissions"></a>變更公用資料夾權限

公用資料夾權限可由使用者與系統管理員在 Outlook 中的變更。系統管理員也可以控制權限從 Exchange 系統管理中心 (EAC)，執行下列動作：
  
1. 在 O365 入口網站中使用左邊瀏覽] 功能表中，瀏覽至**管理中心** \> **Exchange**。
    
2. 瀏覽至 [**公用資料夾**。
    
3. 如此，您可以變更個別公用資料夾的權限指派給權限的安全性群組。若要變更公用資料夾權限的使用者，如使用者必須具備擁有者權限的資料夾。
