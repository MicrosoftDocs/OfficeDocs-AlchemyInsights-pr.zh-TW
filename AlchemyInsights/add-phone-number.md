# <a name="add-phone-number"></a>新增電話號碼

您必須先針對貴組織的使用者[取得電話號碼](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview)，才能將其設定為撥打和接聽電話。 我們已提供完成電話系統設定的逐步程序。 請檢閱[設定組織的電話系統](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview)。
