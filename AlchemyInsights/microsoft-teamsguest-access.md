# <a name="microsoft-teams---guest-access"></a>Microsoft 團隊-來賓存取

您需要在 Microsoft 小組中協助您開啟和設定來賓存取功能嗎？

以下是您需要執行的動作:

1. 授權來賓存取。
1. 將來賓使用者新增至您的小組。
1. 設定來賓使用者權限。

如需詳細資訊 (包括操作方法影片), 請參閱[Microsoft 團隊中的來賓存取](https://docs.microsoft.com/microsoftteams/guest-access)。
