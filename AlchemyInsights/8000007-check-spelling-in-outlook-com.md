# <a name="use-your-browser-to-check-spelling"></a>使用瀏覽器自動拼字檢查

在 Outlook.com 中的沒有拼字檢查選項。若要檢查在拼字檢查，您需要使用網頁瀏覽器中。使用 Microsoft Edge、 Internet Explorer 10 及更新版本、 Firefox、 Chrome 和 Safari 的目前版本中拼字檢查。檢查網頁瀏覽器中若要深入了解如何檢查拼字檢查的選項。
