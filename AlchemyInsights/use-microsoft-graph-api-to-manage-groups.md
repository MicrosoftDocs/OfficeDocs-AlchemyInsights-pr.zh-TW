# <a name="use-microsoft-graph-api-to-manage-groups"></a>使用 Microsoft Graph API 管理群組

群組是使用者和其他主要使用者的集合，可在 Microsoft 服務或您的應用程式中共用資源的存取權。 Microsoft Graph 提供 APIs，您可以根據您的案例，用來建立及管理不同類型的群組和群組功能。 Microsoft Graph 中的所有群組相關作業都需要管理員同意。 如需詳細資訊，請參閱使用 [Microsoft Graph 中的群組](https://docs.microsoft.com/graph/api/resources/groups-overview)。
