# <a name="how-to-select-all-in-outlookcom"></a>如何在 「 選取所有 」 在 Outlook.com 中

您只能選取所有如果是可見的所有郵件。 這是要避免意外動作無法看到的訊息。

**CTRL + 按一下**會選取多個個別的郵件。

**SHIFT + 按一下 [** 選取您選取的第一個及最後一個訊息之間的所有郵件。
