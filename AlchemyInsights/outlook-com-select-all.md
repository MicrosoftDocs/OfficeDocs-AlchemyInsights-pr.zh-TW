# <a name="how-to-select-all-in-outlookcom"></a>操作方法 Outlook.com 中的「全選」

只要所有的郵件都是可見的，您就可以選取 [全部]。 這是為了避免對您看不到的郵件執行意外動作。

**按住 CTRL 鍵並按一下**選取多個個別的郵件。

**SHIFT + 按一下**選取選取的第一和最後一封郵件之間的所有郵件。
