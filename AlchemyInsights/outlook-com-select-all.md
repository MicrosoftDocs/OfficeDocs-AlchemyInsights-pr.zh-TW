# <a name="how-to-select-all-in-outlookcom"></a>如何在 Outlook.com 中 "全選"

如果所有郵件都是可見的, 您就只能選取 [全部]。 這是為了防止對您無法看到的郵件執行意外動作。

**按住 CTRL 鍵並按一下**會選取多個個別的郵件。

**按住 SHIFT 鍵並按一下**選取第一個及最後一封郵件之間的所有郵件。
