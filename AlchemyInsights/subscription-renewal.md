# <a name="subscription-recurring-billing"></a>訂閱週期性計費

若要了解當您每月或每年的訂閱 （如果已開啟**週期性計費**） 將計費或過期 （如果**週期性帳務**已關閉）： 
  
1. 在管理中心中，移至 [[訂閱](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面或選擇 [**計費** \> **訂閱**。
    
2. 選擇您想要更了解訂閱。
    
3. 在右側之訂閱卡是訂閱狀態、 [**週期性計費**] 設定及後續的帳單到期日期相關資訊。 
    
4. 若要繼續使用超過此日期服務，切換**週期性帳務設定**切換至**上**。狀態會顯示訂閱的下一個付款日期。
    
5. 若要停止服務於此日期，切換**週期性帳務**切換至**關閉**。狀態會顯示訂閱到期日期。
