# <a name="learn-how-microsoft-edge-supports-windows-information-protection-wip"></a>瞭解 Microsoft Edge 如何支援 Windows 資訊保護 (WIP) 

隨著遠端工作者數目的增加，在公司裝置上所發生的個人和工作活動的數目也會隨之增加。 這進而會增加公司資料可能會在工作場所以外共用的風險。 為了降低此風險，Microsoft Edge (版本81或更新版本) 支援 Windows 資訊保護 (WIP) ：一種 Windows-10 功能，可協助保護企業資料，避免未經授權和意外的披露。
