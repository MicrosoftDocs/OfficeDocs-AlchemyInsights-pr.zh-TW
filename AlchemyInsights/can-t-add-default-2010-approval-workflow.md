# <a name="cant-add-default-2010-approval-workflow"></a>不能新增預設 2010年核准工作流程

在 Microsoft SharePoint 網站集合，您無法新增至清單或文件庫的全域可重複使用的工作流程 （例如 「 核准-SharePoint 2010） 」。核准工作流程可能會遺失。若要解決此問題，請參閱[無法](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library)新增至 SharePoint 清單或文件庫的全域可重複使用的工作流程。
