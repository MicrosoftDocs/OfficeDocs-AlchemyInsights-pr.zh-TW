# <a name="cant-add-default-2010-approval-workflow"></a>無法新增預設 2010年核准工作流程

在 [Microsoft SharePoint 網站集合，您無法新增至清單或文件庫的全域可重複使用的工作流程 （例如 「 核准-SharePoint 2010） 」。 核准工作流程可能會遺失。 若要解決此問題，請參閱[不能](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library)新增至 SharePoint 清單或文件庫的全域可重複使用工作流程。 

如需工作流程核准的詳細資訊，請參閱 [所有關於核准工作流程] (https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742)。 
 
深入了解 [流程中的新式核准] (https://flow.microsoft.com/blog/introducing-modern-approvals)。
