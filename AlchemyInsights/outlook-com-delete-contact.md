# <a name="delete-a-contact"></a>刪除連絡人

1. 在左下角的 [] 頁面上，選取 [**人員**] 圖示。
2. 選取您想要刪除，然後選取 [**刪除**的連絡人。

如果無法選取 [**刪除**]，連絡人可能是來自 Skype 或已連線的社交網路帳戶。 若要刪除的連絡人，請移至 Skype 或帳戶連絡人所在的位置。
