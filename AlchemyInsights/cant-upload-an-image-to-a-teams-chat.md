# <a name="cant-upload-an-image-to-a-teams-chat"></a>無法將影像上傳到小組聊天

疑難排解的建議：

- 如果您正在使用網頁上的團隊，請嘗試其他瀏覽器。 如需支援的瀏覽器清單，請參閱 [Microsoft 小組之限制與規格](https://docs.microsoft.com/microsoftteams/limits-specifications-teams)的「瀏覽器」一節。
- 檢查硬體和作業系統的相容性。 如需詳細資訊，請參閱 [Microsoft 小組的硬體需求](https://docs.microsoft.com/microsoftteams/hardware-requirements-for-the-teams-app)。
- 檢查您是否有最新的小組版本。 如需詳細資訊，請參閱 [小組更新](https://docs.microsoft.com/microsoftteams/teams-client-update) 程式以取得詳細資訊。
