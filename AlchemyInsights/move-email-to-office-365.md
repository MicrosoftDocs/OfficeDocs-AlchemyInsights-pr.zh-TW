# <a name="move-email-to-microsoft-365"></a>將電子郵件移至 Microsoft 365

您有數個選項可將電子郵件遷移至 Microsoft 365。 [Microsoft 365 mail 遷移顧問](https://aka.ms/alchemyinsight-mailmigrationadvisor)可協助您為您的環境選擇最佳的功能，也能引導您完成郵件遷移。
  
若要閱讀您的選項但不啟動遷移，請參閱 [決定遷移路徑](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path)。

請另外檢查 [ [採用 Microsoft 365 with FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) ] 頁面，以取得遷移的輔助協助（如有資格）。
