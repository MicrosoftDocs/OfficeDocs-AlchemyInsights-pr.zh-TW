# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>取得在應用程式支援的 Outlook 行動裝置應用程式

1. 在 Outlook 行動裝置應用程式中，開啟**功能表**。
    
2. 選取 [開啟問號**協助&amp;意見反應**畫面。 
    
3. 點選 [**連絡人支援**和說明問題。 
    
4. 點選 [**傳送**和您會收到的應用程式通知支援代理程式的連絡人您時。 
    
如需詳細指示，請參閱[取得 iOS 及 Android outlook 的應用程式說明](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support)。
