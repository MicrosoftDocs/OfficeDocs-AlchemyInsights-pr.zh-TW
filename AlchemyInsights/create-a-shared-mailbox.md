# <a name="create-a-shared-mailbox"></a>建立共用信箱

共用的信箱可讓多人傳送及接收電子郵件別名，類似**sales@contoso.com**，而不指定其他的授權。
  
如需詳細資訊，請參閱[建立共用信箱](https://support.office.com/client/871a246d-3acd-4bba-948e-5de8be0544c9)。
