# <a name="managed-metadata-service-error-message"></a>Managed Metadata Service 錯誤訊息

當您開啟術語存放區時, 您可能會看到錯誤訊息: 「Managed Metadata Service 或連線目前無法使用。 可能尚未啟動應用程式集區或受管理的中繼資料 Web 服務。 請與您的系統管理員聯繫。
  
如果在您第一次看到此問題後超過24小時, 請記錄支援案例。 我們瞭解24小時似乎很長的時間。 在許多情況下, 我們已經在處理解決方案, 所以請至少提供24小時。
