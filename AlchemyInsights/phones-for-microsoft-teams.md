# <a name="are-you-looking-for-phones-that-are-compatible-with-teams"></a>您在尋找與團隊相容的電話嗎？

身為系統管理員，您可以從 Microsoft 365 系統管理中心管理組織中的團隊所使用的 [所有裝置](https://docs.microsoft.com/microsoftteams/device-management) 。 如需詳細資訊： 

- 查看 [Microsoft 小組的電話](https://docs.microsoft.com/microsoftteams/phones-for-teams)清單。 
- 查看 [目前已驗證的 IP 電話](https://docs.microsoft.com/microsoftteams/teams-ip-phones#currently-certified-ip-phones)清單。 
- 查看[具有已知限制和問題的裝置](https://support.office.com/article/control-calls-using-a-headset-in-teams-65d6e104-444d-4013-b8c2-f11317dd69a8)清單。
