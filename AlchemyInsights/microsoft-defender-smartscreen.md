# <a name="microsoft-defender-smartscreen"></a>Microsoft Defender Smartscreen

當您使用新的 Microsoft Edge 時，Microsoft Defender SmartScreen 可協助您辨識已報告的網路釣魚和惡意程式網站，並針對下載做出明智決定。

若要深入了解，請參閱[何謂 SmartScreen？此工具如何保護我？](https://support.microsoft.com/microsoft-edge/what-is-smartscreen-and-how-can-it-help-protect-me-1c9a874a-6826-be5e-45b1-67fa445a74c8)。
