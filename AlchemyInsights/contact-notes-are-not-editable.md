# <a name="cant-edit-the-notes-field-for-a-contact-in-outlook"></a>無法編輯 Outlook 中連絡人的 [附注] 欄位

此為已知問題。 我們正在努力修正它, 您可以在 [[無法編輯連絡人的附注] 欄位](https://support.office.com/article/fb8394ce-04ce-48b5-bae4-be46f77f10fe)中看到最新的狀態。

**解決方法**: 如果您使用的是 Office 365 或 Outlook.com 帳戶, 您可以使用 web 來編輯您的連絡人。 如果您使用的是 IMAP 或 POP 帳戶, 目前沒有可用的解決方法。
