# <a name="add-a-guest-to-microsoft-teams"></a>新增來賓至 Microsoft 團隊

有時候，您可能想要與您的小組組織中的人員進行共同作業。 您通常會在電子郵件中執行這項作業，但是如果您想要在小組中與他們共同作業，您可以將他們新增為來賓。

如需詳細資訊，請參閱 [將客人新增至小組中的團隊](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop)。

**附注：** 您必須是小組擁有人，才可在小組中新增來賓 (也請參閱 [Set guest 許可權](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)) 。
