# <a name="my-status-always-shows-as-available"></a>我的狀態永遠顯示為可用

每次登入 Outlook.com 時，您的帳戶狀態都會重設為 [可用]。 您可以防止您的狀態因未登出 Outlook.com 而變更。 若要在關閉瀏覽器時保持登入，請選取 [Outlook.com 登入] 頁面上的 [ **保留我已登入** ] 方塊。
