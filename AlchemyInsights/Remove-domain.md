# <a name="trying-to-remove-your-domain"></a>嘗試移除您的網域嗎？

遵循[移除來自 Office 365 的網域](https://support.office.com/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx)] 中的步驟。
  
您必須更新任何使用者或群組指派給該網域名稱，然後再移除網域。檢查[網域常見問題集](https://support.office.com/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx)的其他常用的解決方案。
