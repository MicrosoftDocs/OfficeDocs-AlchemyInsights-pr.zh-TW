# <a name="trying-to-remove-your-domain"></a>您正試著要移除網域嗎？

請遵照[移除網域](https://docs.microsoft.com/office365/admin/get-help-with-domains/remove-a-domain)中的步驟進行。
  
您將必須在移除網域之前，更新任何指派至該網域名稱的使用者或群組。 如需其他常見解決方案，請查看[網域常見問答集](https://docs.microsoft.com/office365/admin/setup/domains-faq)。
