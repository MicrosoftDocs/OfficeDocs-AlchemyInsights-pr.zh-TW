# <a name="not-receiving-facebook-codes"></a>不接收 Facebook 代碼

如果您要求來自 Facebook 的新密碼, 但未收到您的密碼重設電子郵件:

- 檢查 [[垃圾郵件] 或 [垃圾郵件] 資料夾](https://outlook.live.com/mail/junkemail)。
- 嘗試 [再次重設 Facebook 密碼](https://www.facebook.com/help/213395615347144?helpref=faq_content)。

如果您無法重設您的密碼, 因為您無法存取您帳戶上的電子郵件或行動電話號碼, 請嘗試使用[Facebook 的密碼重設提示](https://www.facebook.com/help/218815984812734)。
