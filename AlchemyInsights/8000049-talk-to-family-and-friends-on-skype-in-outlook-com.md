# <a name="talk-to-family-and-friends-on-skype-in-outlookcom"></a>在 Outlook.com 中 Skype 詢問朋友和家人

Outlook.com 客戶可以與朋友和家人與聊天室、 語音及視訊功能 Skype 對話。當您[合併 Microsoft 和 Skype 帳戶](https://go.microsoft.com/fwlink/p/?linkid=2001101&amp;clcid=0x409)，您自動可以存取 Skype Outlook.com 中的 web。
  
深入了解如何[在 Outlook.com 中 Skype 朋友和家人與對話](https://go.microsoft.com/fwlink/p/?linkid=2001407&amp;clcid=0x409)。
