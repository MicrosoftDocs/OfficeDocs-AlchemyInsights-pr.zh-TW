# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>在網頁上的 Outlook 中取得帳戶停用的錯誤？

如果您收到錯誤**Your account has been disabled** `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` ，當您嘗試在網頁上開啟 outlook (之前稱為 outlook web App) 時，您的帳戶已經停用，您的組織管理員可能已停用您存取網頁上的 outlook。 您的系統管理員可以遵循 [啟用或停用信箱的 Outlook Web App](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)主題中的步驟，修正此錯誤。
