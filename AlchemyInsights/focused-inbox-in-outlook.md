# <a name="focused-inbox-in-outlook"></a>Outlook 中的焦點收件匣

焦點收件匣會將您的收件匣分割為兩個索引標籤。 您最重要的訊息是在 [焦點] 索引標籤上, 而 rest 仍然可在 [其他] 索引標籤上輕鬆存取。
