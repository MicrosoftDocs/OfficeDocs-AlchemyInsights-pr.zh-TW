# <a name="setup-email-accounts"></a>設定電子郵件帳戶

1. 在 Outlook 中, **** \>選取 [檔案] [**新增帳戶**]。

2. 輸入您的電子郵件地址, 然後按一下 **[連接]**。

3. 輸入您的密碼, 然後按一下 **[確定]**。

4. 您可以重複這些步驟來新增多個帳戶。

    > [!NOTE]
    > 某些電子郵件帳戶 (如 Gmail、Yahoo 和 iCloud) 需要您設定兩個要素驗證, 才能使用 Outlook 傳送和接收電子郵件。 如需詳細資訊, 請參閱[將電子郵件帳戶新增至 Outlook](https://support.office.com/article/6e27792a-9267-4aa4-8bb6-c84ef146101b.aspx)。
