# <a name="share-your-videos-in-microsoft-stream"></a>共用您在 Microsoft 資料流的視訊

您可能想要與否則沒有存取權的人員共用 Microsoft Stream 中的影片。 有各種方式共用您的視訊： 

- [共用您在 Microsoft 資料流的視訊](https://docs.microsoft.com/stream/portal-share-video)

- [外部共用 Microsoft Stream 中](https://docs.microsoft.com/stream/portal-share-video#external-sharing)

- [視訊的權限](https://docs.microsoft.com/stream/portal-share-video#permissions-on-your-video)
