# <a name="administrative-units"></a>管理單位

系統管理單位是 Azure AD 資源，可以是其他 Azure AD 資源的容器。 系統管理單位只能包含使用者和群組。

如需深入了解管理單位，請參閱 [Azure Active Directory 管理單位](/azure/active-directory/roles/administrative-units)。
