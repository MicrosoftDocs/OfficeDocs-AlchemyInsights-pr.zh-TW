# <a name="import-contacts"></a>匯入連絡人

之前將連絡人匯入 outlook 網頁版，您需要將它們匯出從其他電子郵件提供者為.csv 檔案。 然後，遵循下列指示。

1. 在頁面底部，選取 [移至 [人員] 頁面上的 [人員] 圖示。
2. 在工具列上，選取 [**管理** > **匯入連絡人]**。
3. 選取 [**瀏覽]**，選擇您的 CSV 檔案，然後選取 [**開啟**。
4. 選取 [**匯入**。

若要深入了解，請參閱 <<c0>將連絡人匯入至 Outlook。
