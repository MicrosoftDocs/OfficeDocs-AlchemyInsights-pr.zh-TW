# <a name="get-started-with-remotely-debugging-windows-10-devices"></a>立即開始使用 Windows 10 裝置的遠端偵錯

若要從 Windows 或 macOS 電腦從 windows 10 裝置遠端偵錯即時內容，請執行下列動作：

1. 針對遠端偵錯設定 Windows 10 裝置，然後從開發電腦連線到該裝置。
2. 在您的開發電腦上檢查和調試您 Windows 10 裝置上的即時內容。
3. 將 Windows 10 裝置的內容 Screencast 至開發電腦上的 DevTools 實例。

若要深入瞭解，請參閱 [使用 Microsoft Edge 從遠端偵錯 Windows 10 裝置開始](https://go.microsoft.com/fwlink/?linkid=2142172)使用。
