# <a name="how-to-install-office-on-an-ipad"></a>如何在 iPad 上安裝 Office

- 讓使用者點擊其行動裝置[的 [下載] 連結](https://support.office.com/article/9df6d10c-7281-4671-8666-6ca8e339b628?wt.mc_id=Alchemy_ClientDIA)，以供 Word、Excel、PowerPoint 和 OneNote 使用。

- 讓使用者啟動應用程式，請點擊 [登**入**]，然後輸入他們的工作或學校認證。
