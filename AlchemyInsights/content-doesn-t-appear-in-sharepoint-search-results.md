2. <span data-ttu-id="fde26-p102">檢查**清單**或**文件庫**包含預期的內容設定為允許會出現在搜尋結果中的內容。請遵循中[顯示內容從清單或文件庫在搜尋結果中](https://docs.microsoft.com/sharepoint/make-site-content-searchable#show-content-from-lists-or-libraries-in-search-results)的步驟。</span><span class="sxs-lookup"><span data-stu-id="fde26-p102">Check that the **list** or **library** that contains the expected content is set to allow content to appear in search results. 4. <span data-ttu-id="fde26-p104">確認使用者具有**權限**檢視內容。請遵循[了解在 SharePoint 中的權限層級](https://go.microsoft.com/fwlink/?linkid=867071)中的步驟。</span><span class="sxs-lookup"><span data-stu-id="fde26-p104">Verify that the user has **permissions** to view the content.
