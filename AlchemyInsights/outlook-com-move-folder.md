# <a name="move-a-folder"></a>移動資料夾

您可以拖曳下重新排列其 **[我的最愛]** 資料夾。
  
您無法重新排序**資料夾**下的資料夾，或移動成為資料夾的子資料夾。

*附註：* 如果您將資料夾移至成為子資料夾中，建立新的資料夾，並從子資料夾中所有的電子郵件複製到新的資料夾。
  
若要深入了解，請參閱[Working with Outlook.com 中的郵件資料夾](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a)。
