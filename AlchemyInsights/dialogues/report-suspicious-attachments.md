# <a name="report-suspicious-attachments"></a>報告可疑的附件

1. 在 **新的提交** 快顯視窗中，選取 [ **附件**]，然後上傳您想要掃描的檔案。
    > [!NOTE]
    > 此時間限制為 3 MB。 如果檔案超過 3 MB，請考慮改為使用電子郵件識別碼加以提交。
2. 如果您選取 [ **應該已經封鎖**，它會自動將專案標記為 **惡意** 代碼]。
3. 選取 **[提交]**。
