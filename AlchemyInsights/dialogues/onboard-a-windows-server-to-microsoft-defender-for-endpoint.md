# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>將 Windows server 上架至 Microsoft Defender for Endpoint

針對 Windows Server 作業系統，Microsoft Defender for Endpoint 會提供高級功能來偵測和調查攻擊。 這些功能是透過 Microsoft Defender Security Center 主控台以無縫提供。

Defender for Endpoint 支援下列 Windows Server 版本：

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (SAC) 版本1803和更新版本
- Windows Server 2019 和更新版本
- Windows Server 2019 core edition

若要深入瞭解，請參閱 [適用于 Microsoft Defender For Endpoint service 的板載 Windows server](https://go.microsoft.com/fwlink/?linkid=2143627)。
