# <a name="remove-a-hub-site"></a>移除中樞站台

取消登錄新的 SharePoint 系統管理中心的中樞站台
  
1. 在新的 SharePoint 系統管理中心的 [**網站**]，按一下 [**作用中的網站**。 
    
2. [選取站台、 [**中樞站台**] 和 [**取消註冊為中樞站台**。 
    
3. 按一下 [確定]****。 
    
[深入了解中樞站台](https://support.office.com/en-us/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
