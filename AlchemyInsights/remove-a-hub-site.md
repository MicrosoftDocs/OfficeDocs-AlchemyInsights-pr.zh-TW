# <a name="remove-a-hub-site"></a>移除中樞網站

在新的 SharePoint 系統管理中心內登出中樞網站
  
1. 在新的 SharePoint 系統管理中心的 [**網站**] 下, 按一下 [**即時網頁**]。

2. 選取網站, 按一下 [ **Hub site**], 然後按一下 [**以 Hub 網站登出**]。

3. 按一下 [確定]****。

[深入瞭解中樞網站](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
