# <a name="remove-a-hub-site"></a>移除中樞站台

取消登錄中新的 SharePoint 系統管理中心中樞站台
  
1. 在新的 SharePoint 系統管理中心，在 [**網站**] 下按一下 [**作用中網站**]。 
    
2. 選取 [網站]、 按一下**中樞站台**]，然後按一下 [**取消註冊為中樞站台**。 
    
3. 按一下 [確定]****。 
    
[深入了解中樞站台](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
