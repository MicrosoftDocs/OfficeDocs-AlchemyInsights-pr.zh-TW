# <a name="change-billing-country-or-region"></a>變更帳單的國家或地區

您無法在 Microsoft 365 系統管理中心的 **計費**  >  **帳單帳戶** 中變更國家/地區。 您必須註冊新帳戶，然後選擇所需的國家/地區。
