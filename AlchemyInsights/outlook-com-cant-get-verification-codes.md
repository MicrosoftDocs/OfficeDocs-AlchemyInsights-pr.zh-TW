# <a name="cant-get-verification-codes"></a>無法取得驗證碼

如果您無法存取您的替代電子郵件或電話號碼, 請移至您的 Microsoft 帳戶[安全性基本](https://account.microsoft.com/security)版頁面上, 選取 [我在登入頁面上**沒有任何這些**], 並依照指示取代您的安全性資訊。

*附注:*[Microsoft 驗證者應用程式](https://go.microsoft.com/fwlink/?linkid=2016117)不需要網際網路連線, 就能提供您的驗證碼。

若要深入瞭解, 請參閱[Microsoft 帳戶安全性資訊和驗證碼](https://support.microsoft.com/help/12428/)。
