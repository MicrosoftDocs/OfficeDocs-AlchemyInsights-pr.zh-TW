# <a name="cant-get-verification-codes"></a>無法取得驗證代碼

如果您不能存取您的備用電子郵件或電話號碼，請移至您 Microsoft 帳戶[安全性的基本概念](https://account.microsoft.com/security)] 頁面上，選取 [在登入] 頁面上的 [**我沒有任何安全性**並遵循指示來取代您的安全性資訊。

*附註：*[Microsoft Authenticator app](https://go.microsoft.com/fwlink/?linkid=2016117)不需要網際網路連線來提供您的驗證碼。

若要深入了解，請閱讀[Microsoft 帳戶的安全性資訊] 以及 [驗證代碼](https://support.microsoft.com/help/12428/)。
