# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>了解舊版 Microsoft Edge 原則與 Microsoft Edge 原則之間的對應

舊版 Microsoft Edge 的群組原則 (版本 45 及更早版本) 與行動裝置管理 (MDM) 設定會與 Microsoft Edge 版本 80 的相關原則對應。 如需詳細資訊，請參閱[舊版 Microsoft Edge 與 Microsoft Edge 的原則對應](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge)。
