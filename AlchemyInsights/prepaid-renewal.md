# <a name="prepaid-renewal"></a>預付的更新

1. 在管理中心中，移至 [[訂閱](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面或選擇 [**計費** \> **訂閱**。
    
2. 選擇您想要擴充，然後選擇 [**延伸**的預付的訂閱。
    
3. 選擇 [**產品**金鑰] 選項。（如果這是在零售位置購買的產品金鑰，您可能會有選項可使用其他付款方法來擴充）。
    
4. 輸入產品金鑰。
    
5. 請確定資訊是正確，然後選擇 [**下一步**]。
    
6. 選擇 [擴充現有的授權、 結束日期，請選擇 [**下一步**] 選項按鈕。
