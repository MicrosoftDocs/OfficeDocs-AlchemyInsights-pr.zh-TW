# <a name="issues-while-migrating-data-to-sharepoint-online"></a>問題時將資料移轉到 SharePoint Online

它會出現超過 100TB 的資料移轉至 SharePoint Online。

請透過電話/電子郵件下方按一下 [**新增服務要求**，並保留為 SharePoint 移轉超過 100 TB 在文字方塊中的查詢。

一旦票證已送出，請更新下列資訊： 

- 您的公司名稱和租用戶 URL
- 預估移轉大小
- 想要開始與完成移轉的預計時間
- 說明移轉內容的來源，例如 SharePoint Server、Box、GDrive、檔案共用等。
