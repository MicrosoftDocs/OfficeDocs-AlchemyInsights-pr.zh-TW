# <a name="issues-while-migrating-data-to-sharepoint-online"></a>問題時將資料移轉到 SharePoint Online

它會出現超過 100TB 的資料移轉至 SharePoint Online。 請遵循下列步驟，因此我們可以幫助您儘速。 

1. 選取**新的服務要求**，然後再**新增服務要求**。 
2. 保留的標題與描述為**SharePoint 移轉超過 100 TB**。
3. 一旦票證已送出，請更新下列資訊： 

    - 移轉評估的大小。
    - 當您想要啟動並完成移轉估計。
    - 說明從、 SharePoint Server、] 方塊中，GDrive、 檔案共用，例如，要移轉您的內容等..
