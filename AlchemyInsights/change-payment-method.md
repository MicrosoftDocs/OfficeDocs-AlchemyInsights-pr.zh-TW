# <a name="change-payment-method"></a>變更付款方式

您可以驗證及變更 Microsoft 365 系統管理中心中所設定的支付方式。 選擇您要驗證的訂閱。 為訂閱設定的支付方式會顯示在 [**帳單**] 底下。
  
1. 在系統管理中心，移至 [帳單]**** \> [產品與服務][](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面。

2. 尋找您想要變更付款條件的訂閱。 在支付方式旁，選取 [**編輯**]。

3. 依照畫面上的步驟完成變更。

若要深入瞭解，請參閱[變更適用于商務用 Microsoft 365 的支付方式](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/change-payment-method)
