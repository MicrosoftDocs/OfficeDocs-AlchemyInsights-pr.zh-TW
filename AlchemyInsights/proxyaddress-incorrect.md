# <a name="proxyaddress-incorrect"></a>不正確的 ProxyAddress

當物件同步處理至 Azure AD 時、 Active Directory 中 proxyAddresses 屬性中指定的值進行比較以 Azure AD 規則及 proxyAddresses 屬性中 Azure AD 的填入然後。因此，在 Active Directory 中物件之 proxyAddresses 屬性的值可能不處於 proxyAddresses 屬性的值相同 Azure AD。
  
若要深入了解如何在 proxyaddress 會填入，請參閱[如何在 Azure AD 填入 proxyAddress 屬性](https://support.microsoft.com/en-us/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)。
