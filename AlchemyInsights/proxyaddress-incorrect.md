# <a name="proxyaddress-incorrect"></a>ProxyAddress 不正確

當物件同步處理到 Azure AD 時，Active Directory 中之 proxyAddresses 屬性中指定的值進行比較與 Azure AD 的規則，並再之 proxyAddresses 屬性會填入 Azure AD 中。 因此，在 Active Directory 中物件之 proxyAddresses 屬性的值不可能的 proxyAddresses 屬性中的值相同 Azure AD 中。
  
若要深入了解如何在 proxyaddress 填入，請參閱[proxyAddress 屬性如何在 Azure AD 中填入](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)。
