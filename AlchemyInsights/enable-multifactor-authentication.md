# <a name="enable-multi-factor-authentication"></a>啟用多重要素驗證

若要啟用 Microsoft 365 系統管理中心中的多重要素驗證：

1. 移至**使用者** \> **作用中使用者**頁面。
    
2. 在頁面頂端，選取 [**多重要素驗證**]。 
    
3. 尋找使用者或想要啟用多重要素驗證的使用者。 若要查看所有使用者，您可能需要變更頂端的**多重要素驗證狀態**篩選。
    
4. 選取您想要啟用的使用者旁的核取方塊。
    
5.  權限的使用者資訊] 窗格中，[**快速步驟**，選取 [**啟用**]。 
    
6. 在 [開啟] 對話方塊方塊中，選取 [**啟用多重要素驗證**。
