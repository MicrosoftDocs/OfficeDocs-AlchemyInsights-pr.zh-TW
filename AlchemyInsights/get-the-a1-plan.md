# <a name="get-the-a1-plan"></a>取得 A1 方案

如果您的機構已符合 Microsoft 365 教育版的資格，您可以從 **Microsoft 365 系統管理中心**新增 A1 訂閱。 

1. 移至 [帳單] > [產品與服務]**[](https://go.microsoft.com/fwlink/p/?linkid=868433)**。

2. 使用搜尋方塊來搜尋 "A1"。

3. 選取將使用 A1 方案的使用者數目。

如果您的機構尚未設定 Microsoft 365 教育版，但是您想要免費新增 Office 365，您可以提供您的電子郵件來[開始使用](https://www.microsoft.com/education/products/office)。
