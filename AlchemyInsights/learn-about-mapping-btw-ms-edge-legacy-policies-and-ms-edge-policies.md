# <a name="learn-about-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>深入瞭解 Microsoft Edge 舊版原則與 Microsoft Edge 原則之間的對應

Microsoft Edge 舊版的群組原則 (版本45和更舊版本) 與行動裝置管理 (MDM) 設定會對應至 Microsoft Edge 版本80的相關原則。 若要深入瞭解，請參閱 [microsoft Edge 舊版 To Microsoft edge 原則對應](https://go.microsoft.com/fwlink/?linkid=2141665)。
