# <a name="change-your-language"></a>變更您的語言

1. 在右上角中，選取 [**設定**![設定](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) \> **選項**。
    
2. 選取 [**選項**] 下的 [**一般** \> **區域及時間的時區**。
    
3. 在 [**地區及時間的區域設定**] 選擇您的語言。
    
4. 選取 **[儲存]**。
    
若要深入了解，請參閱[變更時區及 Outlook.com 中的語言設定](https://go.microsoft.com/fwlink/p/?linkid=873132)。
