# <a name="troubleshoot-issues-with-enrolling-macos-devices-in-microsoft-intune"></a>疑難排解問題註冊 Microsoft Intune 中的 macOS 裝置

請先檢閱下列資源以立即解決您的問題。
  
1. 使用[Intune 疑難排解入口網站](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade)來診斷和解決常見註冊失敗。 檢閱[本文件](https://docs.microsoft.com/intune/help-desk-operators)如需詳細資訊。

2. 檢閱[本文件](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)的清單中的每個防止註冊和解決方法的常見錯誤。

3. [了解如何註冊 Microsoft Intune 中的 macOS 裝置](https://docs.microsoft.com/intune/macos-enroll)。

[請查看註冊 Microsoft Intune 中的 macOS 裝置的使用者指南](https://docs.microsoft.com/intune-user-help/enroll-your-device-in-intune-macos-cp)
