# <a name="troubleshoot-issues-with-enrolling-macos-devices-in-microsoft-intune"></a>疑難排解問題註冊中 Microsoft Intune macOS 裝置

請檢閱下面列出現在解決問題的資源。 
  
1. 使用[Intune 疑難排解入口網站](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade)來診斷及解決常見的註冊失敗。請檢閱[本文件](https://docs.microsoft.com/intune/help-desk-operators)如需詳細資訊。 
    
2. 請檢閱[本文件](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)的每個防止註冊和解決方法的常見錯誤的清單。 
    
3. [解如何註冊中 Microsoft Intune macOS 裝置](https://docs.microsoft.com/intune/macos-enroll)。
    
[使用者指南 （英文） 註冊 macOS 裝置的 Microsoft Intune 先取出](https://docs.microsoft.com/intune-user-help/enroll-your-device-in-intune-macos-cp)
