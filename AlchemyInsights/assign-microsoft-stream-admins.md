# <a name="assign-microsoft-stream-admins"></a>指派 Microsoft Stream 系統管理員

若要將 Stream 系統管理員角色指派給使用者，您必須擁有 Office 365 全域系統管理員角色。 如需詳細資訊，請參閱：

- [指派 Microsoft Stream 系統管理員](https://docs.microsoft.com/stream/assign-administrator-user-role)
