# <a name="how-to-view-message-headers-or-message-source"></a>如何查看郵件頭或郵件來源

若要查看電子郵件中的標頭和其他詳細資料, 請選取 [讀取窗格] 上角的三個點, 然後選取 [**查看訊息來源**]。
