# <a name="inbox-not-loading-with-firefox"></a>無法載入與 Firefox 的收件匣

某些 Firefox 瀏覽器中的附加元件可能會防止載入 Outlook.com 收件匣。
  
請造訪 Firefox 支援，以了解如何[停用或移除附加元件](https://support.mozilla.org/kb/disable-or-remove-add-ons)。
