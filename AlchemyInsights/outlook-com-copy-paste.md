# <a name="copy-and-paste-in-outlookcom"></a>複製並貼上 Outlook.com

Outlook.com 中沒有 [複製] 或 [貼上] 按鈕，但您可以使用快速鍵：

**在電腦上：** CTRL+C 若要複製，請按 CTRL + X 進行剪切，然後 CTRL+V 以貼上。

**在 Mac 上：** CMD + C to copy、CMD + X to cut 及 CMD + V 來貼上。
