# <a name="copy-and-paste-in-outlookcom"></a>複製並貼上 Outlook.com

在 Outlook.com 中沒有複製或貼上] 按鈕，但您可以使用鍵盤快速鍵：

**電腦上：** CTRL + C 複製、 剪下、 CTRL + X 和 CTRL + V 來貼上。

**Mac:** CMD + C 複製、 剪下、 CMD + X 和 CMD + V 來貼上。
