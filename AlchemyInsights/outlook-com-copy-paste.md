# <a name="copy-and-paste-in-outlookcom"></a>在 Outlook.com 中複製及貼上

Outlook.com 中沒有 [複製] 或 [貼上] 按鈕, 但是您可以使用鍵盤快速鍵:

**在電腦上:** CTRL + C to copy、CTRL + X to cut 和 CTRL + V 貼上。

**在 Mac 上:** 要複製的 CMD + C、CMD + X to cut, 以及 CMD + V 來貼上。
