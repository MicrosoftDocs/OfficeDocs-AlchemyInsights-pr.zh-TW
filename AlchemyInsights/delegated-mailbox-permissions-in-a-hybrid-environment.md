# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>混合式環境中的委派的信箱權限

根據您已安裝在內部部署組織中的 Exchange 版本，您可能需要採取一些額外的步驟，以設定在內部部署組織與 Office 365 中的 Exchange Online 之間的委派的信箱權限。 如需如何設定委派的權限的詳細資訊，看看[設定 Exchange 以支援混合式部署中的委派的信箱權限](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx)。
