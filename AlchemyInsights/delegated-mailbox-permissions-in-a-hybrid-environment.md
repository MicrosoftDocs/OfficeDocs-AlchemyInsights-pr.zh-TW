# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>在混合環境中的委派的信箱權限

根據您的內部部署組織中安裝的 Exchange 的版本，您可能需要花一些額外的步驟來設定內部部署組織與 Office 365 中的 Exchange Online 之間的委派的信箱權限。如需如何設定委派權限的詳細資訊，請[以支援混合式部署中的委派的信箱權限設定 Exchange](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx)查看。
