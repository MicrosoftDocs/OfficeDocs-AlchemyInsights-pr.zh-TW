# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>混合式環境中的委派信箱許可權

根據您在內部部署組織中安裝的 Exchange 版本而定, 您可能需要執行一些額外的步驟, 才能在 Office 365 的內部部署組織與 Exchange Online 之間設定委派的信箱許可權。 如需如何設定委派許可權的詳細資訊, 請參閱設定[Exchange 在混合部署中支援委派的信箱許可權](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx)。
