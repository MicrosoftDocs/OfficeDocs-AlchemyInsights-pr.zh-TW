# <a name="deploy-office-apps"></a>部署 Office 應用程式

透過 Office 部署工具、System Center Configuration Manager、作為作業系統磁片映射的一部分, 或在使用遠端桌面服務 (RDS) 或終端的 Windows Server 的共用電腦案例中, 將 Office 應用程式部署至多部使用者服務, 請考慮使用**Office 365 專業增強版部署顧問**中的逐步指南。
  
若要開始, 請確定您已以租使用者管理員或全域系統管理員身分登入, 然後按一下 [這裡[https://aka.ms/proplusguidance](https://aka.ms/proplusguidance)]:。
