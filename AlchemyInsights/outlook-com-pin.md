# <a name="change-your-pin"></a>變更您的 pin 碼

Outlook.com 不會使用 pin 碼，但您可能登入您的電腦與 pin 碼或 Windows Hello。 若要變更您在 Windows 10 中的 pin 碼：

1. 選取 [開始] 按鈕或在鍵盤上按下 Windows 鍵，然後選取 [**設定** > **帳戶** > **登入選項**。
2. 選取 [ **pin 碼**的 [**變更**] 按鈕。
3. 輸入您目前的 pin 碼，然後輸入並確認移至新的 pin 碼。
