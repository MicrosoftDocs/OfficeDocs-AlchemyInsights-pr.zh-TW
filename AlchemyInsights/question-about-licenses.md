# <a name="question-about-licenses"></a>關於授權問題

如果您想要知道您已購買、 多少個授權或從訂閱指派多少授權：
  
1. 在管理中心中，移至 [[訂閱](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面或選擇 [**計費** \> **訂閱**。
    
2. 選擇您想要更了解的訂閱。
    
3. 在左側的訂閱卡片是授權的相關資訊：
    
  - **購買**數目是您所支付的授權數。 
    
  - **已指派**的號碼是指派給使用者的授權數。 
    
4. 如果您想要新增更多授權，或者您有太數量，選擇 [**新增/移除授權**指派給調整購買量。
