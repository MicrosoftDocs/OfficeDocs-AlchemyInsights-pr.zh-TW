# <a name="question-about-licenses"></a>有關授權的問題

如果您想要知道您購買的授權數目, 或是指派給訂閱的授權數目:
  
1. 在系統管理中心中，移至 [訂閱][](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面，或選擇 [計費]**** \> [訂閱]****。

2. 選擇您想要深入瞭解的訂閱。

3. 在訂閱卡片的左邊是您授權的相關資訊:

  - [**購買**號碼] 是您要支付的授權數目。

  - [**指派**的號碼] 是指派給使用者的授權數目。

4. 如果您想要新增更多授權, 或如果您有太多授權, 請選擇 [**新增/移除授權**] 以調整所購買的金額。
