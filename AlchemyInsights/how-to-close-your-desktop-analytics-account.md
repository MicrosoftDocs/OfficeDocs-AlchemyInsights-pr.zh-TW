# <a name="how-to-close-your-desktop-analytics-account"></a>如何關閉您的桌面分析帳戶

如果您在環境中，設定桌面分析，並接著，決定您需要移除它，請在下列連結中使用程序：

[如何關閉您的帳戶](https://docs.microsoft.com/sccm/desktop-analytics/account-close)

第一個步驟是連絡 Microsoft 支援服務。 開啟支援案例來關閉您的桌面分析帳戶。 不繼續進行其他步驟，直到您收到確認 Microsoft 關閉您的帳戶。
