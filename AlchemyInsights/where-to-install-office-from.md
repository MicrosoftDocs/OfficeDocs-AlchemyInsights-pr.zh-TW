# <a name="install-office"></a>我有其他問題

若要安裝 Office，使用者必須具備包含桌面應用程式，並[指派授權](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users) [Office 365 商務版產品](https://support.office.com/article/f8ab5e25-bf3f-4a47-b264-174b1ee925fd?wt.mc_id=Alchemy_ClientDIA)。
  
Office 可以再安裝遵循下列步驟：
  
1. 移至[portal.office.com/OLS/MySoftware.aspx](https://portal.office.com/OLS/MySoftware.aspx)

2. 在 Office 的首頁上，選取您要的語言，然後選取**進階**的 64 位元或 32 位元版本之間進行選擇。

3. 選取 [**安裝 Office**。

如需詳細資訊，請參閱[下載並安裝或重新安裝在 PC 或 Mac](https://support.office.com/article/4414eaaf-0478-48be-9c42-23adc4716658?wt.mc_id=Alchemy_ClientDIA)。
  
如需 Visio 或 Project 的特定指示，請參閱[安裝 Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710)或[安裝的專案](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281)。
