# <a name="got-a-parameter-cannot-be-found-error"></a>獲得 '找不到參數 」 錯誤訊息？

如果您收到錯誤訊息 **」 參數找不到符合參數名稱 ' EmailAddresses 「**，就表示，花費的時間較長的時間來完成設定您的租用戶或您的自訂網域，如果您最近新增的 Office 365。 
  
安裝程序可能需要多達 4 個小時才能完成。請稍候，讓設定程序已完成，然後再試一次的時間。
