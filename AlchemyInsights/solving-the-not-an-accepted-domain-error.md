# <a name="got-a-parameter-cannot-be-found-error"></a>找不到「無法找到 ' 參數」錯誤？

如果您收到錯誤訊息「**找不到符合參數名稱 ' EmailAddresses」的參數**, 則表示如果您最近新增了 Office 365 的設定, 或您的自訂網域, 就會花很長的時間來完成設定您的租使用者。
  
設定程序最多可能需要 4 個小時才能完成。 請稍候，讓設定程序完成，然後再試一次。
