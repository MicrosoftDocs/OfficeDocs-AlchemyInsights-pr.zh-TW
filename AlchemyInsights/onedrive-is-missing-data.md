# <a name="onedrive-is-missing-data"></a>OneDrive 遺失資料

如果您的 OneDrive 檔案大量遭到刪除、覆寫、損毀或遭惡意程式碼感染，我們會為使用者提供各種選項來協助進行復原程序。

- [在 OneDrive 中尋找遺失或遺漏的檔案](https://go.microsoft.com/fwlink/?linkid=2125166)
