# <a name="changing-ews-throttling-settings"></a>變更 EWS 節流設定

請執行我們的自動測試，讓您在移轉期間修改 EWS 節流原則。

請注意，EWS 節流原則變更對下列移轉類型 (使用 Microsoft 工具) 沒有任何影響：混合式、完全移轉/分段 (RPC/HTTP)、IMAP、G Suite、公用資料夾或 PST 匯入服務。
