# <a name="software-updates-are-not-being-deployed"></a>未部署軟體更新

如果您透過 Windows 10 更新通道設定軟體更新，但更新並未部署，請嘗試下列步驟：  

- 請考慮變更 Windows 服務，從 **半年通道** 發行類型，改成更嚴格且更頻繁的發行類型。
- 檢查 **品質更新** 和 **功能更新**的延期期間。 延期期間可能會導致更新最多延遲 180 天。
