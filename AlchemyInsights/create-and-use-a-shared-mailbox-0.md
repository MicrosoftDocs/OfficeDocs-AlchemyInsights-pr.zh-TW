# <a name="create-and-use-a-shared-mailbox"></a>建立並使用共用的信箱

共用的信箱可讓多人傳送及接收電子郵件別名，例如**sales@contoso.com**，卻沒有指定另一個授權。
  
[建立共用信箱](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)說明如何： 
  
- 建立共用信箱
    
- 授與您的使用者使用共用的信箱的權限
    
- 存取共用的信箱在 Outlook 或網頁型 Outlook 中
