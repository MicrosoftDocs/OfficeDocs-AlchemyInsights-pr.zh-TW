# <a name="content-type-publishing"></a>內容類型發佈

如果您想讓所有使用者都能在組織中使用內容類型，最好是發佈所有的網站集合。 若要發佈內容類型，請參閱 [從內容發佈中樞發佈內容類型](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02)。
