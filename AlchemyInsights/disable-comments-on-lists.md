# <a name="disable-comments-on-lists"></a>停用清單上的批註

系統管理員可以在 **Set-SPOTenant** PowerShell Cmdlet 中變更 **CommentsOnListItemsDisabled** 參數，以停用組織層級的功能。

目前不可能停用網站或清單層級的批註。 我們希望在後續的更新中使用這些控制項，這可能會在2021的第一季。
