# <a name="configure-and-troubleshoot-mapped-network-drives"></a>設定和疑難排解已對應的網路磁碟機

將文件庫對應為網路磁碟機是暫時作法，且僅支援 Internet Explorer。 相反地，[將 SharePoint 檔案與新的 OneDrive 同步處理用戶端](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88)同步處理，則可提供[檔案隨選](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e)。 不必使用本機儲存空間，即可在 OneDrive 中存取所有檔案。

- [疑難排解連線至 SharePoint Online 的對應網路磁碟機](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives)
