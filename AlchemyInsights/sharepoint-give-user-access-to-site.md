# <a name="give-users-access-to-sharepoint-site"></a>授與使用者存取 SharePoint 網站的許可權

「存取權要求」功能可讓使用者要求存取他們目前沒有許可權的內容。 作為網站擁有者, 您可以設定功能, 以在某人要求網站存取權時傳送給您的郵件。 如需詳細資訊, 請造訪[設定和管理存取要求](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)。

以下是有關存取要求的一些常見連結:

- [當您在 SharePoint Online 中處理擱置的要求時, 拒絕存取「存取要求清單」或「要求核准失敗」](https://support.office.com/article/-Access-Denied-to-Access-Requests-list-or-Request-approval-failed-when-you-process-a-pending-request-in-SharePoint-Online-3ec71fe1-9111-4a4c-84a5-bdfe05c2e558)

- [變更存取要求設定](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [核准或拒絕網站的存取要求](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [管理外部使用者的邀請 (僅限 SharePoint Online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [重新傳送邀請](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
