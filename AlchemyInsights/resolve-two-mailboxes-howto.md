# <a name="a-user-has-two-mailboxes"></a>使用者有兩個信箱

使用 Azure Active Directory Connect (AAD Connect) 或 DirSync 的混合式環境可能不小心會導致使用者有兩個信箱： 一個內部部署，並在雲端中的一個。 無法在任一處建立重複的信箱。

若要解決此問題，請參閱[How to recover 時信箱存在於 Exchange Online 和內部部署](https://docs.microsoft.com/exchange/troubleshoot/move-mailboxes/mailbox-exists-exo-onpremises)。 如果您想要深入了解如何避免此問題發生在未來，請參閱[我使用者具有信箱這兩個內部部署和 Exchange Online。協助 ！](https://techcommunity.microsoft.com/t5/Exchange-Team-Blog/My-user-has-a-mailbox-both-on-premises-and-in-Exchange-Online/ba-p/846809)。
