# <a name="sharepoint-hub-site-navigation"></a>中樞站台的 SharePoint 導覽

頂端的中樞站台和任何相關聯的網站，上述網站導覽，為中樞站台導覽列中。 如果您是中樞站台擁有者，您可以自訂[導覽列中](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav)。 

**附註：** 建議您選取通訊網站，或是使用新式範本作為中樞站台的小組網站。 如果您使用傳統的小組網站，hub 導覽只會出現在現代的頁面，並中樞站台設定才會出現在現代] 頁面上。
