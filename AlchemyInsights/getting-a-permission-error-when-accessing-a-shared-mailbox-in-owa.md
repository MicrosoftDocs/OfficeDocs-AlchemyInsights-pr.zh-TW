# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>網頁型 Outlook 中開啟共用的信箱時，請取得權限錯誤？

如果您收到錯誤**發生錯誤。您不需要開啟此信箱的權限。** 當您嘗試在網頁型 Outlook 中開啟共用的信箱，您的帳戶可能沒有存取共用的信箱的權限。 您的組織系統管理員可以將您的帳戶權限指派給共用信箱，遵循主題[建立共用信箱](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)中的指示進行。
