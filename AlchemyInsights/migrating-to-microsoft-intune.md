# <a name="migrating-to-microsoft-intune"></a>移轉至 Microsoft Intune

如果您嘗試移轉至 Intune，請參閱 [Intune 移轉指南](https://docs.microsoft.com/intune/fundamentals/migration-guide)。

如需從 System Center Configuration Manager (和 SCCM 混合式) 移轉到 Intune 的特定指引，請參閱我們的[混合式移轉文件](https://docs.microsoft.com/sccm/mdm/deploy-use/migrate-hybridmdm-to-intunesa)。
