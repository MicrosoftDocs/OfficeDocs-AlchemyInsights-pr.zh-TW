# <a name="cannot-add-or-create-a-new-signature-in-outlook"></a>無法在 Outlook 中新增或建立新的簽章

此為已知問題。 我們正在努力修正它, 在 Outlook 中更新狀態會[凍結按一下 [](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02)簽章] 按鈕。

若要解決此問題, 請先嘗試卸載預先安裝的 Microsoft Office 桌面應用程式的[步驟](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02)。 

如果這不適用於受影響的機器, 請嘗試使用登錄的[變通方法](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02)。
