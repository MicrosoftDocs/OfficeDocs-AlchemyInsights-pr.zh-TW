# <a name="close-your-outlookcom-email-account"></a>關閉 Outlook.com 電子郵件帳戶

1. 移至 [[關閉您的帳戶](https://go.microsoft.com/fwlink/p/?linkid=845493)] 頁面上。 
    
2. 遵循指示以確定您的帳戶已準備好要關閉頁面上，選取 [**下一步**。 
    
3. 讀取清單及核取方塊，以確認您已閱讀每個項目。
    
4. 在 [**選取原因**] 下拉式清單中，選擇想要關閉之帳戶的原因。 
    
5. 選取 [**關閉標記帳戶**。 
    
若要深入了解，請參閱[關閉 Outlook.com 帳戶](https://go.microsoft.com/fwlink/p/?linkid=873106)[](https://support.office.com/article/564b801e-2a47-4cb2-afa8-12ead3185038.aspx)。
