# <a name="fix-messages-that-are-stuck-in-the-outbox-with-sara"></a>使用 SaRA 修正已滯留于寄件匣中的郵件

我們建議您在受影響的機器上，從[Microsoft 支援和修復小幫手（SaRa）](https://diagnostics.office.com/#/)工具執行「我正在傳送[、接收或尋找電子郵件訊息時出現問題](https://aka.ms/SaRA-OutlookSendReceive)」案例。
