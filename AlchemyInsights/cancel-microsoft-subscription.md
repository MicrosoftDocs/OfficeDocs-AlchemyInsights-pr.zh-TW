# <a name="cancel-microsoft-subscription"></a>取消 Microsoft 訂閱

如果您想要取消 Microsoft 訂閱或試用版訂閱，包括 Xbox Live 和 Microsoft 365，請參閱[如何取消您的 Microsoft 訂閱](https://support.microsoft.com/help/4027815)。

**附註**：若要退款、退回或交換，請參閱[退回您從 Microsoft Store 購買的商品以進行換貨或退款](https://support.microsoft.com/help/10558)。 您可以查看 [[訂單記錄]](https://account.microsoft.com/billing/orders/)，驗證您已從 Microsoft 訂購的項目。 

**附註**：如果您在關閉週期性計費之後看到計費，可能是在會在到期日之後才進行關閉。 [深入了解](https://support.microsoft.com/help/10640)。
