# <a name="add-a-user-account-in-windows-10"></a>在 Windows 10 中新增的使用者帳戶

以下是如何在 Windows 10 中新增的帳戶：

1. 移至**開始** > **設定** > **帳戶** > **系列 & 其他人**（或**其他人**，如果您使用 Windows 10 企業版） >**新增其他人到此電腦]**。
2. 請依照下列指示，新增 Microsoft 帳戶或本機帳戶。

[深入了解設定帳戶](https://support.microsoft.com/help/17197/)
