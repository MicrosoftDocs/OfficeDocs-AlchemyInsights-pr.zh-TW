# <a name="internal-server-errors-in-outlookcom"></a>Outlook.com 中的內部伺服器錯誤

哦不！ 如果您看到「錯誤500」或「內部伺服器錯誤」訊息，那是我們的資訊，而不是您-對不起。

請查看 [ [Office 服務狀態] 頁面](https://portal.office.com/servicestatus) ，確定沒有任何其他專案正在進行中，但最好的解決方案是在幾分鐘內中斷，然後按 **CTRL** 和 **F5**重新整理您的瀏覽器。
