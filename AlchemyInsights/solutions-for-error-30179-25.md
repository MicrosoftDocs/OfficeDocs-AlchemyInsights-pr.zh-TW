# <a name="solutions-for-error-30179-25"></a>錯誤30179-25 的解決方案

如果您的硬碟空間不足，您可能會看到郵件「無法串流 Office」（錯誤碼[30179-25](https://support.office.com/article/e40d3c7d-98f6-4284-94a0-882beaa44593?wt.mc_id=Alchemy_ClientDIA)）的錯誤。
  
請[在您的硬碟上釋放空間](https://support.microsoft.com/help/12425/windows-10-free-up-drive-space)，然後[重新安裝 Office](https://portal.office.com/OLS/MySoftware.aspx)。
