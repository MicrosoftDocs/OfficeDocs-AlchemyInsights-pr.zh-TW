# <a name="calendar-publishing-information"></a>行事曆發佈資訊

- Microsoft 365 系統管理員可以啟用或停用組織的行事曆發佈和行事曆共用。 如需詳細資訊，請參閱[開啟行事曆發佈功能，讓使用者可以](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC)[與外部使用者](https://docs.microsoft.com/microsoft-365/admin/manage/share-calendars-with-external-users)共用資訊和共用行事曆。

- Outlook 使用者可以共用自己的行事曆。 如需詳細資訊，請參閱[與其他人共用 Outlook 行事曆](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)。
