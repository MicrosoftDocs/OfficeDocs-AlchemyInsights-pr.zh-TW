# <a name="calendar-publishing-information"></a>行事曆發佈資訊

- 系統管理員可以啟用或停用組織的行事曆發佈和行事曆共用。 如需詳細資訊, 請參閱[開啟行事曆發佈, 讓使用者可以共用資訊](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC), 以及[與外部使用者共用行事曆](https://support.office.com/article/FB00DD4E-2D5F-4E8D-8FF4-94B2CF002BDD)。

- Outlook 使用者可以共用自己的行事曆。 如需詳細資訊, 請參閱[與其他人共用 Outlook 行事曆](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)。
