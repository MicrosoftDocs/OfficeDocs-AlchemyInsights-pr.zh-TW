# <a name="working-with-sharepoint-permission-levels"></a>使用 SharePoint 權限等級

您可以使用，例如**參與**] 和 [**僅限檢視**的預設權限等級隨附 SharePoint。 但是，如果其中一種不符合您的需求，您可以建立新的權限等級。 請記住，很容易持續追蹤的權限等級，如果您未變更預設值。

[如何建立和編輯權限等級](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
