# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>將使用者信箱轉換為共用信箱

當您將使用者的信箱轉換成共用信箱時, 會保留所有現有的電子郵件和行事曆。 [! 注意] 現在只有在共用信箱中, 有幾個人才能存取它, 而不是一個人。 在日後, 您可以將共用信箱轉換回到使用者 (私人) 信箱。
