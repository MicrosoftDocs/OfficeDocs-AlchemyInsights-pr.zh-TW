# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>將使用者信箱轉換為共用信箱

當您將個別使用者的信箱轉換為共用信箱時，所有現有的電子郵件和行事曆項目會保留在有多人可以存取共用信箱中。 您也可以回復到使用者 （私人） 信箱轉換為共用的信箱。
