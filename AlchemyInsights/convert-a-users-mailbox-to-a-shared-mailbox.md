# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>將使用者信箱轉換成共用信箱

當您將使用者的信箱轉換成共用信箱時，會保留所有現有的電子郵件和行事曆。僅限現在是在其中數個人員將能夠存取，而不是某個人共用信箱。日後，您可以回到使用者 （私人） 信箱轉換共用的信箱。
