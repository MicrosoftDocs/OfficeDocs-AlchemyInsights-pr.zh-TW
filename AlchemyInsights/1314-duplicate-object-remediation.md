# <a name="duplicate-object-synchronization-errors"></a>複製物件同步處理錯誤

如果您同步處理期間收到下列錯誤：
  
- 您的 Microsoft Online Services 目錄中已經存在具有相同的 proxy 地址的同步處理的物件。
    
- UserPrincipalName 類似的訊息。
    
使用新 AADConnect 狀況診斷解決複製的物件錯誤。如需詳細資訊，請參閱[診斷及修復重複的屬性同步處理錯誤](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors)。
