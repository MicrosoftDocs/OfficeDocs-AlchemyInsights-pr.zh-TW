# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: 疑難排解「處理變更」

如果您在長時間內看到「處理變更」, 可能是因為您已開啟線上檔案、有大量的檔案在佇列中, 或是非常大型的檔同步處理, 請參閱[OneDrive 在處理中停滯](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd)。

請閱讀本文以疑難排解「處理變更」。如果您有一則會變更的檔案 (如 Outlook), 您也會看到這則訊息。PST 檔案或資料庫)。 瞭解如何將您要使用的[Outlook 資料檔案](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e), 從您的 OneDrive 中移出。
