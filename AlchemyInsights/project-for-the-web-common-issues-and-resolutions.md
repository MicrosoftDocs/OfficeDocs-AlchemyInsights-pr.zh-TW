# <a name="project-for-the-web-common-issues-and-resolutions"></a>Project 網頁版常見問題與解決方法

以下是一些 Project 網頁版的一些常見要求主題：

- 了解 [Project 網頁版](https://support.microsoft.com/zh-TW/office/what-is-project-for-the-web-c19b2421-3c9d-4037-97c6-f66b6e1d2eb5)，可在 [三個 Project 訂閱](https://products.office.com/project/compare-microsoft-project-management-software) 中取得全部資訊。
- [為組織的使用者開啟或關閉](https://docs.microsoft.com/project-for-the-web/turn-project-for-the-web-off) Project 網頁版或藍圖。 系統管理員可以透過 Microsoft 系統管理中心中的 **專案設定** 頁面來控制 Project 網頁版存取權。
- 深入了解使用 Project 網頁版 [可以執行的作業](https://support.office.com/article/what-can-you-do-with-project-for-the-web-b30f5442-be5f-43d2-9072-c95bff778ea1)。
- 使用 [Project 網頁版訓練](https://support.office.com/article/get-started-with-project-for-the-web-50bf3e29-0f0d-4b7a-9d2c-7c78389b67ad) 來協助使用者建立及建造專案。
- 熟悉 Project 網頁版 [限制和界限](https://docs.microsoft.com/project-for-the-web/project-for-the-web-limits-and-boundaries)。

如需有關 Project 網頁版的詳細資訊，請參閱 [Project 網頁版系統管理說明](https://docs.microsoft.com/project-for-the-web/projectforweb-admin-home)。
