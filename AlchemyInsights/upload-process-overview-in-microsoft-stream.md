# <a name="upload-process-overview-in-microsoft-stream"></a>在 Microsoft Stream 中上傳處理常式概述

每當您上傳影片時，Microsoft Stream 都會自動將其編碼為多個 bitrates，這樣影片便可使用適應性高位元速率資料流程進行流式處理。 Microsoft Stream 會根據檢視器的可用頻寬，優化播放影片的高位元速率和品質。

- [在 Microsoft Stream 中上傳處理常式概述](https://docs.microsoft.com/stream/upload-process-overview)
