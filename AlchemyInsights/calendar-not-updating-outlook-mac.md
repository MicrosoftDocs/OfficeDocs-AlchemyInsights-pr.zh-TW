# <a name="calendar-not-updating-outlook-for-mac"></a>Mac 版 Microsoft Outlook 行事曆未更新

若遺失或未更新行事曆項目，我們將從您那裡收集具體資料以調查問題：

1. 開啟 Outlook，選取 Windows 功能表，然後選取 **同步錯誤**。

1. 按一下齒輪左側的黑色主控台圖示。 診斷資料夾將自動開啟。

1. 壓縮並上傳最新的診斷檔案以支援分析。
