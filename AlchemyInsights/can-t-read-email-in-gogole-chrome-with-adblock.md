# <a name="cant-read-email-in-google-chrome-with-adblock"></a>無法讀取電子郵件中使用 AdBlock Google Chrome

某些使用者讀取電子郵件時，發生問題或當 AdBlock 副檔名為使用 Google Chrome 撰寫新電子郵件。 請嘗試下列解決方法：
  
- 使用[Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409)或另一個網頁瀏覽器。 
    
- 排除 AdBlock 設定 Outlook.com。 在 Google Chrome 右上角，選取**AdBlock** ，然後選取 [**不要在這個網域的頁面上執行**。 重新整理瀏覽器以重新載入 Outlook.com。
