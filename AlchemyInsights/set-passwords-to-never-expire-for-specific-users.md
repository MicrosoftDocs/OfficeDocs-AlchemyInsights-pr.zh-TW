# <a name="set-passwords-to-never-expire"></a>設定密碼永遠不過期

若要設定所有永遠不過期的密碼，請前往[安全性&amp;隱私權](https://portal.office.com/adminportal/home#/settings/security)。按一下 [**密碼原則**] 旁的 [**編輯**和開啟滑桿以**上**。
  
如需詳細資訊：[設定您的組織的密碼到期原則。](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f)
  
[設定特定使用者永遠不過期的密碼](https://support.office.com/article/f493e3af-e1d8-4668-9211-230c245a0466)
