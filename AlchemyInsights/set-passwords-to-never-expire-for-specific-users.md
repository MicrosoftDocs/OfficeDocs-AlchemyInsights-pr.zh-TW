# <a name="set-passwords-to-never-expire"></a>設定密碼永不過期

若要設定所有的密碼永不過期，移至[安全性&amp;隱私權](https://portal.office.com/adminportal/home#/settings/security)。 按一下 [**編輯**] 旁**的密碼原則**，然後開啟滑桿來**上**。
  
如需詳細資訊，請參閱：[設定貴組織的密碼到期原則。](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f)
  
[設定為永不過期針對特定使用者的密碼](https://support.office.com/article/f493e3af-e1d8-4668-9211-230c245a0466)
