# <a name="set-passwords-to-never-expire"></a>將密碼設為永不過期

若要將所有密碼設為永不過期, 請移至 [[安全性&amp;隱私權](https://portal.office.com/adminportal/home#/settings/security)]。 按一下 [**密碼原則**] 旁的 [**編輯**], 然後將滑杆轉換為 [**開啟**]。
  
如需詳細資訊, 請參閱:[設定組織的密碼到期原則。](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f)
  
[將密碼設為永不過期的特定使用者](https://support.office.com/article/f493e3af-e1d8-4668-9211-230c245a0466)
