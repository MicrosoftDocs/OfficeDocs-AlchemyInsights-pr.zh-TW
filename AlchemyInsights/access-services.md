# <a name="access-services"></a>Access Services

在 SharePoint Online 中的 Access Services 功能已被淘汰。不再可以建立新的 Access web 應用程式與 web 資料庫，並現有的 web 應用程式與 web 資料庫將會關閉由年 4 月、 2018年。Access 桌面資料庫 (.accdb) 不會受到此變更。我們建議 Microsoft PowerApps 另一個到存取 web 應用程式與 web 資料庫。 
  
[了解 Microsoft PowerApps](https://powerapps.microsoft.com/)
