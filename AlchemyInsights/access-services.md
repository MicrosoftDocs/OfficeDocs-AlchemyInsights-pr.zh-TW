# <a name="access-services"></a>Access Services

已停用 SharePoint Online 中的 Access Services 功能。 不能再建立新的 Access web 應用程式和 web 資料庫，而現有的 web 應用程式和 web 資料庫將會在2018年4月關閉。 存取桌面資料庫 ( .accdb) 不會受到此變更的影響。 建議使用 Microsoft PowerApps 作為 Access web 應用程式和 web 資料庫的替代方案。 
  
[深入瞭解 Microsoft PowerApps](https://powerapps.microsoft.com/)
