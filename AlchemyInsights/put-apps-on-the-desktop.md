# <a name="put-apps-on-the-desktop"></a>放在桌面上的應用程式

在 [**開始**] 功能表中，應用程式搜尋並按住滑鼠左鍵然後移動到桌面圖示抓取的應用程式圖示。

您可能也按下**Windows 鍵 + R** ，然後輸入此命令中：

`shell:appsfolder`

這會在**檔案總管**中開啟的**應用程式**資料夾。 您可能會立即拖放到桌面的應用程式圖示。
