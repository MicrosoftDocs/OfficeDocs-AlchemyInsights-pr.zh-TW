# <a name="add-or-remove-an-email-address-for-a-user"></a>新增或移除使用者的電子郵件地址

[請依照](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)Microsoft 365 系統管理中心的下列步驟開始著手。

 **附注**：您可以有多個使用者的電子郵件地址，但其他*電子郵件別名*只可用於接收電子郵件。 如果您需要使用別名傳送電子郵件，請建立[共用信箱](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)。 如此一來，您就可以取得別名以傳送和接收郵件。
