# <a name="set-up-spf-in-office-365"></a>設定 Office 365 中的 SPF

更新 DNS 記錄，讓您可以使用寄件者原則架構 (SPF) 自訂網域來驗證自您自訂的網域傳送的輸出電子郵件。這有助於防止其他人"詐騙"郵件好像其確實傳送自您的網域。
  
- [在 Office 365 中設定 SPF 以協助防止詐騙](https://docs.microsoft.com/en-us/office365/SecurityCompliance/set-up-spf-in-office-365-to-help-prevent-spoofing)
    
- [說明設定所有的 DNS 記錄](https://docs.microsoft.com/en-us/office365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
