# <a name="credit-check-status-request"></a>信用檢查狀態要求

您的訂單為擱置信用驗證。 這通常是在24小時內完成, 但最多可能需要48小時。 完成後, 您的訂閱狀態會從 [擱置] 變更為 [使用中], 而服務將可供使用。
