# <a name="implement-qos-and-monitor-call-quality-in-teams"></a>在小組中實施 QoS 和監視通話品質

通話分析和通話品質儀表板可用於協助疑難排解 Microsoft 小組中的通話品質問題。 若要深入瞭解服務品質（QoS）、呼叫分析和通話品質儀表板（CQD），請參閱[在 Microsoft 團隊中執行 QoS 和監視通話品質](https://docs.microsoft.com/microsoftteams/monitor-call-quality-qos)。
