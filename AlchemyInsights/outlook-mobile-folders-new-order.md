# <a name="my-folders-are-in-a-new-order"></a>我的資料夾是以新順序

您可以將最愛的資料夾和群組新增至 Outlook 的 iOS 和 Android，但您可能會注意到您的資料夾是以新的順序顯示。 如果您有我的最愛，您會在邊欄上方看到 [最愛] 區段。 您可以使用靠近側邊邊的上方鉛筆圖示來新增我的最愛。 這會將所有最重要的資料夾和群組保留在最容易存取的位置上。
