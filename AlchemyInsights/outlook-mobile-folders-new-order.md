# <a name="my-folders-are-in-a-new-order"></a>我的資料夾會以新的順序

加上我的最愛資料夾與 Outlook for iOS 和 Android 的群組，您可能會注意到資料夾會以新的順序。 如果您有 [我的最愛]，您會看到 [我的最愛] 區段頂端的您提要欄位。 您可以新增使用鉛筆圖示您提要欄位頂端附近的 [我的最愛]。 這會保留所有您最重要的資料夾及群組，在其中您可以輕鬆地存取的最頂端。
