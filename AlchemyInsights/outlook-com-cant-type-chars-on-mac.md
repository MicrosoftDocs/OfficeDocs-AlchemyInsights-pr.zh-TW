# <a name="how-to-type-special-characters-on-a-mac"></a>如何在 Mac 上輸入特殊字元

若要在 Mac 上輸入特殊字元，開啟 [**特殊字元**] 功能表中使用鍵盤快速鍵 ⌘**命令** + **控制項** + **空間**。

@ 快速捷徑也是**ALT** + **64**。
