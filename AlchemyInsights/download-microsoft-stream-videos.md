# <a name="download-microsoft-stream-videos"></a>下載 Microsoft Stream 影片

**如果您已上傳影片檔**或**您是影片的擁有者**，則可以**下載**原始上傳的影片。

1. 在 Stream 中，找到您要下載的影片。

2. 按一下其他功能表圖示 [...]**

3. 選取 [下載原始影片]****。
