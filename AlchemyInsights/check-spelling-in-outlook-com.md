# <a name="use-your-browser-to-check-spelling"></a>使用您的瀏覽器進行拼字檢查

沒有在 Outlook.com 中的拼字檢查選項。 若要檢查拼字，您需要使用網頁瀏覽器。 在 Microsoft Edge、 Internet Explorer 10 及更新版本、 和 Firefox、 Chrome 和 Safari 的目前版本中使用拼字檢查。 請檢查您的網頁瀏覽器，若要深入了解如何進行拼字檢查的選項。
