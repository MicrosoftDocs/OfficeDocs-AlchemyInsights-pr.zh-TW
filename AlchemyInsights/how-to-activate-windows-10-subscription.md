# <a name="activating-windows-10-subscriptions"></a>啟動 Windows 10 的訂閱

Microsoft 365 商務版，Microsoft 365 教育版、 Microsoft 365 企業版提供數位授權適用於 Windows 10，做為訂閱的一部分。 若要啟用訂閱，您的裝置必須已有啟用 Windows 10 專業版 （1703年或更新版本）。 若要啟用 Windows 10 訂閱，請參閱[Windows 10 訂閱啟用需求](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements)需求的完整清單。
