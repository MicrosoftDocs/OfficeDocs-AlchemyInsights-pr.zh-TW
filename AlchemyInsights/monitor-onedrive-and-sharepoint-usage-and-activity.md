# <a name="view-reports-on-onedrive-and-sharepoint-activity-and-usage"></a>查看 OneDrive 和 SharePoint 活動和使用狀況的報告

移至[Microsoft 365 系統管理中心的 [使用狀況報告] 頁面](https://admin.microsoft.com/AdminPortal/Home)。 在 [**選取報告**] 底下, 選擇 [OneDrive] 或 [SharePoint 活動] 或 [使用狀況] 報告。
  
[OneDrive 報告的詳細資訊](https://go.microsoft.com/fwlink/?linkid=875239)
  
[有關 SharePoint 報表的詳細資訊](https://go.microsoft.com/fwlink/?linkid=875240)
