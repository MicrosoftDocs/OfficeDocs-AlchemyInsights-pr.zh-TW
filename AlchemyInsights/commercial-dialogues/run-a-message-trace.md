# <a name="run-a-message-trace"></a>執行郵件追蹤

在 Office 365 安全性 & 合規性中心的 [郵件流程] 區段中執行郵件追蹤。

若要瞭解如何執行郵件追蹤，請參閱 [Office 365 Security & 合規性中心中的郵件追蹤](https://go.microsoft.com/fwlink/?linkid=2103855)。
