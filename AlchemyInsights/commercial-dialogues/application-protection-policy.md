# <a name="application-protection-policy-requirements"></a>應用程式保護原則需求

應用程式保護原則需求：

- 使用者擁有 Intune 或 EMS 授權。
- 使用者屬於應用程式保護原則鎖定的群組。
- 裝置上只有一個公司使用者登入受保護的應用程式。
- [應用程式已實現 Intune SDK。如需支援 SDK 的應用程式清單，請參閱 Microsoft Intune 受保護應用程式。](https://docs.microsoft.com/mem/intune/apps/apps-supported-intune-apps)
