# <a name="reset-the-users-password"></a>重設使用者的密碼

您必須先重設使用者的密碼，再執行其他動作。 方法如下：

1. 選取 [**使用者** 作用  >  中 **[使用者](https://go.microsoft.com/fwlink/p/?linkid=834822)**]。
1. 選取其帳戶已遭破壞的使用者。
1. 在出現的快顯視窗中，按一下 [ **重設密碼**]。

> [!NOTE]
> 不要透過電子郵件將密碼傳送給使用者。
