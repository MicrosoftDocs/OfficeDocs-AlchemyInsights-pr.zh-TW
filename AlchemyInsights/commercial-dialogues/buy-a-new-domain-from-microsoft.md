# <a name="buy-a-new-domain-from-microsoft"></a>從 Microsoft 購買新的網域

若要從 Microsoft 購買新的網域，請執行下列步驟：

1. 流覽至 [ **安裝 > 網域** ]，然後按一下 [ **購買網域**]。 
2. 依照精靈的指示進行。
