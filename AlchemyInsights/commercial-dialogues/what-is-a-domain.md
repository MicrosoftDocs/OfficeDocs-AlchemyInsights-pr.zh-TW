# <a name="whats-a-domain"></a>何謂網域

網域是您的電子郵件地址中 @ 符號後面的部分。 例如，在 [位址 rob@contoso.com] 中，網域是 **contoso.com**。 大多數的企業喜歡在其電子郵件地址中使用自己的功能變數名稱。

以下是一些有關網域的秘訣：

- 您可以從 Microsoft 購買功能變數名稱。 如此一來，您就會有自訂的電子郵件地址。
- 或者，您可以只將已擁有的網域新增至 Office 365。
