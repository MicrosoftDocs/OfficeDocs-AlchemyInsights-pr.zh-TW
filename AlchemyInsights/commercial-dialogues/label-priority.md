# <a name="label-priority-order-matters"></a>標籤優先順序 (順序很重要)

在系統管理中心中建立敏感度標籤時，它們會顯示在 **[標籤]** 頁面的 **[敏感度]** 索引標籤上的清單中。 在此清單中，標籤的順序很重要，因為它反映的是其優先順序。 您想要讓最具限制性的敏感度標籤，例如 **[高度機密性]**，顯示在清單的最底端，以及最不具限制性的敏感度標籤，例如 **公用**，顯示在最上方。

您只能將一個敏感度標籤套用至文件或電子郵件。 如果您設定一個選項，要求使用者提供將標籤變更為較低分類的理由，則此清單的順序會識別分類下限。

子標籤的排序會透過[自動標記](https://docs.microsoft.com/microsoft-365/compliance/apply-sensitivity-label-automatically)完成。 將標籤設定為自動套用或建議時，可能對多個標籤造成相符項目。 如需詳細資訊，請參閱[標籤優先順序 (順序很重要)](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels)。
