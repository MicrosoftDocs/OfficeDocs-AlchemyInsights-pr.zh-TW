# <a name="download-files-and-folders-from-onedrive-or-sharepoint"></a>從 OneDrive 或 SharePoint 下載檔案和資料夾

只要按幾下滑鼠，您就能從 Microsoft OneDrive 或 Microsoft 365 的 SharePoint 或 SharePoint Server 2019 中將檔案和資料夾下載到您的電腦。 如需相關資訊，請參閱[從 OneDrive 或 SharePoint 下載檔案和資料夾](https://support.microsoft.com/office/download-files-and-folders-from-onedrive-or-sharepoint-5c7397b7-19c7-4893-84fe-d02e8fa5df05)。
