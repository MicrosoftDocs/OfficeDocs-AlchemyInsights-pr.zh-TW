# <a name="need-to-create-an-email-signature"></a>需要建立電子郵件簽名？

如果您有 Outlook 帳戶或網頁上的 Outlook (工作或學校) 帳戶，您可以建立簽章並自動將其新增至所有外寄郵件，或選擇性地將其新增至特定的外寄郵件。 請注意，Outlook 和 Outlook 網頁版中的電子郵件簽章有所不同。 若要建立您的電子郵件簽章，請參閱在 outlook 網頁版中 [建立及新增簽章給郵件](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) 的功能，或 [建立及新增電子郵件簽名](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)。
