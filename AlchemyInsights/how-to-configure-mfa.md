# <a name="configure-multi-factor-authentication"></a>設定多重要素驗證

若要設定 MFA，請參閱[設定多重要素驗證](https://docs.microsoft.com/office365/admin/security-and-compliance/set-up-multi-factor-authentication)。

請注意，您必須是全域系統管理員，才能啟用和設定 MFA。
