# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>將 Google Chrome extensions 的埠 Chromium 至 Microsoft Edge () 

可輕鬆將 [Google Chrome 擴充功能埠 Chromium 至 Microsoft Edge () ](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension)。 在大多數情況下，只需要最小的變更，即可在 Microsoft Edge 上執行這些擴充。

Google Chrome 支援的分機 APIs 和資訊清單機碼與 Microsoft Edge 相容。 不過，Microsoft Edge 不支援 APIs chrome，getAccounts，chrome，getAuthToken，and chrome. instanceID 的副檔名。
