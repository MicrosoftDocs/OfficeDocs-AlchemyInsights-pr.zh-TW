# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>在行動裝置上使用共用的信箱

您無法從行動裝置已安裝的 Outlook 應用程式存取共用的信箱。您是否希望此功能吗？讓 Outlook 工程小組知道。[以下投票](https://go.microsoft.com/fwlink/?linked=862116)！
  
您只可以在行動裝置上使用共用的信箱開啟瀏覽器、 登入 Office 365，然後移至網路上的 Outlook。指示，請參閱[新增 outlook web 上共用信箱](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)。
  
如果您必須在存取行動裝置上使用 Outlook 應用程式的共用的信箱，請考慮建立 Office 365 群組而不是使用共用的信箱。如需詳細資訊，請參閱[比較群組](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx)。
  
深入了解：[取得 iOS 及 Android outlook 的應用程式說明](https://support.office.com/en-us/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
