# <a name="what-is-autosave"></a>何謂 AutoSave？

您可以在 Excel、Word 及 PowerPoint 中的 Microsoft 365 訂閱者中使用 AutoSave 功能，在您工作時，每隔幾秒便會自動儲存您的檔案。 

如需常見問題，請參閱 [什麼是 AutoSave？](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
