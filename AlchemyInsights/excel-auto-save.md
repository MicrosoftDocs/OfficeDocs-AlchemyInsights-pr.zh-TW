# <a name="what-is-autosave"></a>什麼是自動儲存？

[自動儲存] 是 Excel、Word 和 PowerPoint for Office 365 訂閱者中可用的新功能, 可在您工作時, 每隔幾秒自動儲存您的檔案。 

如需常見問題, 請參閱[何謂自動儲存？](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
