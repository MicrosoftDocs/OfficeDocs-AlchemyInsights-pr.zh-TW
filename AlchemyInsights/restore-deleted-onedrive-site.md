# <a name="restore-a-deleted-onedrive-site"></a>還原已刪除的 OneDrive 網站

當您刪除在 Microsoft 365 系統管理中心 （或使用者透過 Active Directory 同步作業中移除時） 中的使用者時，使用者的 OneDrive 會保留您在 OneDrive 系統管理中心中指定的天數。 （如需資訊，請參閱[設定已刪除的 OneDrive 使用者的預設檔案保留](https://docs.microsoft.com/onedrive/set-retention)。）

* [還原已刪除的 OneDrive 網站](https://docs.microsoft.com/onedrive/restore-deleted-onedrive)

如果您的 OneDrive 檔案大量取得已刪除、 覆寫、 損毀，或感染惡意程式碼，您可以還原整個 OneDrive 至先前的時間。

* [還原您的 OneDrive](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
