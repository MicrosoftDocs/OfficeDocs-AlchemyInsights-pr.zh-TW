# <a name="restore-a-previous-file-version"></a>還原先前的檔案版本

與線上版本歷程記錄，您可以看到及還原舊版您 OneDrive 或 SharePoint 中的檔案。版本歷程記錄適用於所有的檔案類型，包括 Pdf、 CAD 檔案、 照片和影片。若要還原的舊的檔案版本：
  
1. 開啟 SharePoint 文件庫或 OneDrive 位置。
    
2. 按一下以選取它，檔案的左側，然後按一下 [**版本歷程記錄**。如果您未看見版本歷程記錄之命令列上，按一下 [其他 （ **...** ）。 
    
3. 按一下您要檢視，或按一下 [更多 （ **...** ） 右邊的版本，然後按一下 [**還原**的版本。
