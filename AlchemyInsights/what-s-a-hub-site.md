# <a name="whats-a-hub-site"></a>什麼是中樞站台？

中樞站台可協助您組織的網站和授與常見的導覽與商標。全域或 SharePoint 系統管理員可以建立中樞站台，指定哪些使用者可以建立他們的網站與該中樞站台的關聯。如需詳細資訊，請參閱[Office 365 中的新 SharePoint 中樞站台](https://go.microsoft.com/fwlink/?linkid=869388)。
