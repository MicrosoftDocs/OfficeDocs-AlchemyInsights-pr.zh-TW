# <a name="whats-a-hub-site"></a>中樞站台為何？

中樞站台可協助您組織的網站並授與常見的導覽和品牌。 全域或 SharePoint 系統管理員將能夠建立中樞站台，並指定哪些使用者可以建立與該中樞站台關聯其網站。 如需詳細資訊，請參閱 < <b0>Office 365 中新的 SharePoint 中樞站台</b0>。
