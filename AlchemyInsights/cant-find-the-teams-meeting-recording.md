# <a name="cant-find-the-teams-meeting-recording"></a>找不到小組會議錄製

如果您找不到或找不到小組會議錄製，請參閱下列疑難排解檔：

[對小組會議錄製問題進行疑難排解-Microsoft 小組 |Microsoft 檔](https://docs.microsoft.com/microsoftteams/troubleshoot/meetings/troubleshoot-meeting-recording-issues)

您的會議錄製會位於 Stream 中，OneDrive 或 SharePoint 取決於租使用者的設定方式。

如需如何尋找會議錄製的指示，請參閱：

**[問題2：聊天視窗中看不到會議錄製連結](https://docs.microsoft.com/microsoftteams/troubleshoot/meetings/troubleshoot-meeting-recording-issues#issue-2-the-meeting-recording-link-isnt-visible-in-a-chat-window)**
