# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>為 Microsoft Edge 建立端點的允許清單

若要讓完整的 Microsoft Edge 功能陣列可供您處置，您必須將它連線到網際網路。 此外，若要讓瀏覽器透過防火牆和網際網路上的其他安全性機制進行通訊，您必須將一組網域 URLs 新增至其 [允許端點清單](https://go.microsoft.com/fwlink/?linkid=2135054)。
