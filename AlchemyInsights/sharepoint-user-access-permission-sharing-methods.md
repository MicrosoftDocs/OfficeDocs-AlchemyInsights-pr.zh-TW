# <a name="sharing-and-permission-methods-to-provide-or-restrict-user-access-to-sharepoint-site"></a>允許或限制對 SharePoint 網站的使用者存取的共用和權限方法

建立 SharePoint 網站後，您可以允許或限制使用者存取網站或其內容。 這可以透過各種如下所述的共用和權限方法。

- [共用 SharePoint 檔案或資料夾](https://support.office.com/article/share-sharepoint-files-or-folders-1fe37332-0f9a-4719-970e-d2578da4941c)

- [共用網站](https://support.office.com/article/share-a-site-958771a8-d041-4eb8-b51c-afea2eae3658)

- [當使用者取得共用連結，變更預設的連結類型](https://docs.microsoft.com/sharepoint/change-default-sharing-link)

- [停止共用 OneDrive 或 SharePoint 檔案或資料夾，或變更權限](https://support.office.com/article/stop-sharing-onedrive-or-sharepoint-files-or-folders-or-change-permissions-0a36470f-d7fe-40a0-bd74-0ac6c1e13323)

- [共用和權限](https://support.office.com/article/Sharing-and-permissions-ac85fbf1-2431-49bf-8690-f1a2b98af65f#ID0EAABAAA=Manage_permissions)

- [開啟外部共用開啟或關閉 Sharepoint Online](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off)

- [OSE 共用 Sharepoint 和 OneDrive 中的錯誤](https://docs.microsoft.com/sharepoint/sharepoint-onedrive-error-message)
