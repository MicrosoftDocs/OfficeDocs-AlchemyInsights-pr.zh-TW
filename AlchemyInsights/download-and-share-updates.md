# <a name="download-and-share-updates"></a>下載及共用更新

[傳遞最佳化] 下載模式不會個別下載更新，可讓您在多部 電腦之間共用更新。 如需詳細資訊，請參閱 [適用於 Windows 10 更新的傳遞最佳化](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization)。  

可用的下載模式包括：  
- 僅限 HTTP  
- LAN  
- 群組  
- 網際網路  
- 簡單  
- 略過
