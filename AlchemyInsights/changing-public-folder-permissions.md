# <a name="changing-public-folder-permissions"></a>變更公用資料夾權限

您和您的使用者可以變更 Outlook 中的公用資料夾權限。 請參閱[設定或變更公用資料夾權限](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005)。 系統管理員也可以變更在 Exchange 系統管理中心中的權限：
  
1.  在 Microsoft 365 系統管理中心中，移至**系統管理中心**[ \> **Exchange**。
    
2. 選取 [**公用資料夾**。
    
從那裡，您可以變更個別公用資料夾的權限指派安全性群組的使用者權限。 若要變更公用資料夾的權限，您必須是它的擁有者。
