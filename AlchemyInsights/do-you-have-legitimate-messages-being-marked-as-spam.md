# <a name="do-you-have-legitimate-messages-being-marked-as-spam"></a>您有合法郵件被標示為垃圾郵件嗎？


請試著上述診斷或此[逐步指南疑難排解良好郵件被標示為垃圾郵件](https://docs.microsoft.com/office365/securitycompliance/prevent-email-from-being-marked-as-spam-0)。
  
報告這種問題的最佳方式是：
  
1. [啟用回報增益集](https://docs.microsoft.com/office365/securitycompliance/enable-the-report-message-add-in)的使用者。 
    
2. 有使用者[報告為不是垃圾郵件](https://support.office.com/article/use-the-report-message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2?ui=en-US&amp;rs=en-US&amp;ad=US)。
    
如果您想要開啟支援票證，請附加的完整電子郵件標頭，至少有**一整個原始郵件**下方。
