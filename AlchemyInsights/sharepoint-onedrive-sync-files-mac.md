# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>在 Mac 上安裝、設定及同步處理 OneDrive 或 SharePoint 檔案 

使用[MAC OS X 上的 OneDrive 同步處理用戶端同步](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f)處理檔案的指示，在 mac 上安裝、設定及同步處理 OneDrive 或 SharePoint 檔案。

如果已在 Mac 上安裝 OneDrive 應用程式並設定，但使用者遇到同步問題，請參閱[使用 OneDrive 修正同步問題（Mac）](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756)。

> [!NOTE]
> OneDrive Windows sync 中的已知資料夾移動功能不適用於 Mac。
