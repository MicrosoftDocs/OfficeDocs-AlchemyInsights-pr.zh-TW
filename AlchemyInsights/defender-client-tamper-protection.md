# <a name="defender-tamper-protection"></a>Defender 防篡改保護 

使用防篡改保護時，惡意應用程式無法採取下列動作：

- 停用病毒和威脅防護
- 停用即時保護
- 關閉行為監控
- 停用防病毒 (例如 IOfficeAntivirus (IOAV) ) 
- 停用雲端傳送保護
- 移除安全性智慧更新

如需詳細資訊，請參閱 [使用防篡改保護保護安全性設定](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/prevent-changes-to-security-settings-with-tamper-protection)。
