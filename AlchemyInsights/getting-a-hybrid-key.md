# <a name="getting-an-exchange-hybrid-key"></a>取得 Exchange 混合式機碼

若要連線內部部署組織至 Office 365 中的 Exchange Online，您需要設定內部部署組織中的至少一部 Exchange 伺服器。此伺服器用來建立連線至 Exchange Online 的內部部署 Exchange 組織的混合式部署。若要取得此伺服器的混合式機碼，遵循的步驟在[Exchange 混合式產品金鑰的通訊群組精靈]](http://aka.ms/hybridkey)。
