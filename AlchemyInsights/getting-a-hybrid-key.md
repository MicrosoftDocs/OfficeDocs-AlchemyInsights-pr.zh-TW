# <a name="getting-an-exchange-hybrid-key"></a>取得 Exchange 混合式金鑰

若要將內部部署組織連線至 Office 365 中的 Exchange Online, 您必須在內部部署組織中設定至少一部 Exchange 伺服器。 此伺服器用來建立將內部部署 Exchange 組織連線至 Exchange Online 的混合式部署。 若要取得此伺服器的混合式金鑰, 請依照[Exchange 混合式產品金鑰發佈嚮導](https://aka.ms/hybridkey)中的步驟進行。
