# <a name="teams-meeting-recordings-to-onedrive"></a>Teams 會議錄製至 OneDrive

Microsoft Teams 有用於儲存會議錄製的新方法。 做為從傳統 Microsoft Stream 轉換為[新 Stream](https://docs.microsoft.com/stream/streamnew/new-stream)的第一階段，此方法會將錄製檔儲存在 Microsoft 商務用 OneDrive 和 Microsoft 365 的 SharePoint 中，並提供許多優點。  

- [設定商務用 OneDrive 和 SharePoint 的會議錄製選項](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#set-up-the-meeting-recording-option-for-onedrive-for-business-and-sharepoint)
- [退出商務用 OneDrive 和 SharePoint，以繼續使用 Stream](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#opt-out-of-onedrive-for-business-and-sharepoint-to-continue-using-stream)  
- [常見問題集](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#frequently-asked-questions)
