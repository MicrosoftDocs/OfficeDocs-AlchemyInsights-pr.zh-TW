# <a name="im-not-seeing-the-calendars-i-expect-when-choosing-my-availability"></a>我沒有看到預期時選擇 [我的可用性的行事曆

如果您沒有看到您所預期的行事曆，移至行事曆點選 [功能表] 按鈕，與螢幕左上角中的，確定已選取您會看到的行事曆。 選擇時間傳送可用性時，會出現僅啟用此功能表中的行事曆。
