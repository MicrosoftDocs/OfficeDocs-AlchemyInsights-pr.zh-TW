# <a name="using-the-microsoft-authenticator-app"></a>使用 Microsoft 驗證器應用程式

使用指紋、面孔識別或 PIN 碼登入您的個人 Microsoft 帳戶。

- [如何使用 Microsoft 驗證器應用程式](https://support.microsoft.com/help/4026727/microsoft-account-how-to-use-the-microsoft-authenticator-app)。 

- 透過掃描 QR 碼或[手動新增帳戶](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-add-account-manual)，以[安裝驗證器應用程式](https://docs.microsoft.com/azure/active-directory/user-help/security-info-setup-auth-app)。  

如需詳細資訊，請參閱 [驗證器應用程式概述](https://docs.microsoft.com/azure/active-directory/user-help/user-help-auth-app-overview)。
