# <a name="set-passwords-to-never-expire"></a>設定密碼永遠不過期

將密碼設為永不過期從[安全性&amp;隱私權](https://portal.office.com/adminportal/home#/settings/security)。按一下密碼原則旁的 [「 編輯 」 並以"在"開啟滑桿。
  
如需詳細資訊：[設定您的組織的密碼到期原則](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f)。
