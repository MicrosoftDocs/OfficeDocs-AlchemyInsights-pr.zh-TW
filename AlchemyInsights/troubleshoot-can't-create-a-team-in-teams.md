# <a name="what-to-do-if-you-cant-create-a-team-in-microsoft-teams"></a>當您無法在 Microsoft 小組中建立小組時，應執行的動作

請您的小組管理員執行下列作業：

1. 在 Microsoft 365 中建立或變更組織的安全性群組設定。
2. 將您新增至該安全性群組。
