# <a name="remove-an-extension-from-microsoft-edge"></a>從 Microsoft Edge 移除分機號碼

若要從 Microsoft Edge 移除分機：

1. 開啟 Microsoft Edge。
2. 選取 **> 延伸** 部分。
3. 在 [ **已安裝的分機** ] 頁面上，尋找您要移除的副檔名，然後選取它旁邊的 [ **移除** ]。
4. 在 [確認] 對話方塊中，選取 [ **移除**]。
