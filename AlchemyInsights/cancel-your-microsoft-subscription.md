# <a name="cancel-your-microsoft-subscription"></a>取消您的 Microsoft 訂閱

1. 移至 [[服務與訂閱]](https://account.microsoft.com/services/)，並使用您在購買訂閱時所用的 Microsoft 帳戶登入。

2. 尋找您的訂閱，然後按一下 **[管理]**。 如果您看到 **[開啟週期性計費]** 而不是 **[管理]**，請跳至 [疑難排解] 一節；否則請繼續進行步驟 3。。

3. 在下一頁中，根據您的訂閱類型，按一下 **[取消]** 或 **[升級或取消]**。

4. 依照頁面上的指示繼續取消。
