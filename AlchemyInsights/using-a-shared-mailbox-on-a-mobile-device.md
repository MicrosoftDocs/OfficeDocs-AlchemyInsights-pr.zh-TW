# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>在行動裝置上使用共用信箱

您無法從已安裝在行動裝置上的 Outlook app 存取共用信箱。 您想要這項功能嗎？ 讓 Outlook 工程團隊知道。 [在這裡投票](https://go.microsoft.com/fwlink/?linked=862116)!
  
您只能在行動裝置上使用共用信箱, 方法是開啟瀏覽器, 登入 Office 365, 然後移至 web 上的 Outlook。 如需相關指示, 請參閱[在 Outlook 網頁版中新增共用信箱](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)。
  
如果您必須使用行動裝置上的 Outlook 應用程式來存取共用信箱, 請考慮建立 Office 365 群組, 而不要使用共用信箱。 如需詳細資訊, 請參閱[比較群組](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx)。
