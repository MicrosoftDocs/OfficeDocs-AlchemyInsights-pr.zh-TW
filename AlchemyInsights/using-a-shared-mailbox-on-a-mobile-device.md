# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>在行動裝置上使用共用的信箱

您無法從行動裝置安裝 Outlook app 存取共用的信箱。 您想要這項功能嗎？ 可讓知道 Outlook 工程團隊。 [在這裡投票](https://go.microsoft.com/fwlink/?linked=862116)！
  
您只可以在行動裝置上使用共用的信箱，藉由開啟瀏覽器，登入 Office 365，然後移至 [網頁型 Outlook。 如需相關指示，請參閱 < <b0>Add 網頁型 Outlook 中的共用信箱</b0>。
  
如果您必須存取行動裝置上使用 Outlook 應用程式的共用的信箱，請考慮建立 Office 365 群組而不是使用共用的信箱。 如需詳細資訊，請參閱[比較群組](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx)。
