# <a name="troubleshoot-content-search-errors"></a>疑難排解內容搜尋錯誤

當您匯出搜尋結果時，是否遇到內容搜尋問題或取得失敗的問題？

例如，當您執行搜尋時，是否要接收下列專案？

- CS008 或 CS012 錯誤

- 伺服器忙碌/逾時錯誤

- 發生應用程式錯誤

或是從大量信箱（超過100000個信箱）中搜尋結果時，是否取得匯出錯誤？

針對這些類型的錯誤，請嘗試對失敗的內容位置進行搜尋。 如需詳細資訊，請參閱[本文](https://docs.microsoft.com/office365/securitycompliance/retry-failed-content-search)。
