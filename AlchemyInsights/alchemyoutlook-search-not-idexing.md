# <a name="outlook-search-not-indexing"></a>未編制索引的 Outlook 搜尋

我們知道 Outlook 搜尋問題與 Windows 的最近安全性更新相關。
  
當您在 Outlook 中搜尋時，您可能會收到下列其中一項錯誤：發生**錯誤，無法完成搜尋**，或**搜尋結果可能不完整，因為仍在編制索引的專案**。 修正是以選用的 Windows 更新形式發行。 請安裝對應至此問題的 Windows 版本： 
  
> [Windows 7-2017 年6月27日更新](https://support.microsoft.com/kb/4022168.aspx)
    
> [Windows 8.1-6 月27日，2017更新](https://support.microsoft.com/kb/4022720.aspx)
    
> [Windows 10 版本 1703-6 月27日，2017更新](https://support.microsoft.com/kb/4022716.aspx)
    
如需最新的資訊，請參閱文章： [2017 年6月的安全性更新問題 #5 中的 Outlook 已知問題](https://support.office.com/article/Outlook-known-issues-in-the-June-2017-security-updates-3F6DBFFD-8505-492D-B19F-B3B89369ED9B.aspx)。
