# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 「 拒絕存取 」、 租用戶已超出臨界值

太多垃圾郵件或大量郵件已傳送給 Office 365 組織時發生此訊息。若要移除此區塊，您需要了解並說明支援代理程式的原因，以及修正基礎的問題。
