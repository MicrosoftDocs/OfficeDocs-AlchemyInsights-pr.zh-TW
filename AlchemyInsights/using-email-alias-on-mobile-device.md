# <a name="email-aliases-on-mobile-devices"></a>在行動裝置上的電子郵件別名

電子郵件別名是另一個信箱設定的電子郵件地址，讓郵件傳送給使用者的別名將會傳遞至其信箱，將讀取其行動裝置上。 需要進行其他設定。

但是，使用者無法回覆郵件，或傳送新郵件使用的別名做為其從地址。 所有外寄郵件會從他們的主要電子郵件地址。
