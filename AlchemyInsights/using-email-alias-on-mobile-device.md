# <a name="email-aliases-on-mobile-devices"></a>行動裝置上的電子郵件別名

電子郵件別名只是在信箱上設定的另一個電子郵件地址, 因此傳送給使用者別名的郵件會傳遞至其信箱, 並可在其行動裝置上閱讀。 不需要額外的設定。

但是, 使用者無法回復郵件, 或使用別名作為寄件者位址來傳送新郵件。 所有外寄郵件都是來自其主要的電子郵件地址。
