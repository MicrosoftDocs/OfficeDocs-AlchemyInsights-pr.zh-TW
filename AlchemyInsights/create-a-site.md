# <a name="create-a-site"></a>建立網站

- 移至 [[網站] 頁面上，在 Microsoft 365 系統管理中心](https://portal.office.com/adminportal/home#/SitesList)]，然後選取 [**新增網站**。 
    
- 在新 SharePoint 系統管理中心，在 [網站] 頁面上，按一下 [**建立**]。 
    
 **附註：** 您無法建立新的網站具有與現有網站相同的 URL。 
  
如需更多有關建立網站的詳細資訊，請參閱 <<c0>建立網站。
  
如需有關資訊讓使用者能夠建立自己的網站，請參閱 <<c0>在 SharePoint Online 中的管理網站架設。
