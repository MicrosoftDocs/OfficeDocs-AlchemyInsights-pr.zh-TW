# <a name="delete-a-sharepoint-site"></a>刪除 SharePoint 網站

1. 在新 SharePoint 系統管理中心，在 [網站] 頁面上，按一下您想要刪除之網站旁的核取方塊。
    
2. 在功能區] 工具列中，按一下 [**刪除**]。
