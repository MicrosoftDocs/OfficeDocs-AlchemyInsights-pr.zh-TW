# <a name="delete-a-sharepoint-site-collection"></a>刪除 SharePoint 網站集合

1. 在傳統 SharePoint 系統管理中心中，在網站集合] 頁面上，選取網站集合] 旁邊的核取方塊。
    
2. 在 [網站集合] 索引標籤上選取 [**刪除**]。
