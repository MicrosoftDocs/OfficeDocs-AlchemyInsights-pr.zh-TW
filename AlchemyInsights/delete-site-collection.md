# <a name="delete-a-sharepoint-site-collection"></a>刪除 SharePoint 網站集合

1. 在傳統 SharePoint 系統管理中心，在 [網站集合] 頁面上選取 [網站集合] 旁的核取方塊。
    
2. 在 [網站集合] 索引標籤上選取 [**刪除**]。
