# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>如何自訂透過 SharePoint 群組的 SharePoint 網站權限 

SharePoint 群組是一群使用者所有具有相同的網站和內容的權限集合。 而非一次指派權限一位人員，您可以使用群組來輕鬆地一次將相同的權限等級指派給許多人。

[自訂 SharePoint 網站權限](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
