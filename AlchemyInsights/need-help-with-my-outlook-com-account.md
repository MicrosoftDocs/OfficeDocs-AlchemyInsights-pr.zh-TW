# <a name="need-help-with-my-outlookcom-account"></a>需要 Outlook.com 帳戶的協助

如果您需要協助 Outlook.com 服務，我們有專用協助可協助您。 若要在 web 應用程式上直接取得支援： 

1. 按一下 [是]。 圖示位於螢幕右上方。 

2. 在彈出面板上顯示的 [說明] 方塊中，輸入您需要協助的資訊，然後按一下 [ **提交**]。 

3. 若未呈現方案，請向下移動側面板，然後按一下 [以 **電子郵件與我們聯繫**]。
