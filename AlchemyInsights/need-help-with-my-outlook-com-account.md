# <a name="need-help-with-my-outlookcom-account"></a>需要協助您處理我的 Outlook.com 帳戶

如果您需要的 Outlook.com 服務的說明，我們有專用的協助來協助您。 若要取得支援直接在 web 應用程式： 

1. 按一下 [上嗎？ 在上方的圖示螢幕的權限。 

2. 輸入您需要協助上顯示在面板設定 pop 上的 [說明] 方塊，然後按一下 [**送出**。 

3. 如果未出現解決方案，請捲動到側邊面板，然後按一下 [**電子郵件給我們**。
