# <a name="delete-a-folder"></a>刪除資料夾

1. 在左窗格中，以滑鼠右鍵按一下您想要刪除，然後選取 [**刪除] 資料夾**的子資料夾。
2. 選取 **[確定**] 以確認。

若要深入了解，請參閱[刪除 Outlook.com 中的資料夾](https://go.microsoft.com/fwlink/p/?linkid=873134)。
