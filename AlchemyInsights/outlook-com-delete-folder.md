# <a name="delete-a-folder-in-outlookcom"></a>刪除在 Outlook.com 中的資料夾

1. 在左窗格中，以滑鼠右鍵按一下您想要刪除，然後選取 [**刪除] 資料夾**的子資料夾。
2. 選取 **[確定**] 以確認。

若要深入了解，請參閱[Working with Outlook.com 中的郵件資料夾](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)。
