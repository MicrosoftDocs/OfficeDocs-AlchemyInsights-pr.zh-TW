# <a name="policy-error-when-sharing-a-calendar"></a>共用行事歷時的原則錯誤

如果您嘗試共用您的行事曆，而且收到錯誤：「原則不允許將此層級的許可權授與一或多個收件者」，請參閱[本主題](https://support.microsoft.com/help/3187524/policy-does-not-allow-granting-permissions-at-this-level-to-one-or-mor)。
