# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>在 SharePoint 或 OneDrive 中的變更顯示語言設定 

您不能變更 SharePoint 或 OneDrive 的預設語言設定，因為這些設定已定義當您建立您的 Office 365 訂閱。 不過，您可以變更顯示語言設定。

若要變更您個人的設定，請參閱[變更您的個人語言和地區設定](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7)。

若要變更網站的使用者介面，請參閱[選擇您想要用於網站的使用者介面的語言](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8)。

如需疑難排解語言設定的詳細資訊，請參閱[您無法變更 SharePoint Online 網站集合的預設語言](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)。
