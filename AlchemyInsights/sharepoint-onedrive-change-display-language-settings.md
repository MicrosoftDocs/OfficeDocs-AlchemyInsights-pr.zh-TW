# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>在 SharePoint 或 OneDrive 中變更顯示語言設定 

您無法變更 SharePoint 或 OneDrive 的預設語言設定，因為當您建立 Microsoft 365 訂閱時，就會定義這些設定。 不過，您可以變更您的顯示語言設定。

若要變更您的個人設定，請參閱[變更您的個人語言和地區設定](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7)。

若要變更網站的使用者介面，請參閱[選擇您要提供給網站使用者介面的語言](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8)。

如需疑難排解語言設定的相關資訊，請參閱[您無法變更 SharePoint Online 網站集合的預設語言](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)。
