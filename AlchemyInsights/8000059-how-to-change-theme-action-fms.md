# <a name="how-to-change-your-theme"></a>如何變更您的佈景主題

若要變更您的佈景主題 Outlook.com 或網路上的 Outlook 中，移至 [**設定**![設定](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png)並選擇 [**佈景主題**的佈景主題。如需所有可用的佈景主題，請選取 [**檢視所有**的 [**佈景主題**。
