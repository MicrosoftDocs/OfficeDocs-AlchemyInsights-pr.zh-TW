# <a name="how-to-change-your-theme"></a>如何變更您的佈景主題

若要變更您 Outlook.com 或網路上的 Outlook 中的佈景主題，請移至 [**設定**並選擇 [**佈景主題**的佈景主題。如需所有可用的佈景主題，請選取 [**檢視所有**的 [**佈景主題**。
