# <a name="learn-about-using-public-folders-in-outlook"></a>了解如何使用 Outlook 中的公用資料夾

在 Outlook 中的公用資料夾可讓您很理想的收集、 組織及分享資訊特定主題或組織內的專案。 如需詳細資訊，請參閱[建立及共用在 Outlook 中的公用資料夾](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97)。
