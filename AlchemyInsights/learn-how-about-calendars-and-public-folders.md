# <a name="learn-about-using-public-folders-in-outlook"></a>深入瞭解如何在 Outlook 中使用公用資料夾

Outlook 中的公用資料夾可讓您收集、組織及共用組織內特定主題或專案的相關資訊。 如需詳細資訊，請參閱 [在 Outlook 中建立及共用公用資料夾](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97)。
