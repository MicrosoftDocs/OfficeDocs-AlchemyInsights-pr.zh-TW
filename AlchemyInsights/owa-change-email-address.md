# <a name="change-your-email-address"></a>變更您的電子郵件地址 

若要變更您的電子郵件地址，您需要系統管理員許可權。
  
依照[變更 Microsoft 365 中的使用者名稱和電子郵件地址](https://docs.microsoft.com/office365/admin/add-users/change-a-user-name-and-email-address)中的指示進行。
