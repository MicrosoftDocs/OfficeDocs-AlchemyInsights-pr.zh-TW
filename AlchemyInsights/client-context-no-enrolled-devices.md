# <a name="client-context---no-enrolled-devices"></a>用戶端內容-沒有註冊的裝置

您的租用戶已使用 Intune 裝置沒有行動裝置管理 (MDM) 註冊。 如果您想要註冊並管理您的裝置透過 Intune MDM，請檢閱文件[這裡](https://docs.microsoft.com/intune/device-enrollment)所需的平台。 如果您使用協力廠商 MDM 或僅應用程式防護功能可能不需要執行任何動作。
