# <a name="how-to-change-your-theme"></a>如何變更您的佈景主題

若要變更您 Outlook.com 或網頁型 Outlook 中的佈景主題，請移至 [**設定**，然後選擇 [**佈景主題**的佈景主題。 若要查看所有可用的佈景主題，請選取**佈景主題**] 下的**所有的檢視**。
