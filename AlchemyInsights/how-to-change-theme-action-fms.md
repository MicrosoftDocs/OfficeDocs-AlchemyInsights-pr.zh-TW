# <a name="how-to-change-your-theme"></a>變更主題

若要變更 Outlook.com 或 Outlook 網頁版中的主題，請移至 [ **設定** ]，然後選擇 [ **主題**] 底下的主題。 若要查看所有可用主題，請選取 [所有**主題**] 底下的 [**全部查看**]。
