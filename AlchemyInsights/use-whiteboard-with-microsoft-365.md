# <a name="use-whiteboard-with-microsoft-365"></a>在 Microsoft 365 中使用 Whiteboard

Microsoft Whiteboard 是將人員、內容和構想集合一處的自由格式數位畫布。 

- 以 Microsoft 365 系統管理員身分[啟用 Whiteboard](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_07)。 

- 使用[搭配 Microsoft Teams 的 Whiteboard](https://support.microsoft.com/office/use-whiteboard-in-microsoft-teams-7a6e7218-e9dc-4ccc-89aa-b1a0bb9c31ee)。 

- 與其他人在 Whiteboard 上[共同作業](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_27)。 

如需詳細資訊，請參閱 [Microsoft Whiteboard 說明](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920)。
