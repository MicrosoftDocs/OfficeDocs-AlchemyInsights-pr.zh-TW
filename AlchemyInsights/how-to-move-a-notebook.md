# <a name="how-to-move-a-notebook"></a>How to Move 筆記本

* 若要將 OneNote 筆記本移至 OneDrive，前往 [**檔案** > **共用** > **移動筆記本**中 OneNote 2016。 如需詳細資訊請參閱[移到 OneDrive 的 OneNote 筆記本](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082)

* 如需有關最佳作法準備移動筆記本時，請參閱[OneNote 同步處理的最佳作法](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices)

* [移動已與其他人共用 Windows 版 OneNote 筆記本](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4)移動共用的筆記本提供詳細資訊。
