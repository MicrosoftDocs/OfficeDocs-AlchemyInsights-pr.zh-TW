# <a name="self-service-password-reset-sspr"></a>自助式密碼重設 (SSPR)

- 請按一下此處，[讓使用者重設自己的密碼](https://admin.microsoft.com/Adminportal/Home#/featureexplorer/security/Sspr)。  根據預設，系統管理員可使用 SSPR 功能。

- 請按一下這裡來[註冊您的 SSPR 連絡人資訊](https://go.microsoft.com/fwlink/?linkid=849451)。

如需詳細資訊：

基本 SSPR 設定 - [讓使用者重設自己的密碼](https://docs.microsoft.com/microsoft-365/admin/add-users/let-users-reset-passwords?view=o365-worldwide)

[SSPR 常見問題集](https://docs.microsoft.com/azure/active-directory/authentication/active-directory-passwords-faq)

[SSPR 疑難排解員](https://docs.microsoft.com/azure/active-directory/authentication/active-directory-passwords-troubleshoot)
