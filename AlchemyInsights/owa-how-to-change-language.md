# <a name="change-language"></a>變更語言

1.    登入 outlook 網頁版。
2. 在頁面頂端，選取 [**設定** > **檢視所有的 Outlook 設定**。
3. 選取 [**一般**] > **語言和時間**。
4. 選取您想要使用的語言。
5. 選取**儲存**。
 
若要深入了解，請參閱[變更的時區和網頁型 Outlook 中的語言設定](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273)。
