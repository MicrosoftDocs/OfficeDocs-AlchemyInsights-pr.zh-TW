# <a name="already-paid"></a>已支付？

如果您已傳送付款, 您可以讓我們知道:
  
1. 移至 [[訂閱](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面, 或選擇 [**帳單** \> **訂閱**]。

2. 選擇訂閱, 然後在 [**我的帳單**] 區段中, 選擇 [**查看帳單**]。

3. 在 [**帳單明細**] 頁面上, 選擇 [**已支付]。** 系統會要求您輸入與所寄付款相關的資訊。

4. 輸入所有必要的資訊後, 選擇 [**傳送**]。
