# <a name="already-paid"></a>已付費吗？

如果您已經送出付款，讓我們知道您可以：
  
1. 移至 [[訂閱](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面或選擇 [**計費** \> **訂閱**。
    
2. 選擇 [訂閱，並在 [**我 bill** ] 區段中選擇 [**檢視 Bill**。
    
3. 在 [ **Bill 詳細資料**] 頁面上選擇 [**已經付費吗？** 系統會要求您輸入已傳送的付款的相關資訊。 
    
4. 選擇 [**傳送**] 時已輸入的所有必要的資訊。
