# <a name="help-understanding-value-added-tax-vat"></a>協助瞭解加值稅 (加值稅)

如果您所在的國家 (地區) 使用加值稅 (加值稅), 例如英國, Microsoft 開具發票會為您計算加值稅, 並將金額套用至您的發票。
  
基於納稅原因, 您也必須在開具發票設定中指定您的 加值稅 註冊號碼。
  
您無法新增或移除 加值稅 比率, 但是您可以在 [設定] 中變更預設的加值稅率。 根據預設, 開具發票會使用服務來驗證 加值稅 註冊號碼, 但我們建議您閱讀歐洲委員會所發行的[VIES 免責聲明](https://go.microsoft.com/fwlink/?LinkID=841741)。 如果您選擇停用服務, 則必須驗證所有 加值稅 註冊號碼。
