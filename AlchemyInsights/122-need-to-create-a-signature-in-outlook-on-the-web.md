# <a name="need-to-create-an-email-signature"></a>需要建立電子郵件簽章吗？

如果您有 Outlook 帳戶或 Outlook web （Office 365 工作或學校） 帳戶，您可以建立簽章與自動將它新增至所有外寄郵件，或選擇性地將其新增至特定的外寄郵件。請注意您的電子郵件簽章不同 Outlook 和 Outlook web 上。若要建立電子郵件簽章，請參閱[建立與 outlook 訊息中新增簽章](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)或[建立在網路上的 Outlook 中新增電子郵件簽章](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)。
