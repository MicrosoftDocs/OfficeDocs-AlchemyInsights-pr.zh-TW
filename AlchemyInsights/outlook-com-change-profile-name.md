# <a name="change-your-profile-name-in-outlookcom"></a>在 Outlook.com 中變更您的設定檔名稱

1. 移至[您的 Microsoft 帳戶資訊](https://go.microsoft.com/fwlink/p/?linkid=860841)。
2. 選取 [**編輯名稱**]。
3. 輸入名字和姓氏, 然後選取 [**儲存**]。

您的設定檔名稱變更生效時, 請等候最多24小時。
