# <a name="change-your-profile-name-in-outlookcom"></a>變更 Outlook.com 中的設定檔名稱

1. 移至[您的 Microsoft 帳戶資訊](https://go.microsoft.com/fwlink/p/?linkid=860841)。
2. 選取 [**編輯名稱**]。
3. 輸入名字和姓氏，然後選取 [**儲存**]。

請允許最多24小時的變更設定檔名稱生效。
