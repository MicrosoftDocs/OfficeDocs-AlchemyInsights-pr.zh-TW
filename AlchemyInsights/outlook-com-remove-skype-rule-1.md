# <a name="skype-icon-missing"></a>遺失 Skype 圖示

我們已暫時將 Skype 圖示從 Outlook.com 中移除, 做為設計重新整理的一部分, 而功能更新則非常高興。我們在準備新的和改善的體驗時, 感謝您的耐心。

您仍然可以在[web.skype.com](https://web.skype.com/)中使用 Skype 流覽瀏覽器。
