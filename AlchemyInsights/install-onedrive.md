OneDrive 同步處理用戶端會隨 Windows 10 一起預先安裝。 若要安裝 OneDrive for Windows 7 或 Mac 版, 或為您的手機或平板電腦取得行動裝置應用[http://onedrive.com/download](http://onedrive.com/download)程式, 請移至。
