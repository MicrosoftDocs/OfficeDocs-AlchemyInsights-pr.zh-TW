# <a name="install-onedrive"></a>安裝 OneDrive

OneDrive sync 用戶端會與 Windows 10 預先安裝。若要安裝 OneDrive for Windows 7 或 Mac，或取得行動裝置應用程式的電話或平板電腦，移至[http://onedrive.com/download](http://onedrive.com/download)。
