# <a name="onedrive-puid-mismatch"></a>OneDrive PUID 不符
您似乎在商務 OneDrive 中遇到[PUID 不符](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-onedrive-site)的情況。 請依照下列步驟進行，我們可能會儘快協助您：

1. 按一下 [**連絡人支援**]，然後按一下 [**新增服務要求**]。
2. **在 OneDrive 網站上**，將標題和描述保留為 PUID 不相符。
3. 送出票證後，請使用下列資訊更新它：
    - 受影響的 OneDrive 網站 URL。
    - 受影響使用者的使用者主要名稱（UPN）。
