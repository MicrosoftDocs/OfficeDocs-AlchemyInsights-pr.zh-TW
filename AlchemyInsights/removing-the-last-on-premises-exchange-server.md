# <a name="removing-the-last-on-premises-exchange-server"></a>移除最後一個內部部署 Exchange 伺服器

在從內部部署組織中移除最後一部 Exchange 伺服器之前, 請務必先瞭解其含義, 以協助您決定是否需要最後一部伺服器進行任何其他用途。 如需詳細資訊, 請參閱在[混合式部署中如何以及何時解除委任您的內部部署 Exchange 伺服器](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx)。
