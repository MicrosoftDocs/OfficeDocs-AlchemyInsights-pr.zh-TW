# <a name="search-experience-updated"></a>更新的搜尋體驗

搜尋現在是位於底端的 [應用程式。 點選上它可讓您直接存取上方的連絡人、 即將來臨的旅遊行程、 封裝傳遞和最近使用的附件。 您可以在任何您的帳戶從一個單一搜尋方塊中搜尋的電子郵件、 檔案及連絡人，或使用我們新的篩選，以縮小搜尋範圍。
