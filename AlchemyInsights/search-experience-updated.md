# <a name="search-experience-updated"></a>已更新搜尋體驗

搜尋現在位於應用程式的底部。 點擊它可讓您直接存取您的主要連絡人、即將到來的旅行行程、封裝傳遞和最近的附件。 您可以在單一搜尋方塊中搜尋任何帳戶中的電子郵件、檔案和連絡人, 或使用新的篩選器來縮小搜尋範圍。
