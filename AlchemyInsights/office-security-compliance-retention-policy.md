# <a name="more-info-about-retention-policies"></a>保留原則的詳細資訊

大多數的組織、 磁碟區和其資料的複雜度會增加每日-電子郵件，如文件、 立即訊息，等等。 有效率地管理或管理這項資訊非常重要，並保留原則可協助您達成這些目標。 如需有關保留原則的詳細資訊，請造訪這些連結：

## <a name="a-from-security-and-compliance-center"></a>A） 從安全與規範中心

- 移至安全性與合規性 > 資料控管 > 保留
  - [請參閱本文](https://docs.microsoft.com/office365/securitycompliance/retention-policies)

- 移至安全性與合規性 > eDiscovery> 進階電子文件 
  - [請參閱本文](https://docs.microsoft.com/office365/securitycompliance/ediscovery-cases)

- 移至安全性與合規性 > 分類 > 保留
  - 請參閱下列文章
    - [敏感度標籤](https://docs.microsoft.com/office365/securitycompliance/sensitivity-labels)
    - [標籤概觀](https://docs.microsoft.com/office365/securitycompliance/labels)

## <a name="b-legacy-ediscovery-policies"></a>B） 舊版 eDiscovery 原則

- [請參閱本文](https://support.office.com/article/Set-up-an-eDiscovery-Center-in-SharePoint-Online-A18F8975-AA7F-43B4-A7D6-001D14744D8E)

## <a name="c-site-closure-and-deletion-policies"></a>C） 網站關閉和刪除原則

- [請參閱本文](https://support.office.com/article/Use-policies-for-site-closure-and-deletion-A8280D82-27FD-48C5-9ADF-8A5431208BA5)  

## <a name="d-information-rights-management-irm"></a>D） 資訊版權管理 (IRM)

- 請參閱下列文章
  - [清單/文件庫上的 IRM](https://support.office.com/article/apply-information-rights-management-to-a-list-or-library-3bdb5c4e-94fc-4741-b02f-4e7cc3c54aa1)
  - [IRM 概觀](https://support.office.com/article/create-and-apply-information-management-policies-eb501fe9-2ef6-4150-945a-65a6451ee9e9)

> [!Note]
> 我們強烈建議將移至安全性與合規性原則如果您仍在使用舊版 eDiscovery 或 IRM 或網站關閉和刪除原則。
