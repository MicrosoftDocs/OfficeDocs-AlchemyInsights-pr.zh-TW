# <a name="creating-inbox-rules-for-shared-mailboxes"></a>建立共用信箱的收件匣規則

您可以在具有存取權的共用信箱上新增規則, 以與如何將規則新增至自己帳戶的方式類似。
  
請確定您是相關共用信箱的成員, 然後依照本文中的步驟進行:[將規則新增至共用信箱](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164)
