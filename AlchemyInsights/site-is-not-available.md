# <a name="site-is-not-available"></a>網站不是可用

如果您的 SharePoint 或 OneDrive 網站不是提供給多位使用者，可能會暫時的服務問題。 檢查[服務健康狀況儀表板](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)若要查看您的租用戶會受到影響。
