# <a name="site-is-not-available"></a>網站不提供

若未提供給多位使用者將 SharePoint 或 OneDrive 網站，則有可能暫時服務問題。檢查[服務健康狀況儀表板](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)查看您的租用戶受影響。
