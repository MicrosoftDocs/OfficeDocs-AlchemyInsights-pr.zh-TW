# <a name="site-is-not-available"></a>無法使用網站

如果您的 SharePoint 或 OneDrive 網站無法供多位使用者使用，可能會發生暫時的服務問題。 檢查[服務健康情況儀表板](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)，查看您的承租人是否受到影響。
