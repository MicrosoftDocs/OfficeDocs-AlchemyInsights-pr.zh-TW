# <a name="import-contacts"></a>匯入連絡人

之前將連絡人匯入 Outlook.com，您需要將它們匯出從其他電子郵件提供者為.csv 檔案。 然後，遵循下列指示。
  
1. 然後在頁面底部，選取 [**人員**。 
    
2. 在功能表列中，選取 [**管理** \> **匯入連絡人]**。 
    
3. 選取 [ **Gmail** （即使您的.csv 檔案不可能有來自 Gmail） 因為 Gmail 使用.csv 檔案類型。 
    
4. 選取 [**瀏覽]** 瀏覽至您儲存您匯出的連絡人資料夾，選取 [**開啟**]。 
    
若要深入了解，請參閱 <<c0>將連絡人匯入到 Outlook.com 帳戶。
