# <a name="public-folder-access-from-mobile-devices"></a>移動裝置的公用資料夾存取權

無法從行動裝置存取公用資料夾。
  
如果您想要具有與公用資料夾類似的功能，並從行動裝置存取，請參閱 [瞭解 Microsoft 365 群組](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) 以取得更好的替代。
