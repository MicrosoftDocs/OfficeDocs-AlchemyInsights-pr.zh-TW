# <a name="public-folder-access-from-mobile-devices"></a>從行動裝置存取公用資料夾

公用資料夾不是從行動裝置可存取的。
  
如果您想要有類似的公用資料夾的功能和從行動裝置的裝置存取，請參閱[了解 Office 365 群組](https://support.office.com/en-us/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)較佳的替代方法。
