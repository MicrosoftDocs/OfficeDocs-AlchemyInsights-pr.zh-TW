# <a name="block-pop-up-windows-in-microsoft-edge"></a>在 Microsoft Edge 中封鎖快顯視窗

1. 前往 **... (設定，以及更多)**  >  **設定**  >  **網站許可權**。
2. 選取 [ **快顯視窗] 和**[重新導向]。
3. 開啟 **區塊**。
