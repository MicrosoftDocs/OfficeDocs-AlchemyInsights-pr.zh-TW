# <a name="suspend-or-pause-licenses"></a>暫止或暫停授權

若要暫止或暫停使用者授權，您可以從 **[Microsoft 365 系統管理中心]** 移除授權，以後再重新加入。

1. 在將授權從您的訂閱中移除之前，您必須先從使用者[取消指派](https://docs.microsoft.com/microsoft-365/admin/manage/remove-licenses-from-users?view=o365-worldwide)。

2. 然後移至 **[帳單] > [[您的產品]](https://go.microsoft.com/fwlink/p/?linkid=842054)**。

3. 在 **[您的產品]** 頁面上，找到您要移除授權的訂閱，然後按一下 **[新增/移除授權]**。

4. 在 [授權總數] 方塊中，輸入此訂閱所需的授權總數，然後選擇 **[提交變更]**。 例如，如果您有 100 個授權，且想要暫止或暫停 50 個，請輸入 50 做為訂閱的授權總數。

如需詳細資訊，請參閱[移轉訂閱授權](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide)。
