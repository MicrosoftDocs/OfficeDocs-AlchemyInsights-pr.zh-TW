# <a name="all-subscriptions-are-active"></a>所有的訂閱是作用中

您的訂閱是作用。 這可以移至 [[訂閱](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面上，或選擇 [**計費**驗證\>**訂用帳戶**。 依狀態，與使用中的訂閱，在頂端排序您的訂閱。 其他資訊，例如授權計數和付款方法，可以看到所選擇訂閱。
  
如果您遇到問題與您的服務，使用 「 讓我們撥話給您 」 選項按下以講話支援工程師。
