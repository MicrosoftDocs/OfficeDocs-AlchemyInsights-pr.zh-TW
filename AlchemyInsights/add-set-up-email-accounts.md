# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>設定在 Outlook 中的 Exchange 電子郵件帳戶的問題

如果您遇到問題設定在 Outlook 中的 Exchange 帳戶，您可以在受影響的電腦上執行[Outlook 帳戶安裝程式工具](https://aka.ms/SaRA-OutlookSetupProfile)。 它會執行各種檢查，並提供可能的解決方案，您的帳戶設定問題。
  
如果您仍然發生問題，請參閱[Outlook 無法設定新的設定檔所使用的 Office 365 中的 Exchange Online 信箱的 Exchange 自動探索](https://support.microsoft.com/help/2404385/outlook-can-t-set-up-a-new-profile-by-using-exchange-autodiscover-for)。
