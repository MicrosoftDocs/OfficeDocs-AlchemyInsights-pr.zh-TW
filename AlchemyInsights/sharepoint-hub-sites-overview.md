# <a name="sharepoint-hub-sites-overview"></a>SharePoint 中樞站台概觀

SharePoint 中樞站台說明連線，並將組織網站符合組織的需求取決於專案、 部門、 部門、 區域、 更輕鬆的等：

- 探索相關的內容，例如新聞和其他網站的活動。


- 適用於一般瀏覽和相關聯的網站之間的品牌。


- 搜尋所有相關聯的網站。


如需如何從新的系統管理中心或 Microsoft PowerShell 建立中樞站台詳細資訊，請參閱[建立 SharePoint Online 中中樞站台](https://docs.microsoft.com/en-us/sharepoint/create-hub-site)。 

如需如何從新的系統管理中心或 PowerShell 將現有的網站關聯至中樞站台詳細資訊，請參閱[建立具有中樞站台的 SharePoint 網站的關聯](https://support.office.com/en-us/article/associate-a-sharepoint-site-with-a-hub-site-ae0009fd-af04-4d3d-917d-88edb43efc05)。  

如需如何從新的系統管理中心或 PowerShell 移除現有的中樞站台詳細資訊，請參閱[移除中樞站台](https://docs.microsoft.com/en-us/sharepoint/remove-hub-site)。
