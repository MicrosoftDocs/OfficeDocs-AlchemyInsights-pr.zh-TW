# <a name="prevent-microsoft-search-in-bing-from-installing-with-microsoft-365-apps"></a>防止 Bing 的 Microsoft 搜尋使用 Microsoft 365 應用程式安裝

若要防止在 Bing 使用 microsoft 365 應用程式安裝 microsoft Search 的後臺服務，請使用下列其中一個選項：

- Office 部署工具或群組原則
- Microsoft Endpoint Configuration Manager (目前的分支) 或 Microsoft Intune

若要深入瞭解，請參閱 [如何在 Bing 使用 microsoft 365 應用程式安裝的 Microsoft Search 的後臺服務](https://go.microsoft.com/fwlink/?linkid=2151946)。
