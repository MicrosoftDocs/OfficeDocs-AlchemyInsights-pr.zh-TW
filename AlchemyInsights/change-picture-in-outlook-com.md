# <a name="change-your-account-picture"></a>變更您的帳戶圖片

1. 移至[您的 Microsoft 帳戶的資訊](https://go.microsoft.com/fwlink/p/?linkid=860841)。
    
2. 選取 [**變更圖片**。 
    
3. 選取**新的圖片**。 
    
4. 選擇您想要使用並選取 [**開啟**] 的圖片。 
    
5. 拖曳或調整大小圖像] 方塊中，並選取 [**儲存**]。 
    
請讓 24 小時的時間，變更您的設定檔相片或移入效果的名稱。
