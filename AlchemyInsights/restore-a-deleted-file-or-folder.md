# <a name="restore-a-deleted-file-or-folder"></a>還原已刪除的檔案或資料夾

1. 移至 SharePoint 網站或 OneDrive。
    
2. 按一下左窗格中的 [**資源回收筒**]。如果看不到資源回收筒，按一下 [設定] 圖示] 和 [**網站內容**。資源回收筒是在最右邊的頂端之命令列。
    
3. 按一下左邊的 [您想要還原的項目。
    
4. 按一下 [**還原**]。
    
刪除的項目會保留 93 天。如果您沒有看到您要尋找的項目，則可能會有已此資源回收筒中已刪除並移至網站集合資源回收筒。若要檢查，請按一下**第二階段資源回收筒**連結下方的 [資源回收筒]。 
  
如果您仍找不到項目可以在如果記錄已啟用搜尋稽核記錄。[Office 365 安全性搜尋稽核記錄&amp;規範中心](https://support.office.com/article/0d4d0f35-390b-4518-800e-0c7ec95e946c.aspx)
