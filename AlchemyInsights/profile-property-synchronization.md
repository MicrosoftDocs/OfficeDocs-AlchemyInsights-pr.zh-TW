# <a name="profile-property-synchronization"></a>設定檔屬性同步處理

您不能將對應至使用者設定檔應用程式從 Azure Active Directory 同步處理 AD 匯入的其他屬性。 AD 匯入僅限於預先設定的一組屬性，以保證一致的計時器工作的效能。
