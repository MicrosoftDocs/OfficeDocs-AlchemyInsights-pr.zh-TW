# <a name="ios-provisioning-profiles"></a>iOS 布建設定檔

Apple iOS 企業營運使用提供設定檔的概念, 用來驗證應用程式的完整性, 並強制執行設定檔中定義的原則。 這些布建設定檔會在12個月後到期, 因此新的設定檔必須與已部署的應用程式相關聯, 應用程式才能繼續執行。
  
下列文章說明如何建立將新的布建設定檔與透過 Intune 部署的 iOS LOB 應用程式建立關聯:[使用 ios 應用程式布建設定檔來防止您的應用程式過期](https://docs.microsoft.com/intune/app-provisioning-profile-ios)
