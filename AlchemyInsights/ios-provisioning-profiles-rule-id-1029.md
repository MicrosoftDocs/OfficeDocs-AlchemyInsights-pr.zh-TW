# <a name="ios-provisioning-profiles"></a>iOS 佈建的設定檔

Apple iOS-營運使用佈建的設定檔是用來驗證應用程式的完整性，並強制執行原則所定義的設定檔中的概念。 這些佈建設定檔到期後的 12 個月，因此新設定檔必須是為了讓若要繼續執行的應用程式的已部署應用程式相關聯。
  
下列文章說明如何建立新的佈建設定檔與 iOS 透過 Intune 部署的 LOB 應用程式的關聯：[使用 iOS app 佈建設定檔，以防止您從過期的應用程式](https://docs.microsoft.com/intune/app-provisioning-profile-ios)
