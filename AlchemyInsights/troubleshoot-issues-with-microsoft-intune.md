# <a name="troubleshoot-issues-with-microsoft-intune"></a>疑難排解 Microsoft intune 的問題

請檢閱下面列出現在解決問題的資源。
  
1. 使用[Intune 疑難排解入口網站](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade)來診斷及解決某些常見失敗。請檢閱[本文](https://docs.microsoft.com/intune/help-desk-operators)件如需詳細資訊。  
2. 檢閱[Intune TechNet](https://social.technet.microsoft.com/forums/home?forum=microsoftintuneprod)來尋找答案與一般問題的解決方案。  
3. 若要查看之服務的狀態[服務健康狀況](https://portal.office.com/AdminPortal/Home#/servicehealth)先取出。   
4. 檢閱[Intune 文件](https://docs.microsoft.com/intune/)的規劃、 用法、 疑難排解文章等等。
