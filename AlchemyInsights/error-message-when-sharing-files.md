# <a name="error-messages-when-sharing"></a>共用時的錯誤訊息

如果您收到錯誤訊息嘗試與外部使用者共用時，未啟用很有可能外部共用或共用設定為不允許 [嘗試共用使用者層級。
  
在 [* 傳統 * SharePoint 管理置中，按一下 [**共用**] 的左窗格中。選擇 [共用您的層級想要允許。請確定外部共用網站已開啟。 
  
如果嘗試登入使用者會收到錯誤訊息，請嘗試下列步驟：
  
- 請確定使用者具有接受傳送共用邀請。
    
- 請確定使用者接受邀請中的已傳送至電子郵件地址。
