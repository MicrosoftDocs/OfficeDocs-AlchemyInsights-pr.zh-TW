# <a name="error-messages-when-sharing"></a>共用時的錯誤訊息

如果嘗試與外部使用者共用時，您會收到錯誤訊息，不會啟用最可能的外部共用或共用設為不允許 [嘗試共用使用者層級。
  
在新的 SharePoint 系統管理中心，按一下 [的左窗格中的 [**共用**]。 選擇 [共用您的層級想要允許]。 請確定外部共用網站已開啟。 
  
如果嘗試登入的使用者會收到錯誤訊息，請嘗試下列步驟：
  
- 請確認使用者已接受傳送共用邀請。
    
- 請確認使用者已接受邀請已傳送至電子郵件地址。
