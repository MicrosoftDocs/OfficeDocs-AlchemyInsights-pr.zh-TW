# <a name="create-a-communication-site"></a>建立通訊網站

若要建立通訊網站，請完成下列步驟： 
  
1. 在頁面左上角，按一下應用程式啟動器並選取 [SharePoint] 磚。如果您沒有看到 [SharePoint] 磚，按一下 [**網站**] 或 [**所有程式]、** 如果看不到 SharePoint。 
    
2. 在 [SharePoint] 頁面的上方，按一下 [**建立網站**並選擇**通訊網站**。請遵循網站建立精靈中的步驟。 
    
 **請注意**： 如果您未看見**建立網站**的選項，自助網站架設可能會停用 Office 365 中。連絡人管理 Office 365 組織中建立小組網站的人員。如果您是租用戶系統管理員，請參閱[管理 SharePoint Online 啟用自助網站架設您的組織中的網站架設。](https://go.microsoft.com/fwlink/?linkid=2018780)
  
 **附註：** 如果您嘗試建立通訊網站 URL 快速附加至隨機號碼，很可能具有該相同的 URL 的網站已經存在的 SharePoint 環境中或已遭刪除和在 [刪除的網站 （資源回收筒） 仍然存在。使用[https://aka.ms/Get-SPODeletedSite](https://aka.ms/Get-SPODeletedSite)用來評估資源回收筒及[https://aka.ms/Remove-SPODeletedSite](https://aka.ms/Remove-SPODeletedSite)移除資源回收筒中的網站。
