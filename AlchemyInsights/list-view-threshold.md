# <a name="list-view-threshold"></a>清單檢視閾值

SharePoint 清單和文件庫能容納多達 3000 萬個項目，但當項目超過 5000 個時，您可能會在使用這些項目時看到清單檢視閾值錯誤。 此閾值存在的目的是為了維持服務效能。 您無法對它進行變更。 當您嘗試共用或中斷資料夾的繼承時，也可能會收到此錯誤。

若要避免達到此閾值，請造訪：

- [管理 SharePoint 中的大型清單和文件庫](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59)

- [錯誤共用/中斷繼承](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance)
