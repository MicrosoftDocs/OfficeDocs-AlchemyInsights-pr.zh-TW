# <a name="deploy-microsoft-365"></a>部署 Microsoft 365

[Microsoft 365 部署顧問](https://go.microsoft.com/fwlink/?linkid=2072646)會在設定生產力工具、安全性原則及裝置管理功能時，為商業客戶提供指導方針。 使用 Microsoft 365 商務版 Premium 或 Microsoft 365 for enterprise 訂閱，您可以使用此顧問來設定和設定組織的裝置。

您將會收到對資源的指導和存取權，讓您的雲端服務、更新裝置至最新支援的 Windows 10 版本，以及將裝置加入 Azure Active Directory (Azure AD) ，所有功能都位於一個集中位置。
