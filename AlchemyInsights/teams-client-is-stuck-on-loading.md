# <a name="teams-client-is-stuck-on-loading"></a>團隊用戶端在載入時停滯

Microsoft 已注意到最近小組的「桌面應用程式組建」的錯誤，可防止使用者登入。 受影響的使用者會看到 [載入 Microsoft 小組] 訊息出現。

使用小組 web 應用程式的時間，最簡單的解決方法 <https://teams.microsoft.com/> 。

如需詳細資訊和目前的狀態，請參閱服務健康情況儀表板上的文章 <https://admin.microsoft.com/AdminPortal/Home#/servicehealth/advisories/:/alerts/TM236316>
