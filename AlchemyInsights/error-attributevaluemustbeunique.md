2. <span data-ttu-id="0e898-108">找出哪個物件應繼續有重複的值並不應該哪個物件。</span><span class="sxs-lookup"><span data-stu-id="0e898-108">Identify which object should continue to have the duplicated value and which object should not.</span></span>
