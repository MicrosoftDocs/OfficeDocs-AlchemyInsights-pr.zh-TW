# <a name="search-for-email-in-the-archive-mailbox"></a>搜尋封存信箱中的電子郵件

當封存信箱啟用 Microsoft 365 時，您可以使用**EDiscovery 搜尋和匯出**，以在主要和封存信箱中探索內容。 如需使用**EDiscovery 搜尋匯出**來尋找該內容的詳細資訊，請遵循[這裡](https://docs.microsoft.com/office365/securitycompliance/export-search-results)所述的步驟。
  
您也可以將信箱新增至保留原則，以保留該內容。 這會將內容同時保留在主要和封存信箱中。 如需建立保留原則以保留電子郵件的詳細資訊，請參閱[此處](https://docs.microsoft.com/Office365/securitycompliance/retention-policies)所述的步驟。
