# <a name="how-to-enabledisable-autosave"></a>如何啟用/停用自動儲存

如果您要針對所有檔案將 [自動儲存] 預設為開啟或關閉，請移至 [檔案 **]**  >  [選項 **]**  >  [儲存 *]*，並勾選或取消勾選 [預設為在 <application> 上自動儲存 OneDrive 與 SharePoint Online 檔案 **]** 旁的核取方塊。 重新啟動應用程式 (Excel、Word、PowerPoint 等)，變更即會生效。 

如需詳細資訊，請參閱[什麼是自動儲存？](https://support.microsoft.com/topic/what-is-autosave-6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5?ui=en-us&rs=en-us&ad=us)。
