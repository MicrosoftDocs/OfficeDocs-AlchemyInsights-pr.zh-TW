# <a name="create-email-signature-in-outlook-on-the-web"></a>在 web 上的 Outlook 中建立電子郵件簽名

1. 登入 Outlook 網頁版。
2. 在上方導覽列上，選擇 [**設定**  >  **郵件**]。
3. 在 [**選項**] 面板的 [**郵件**] 下，選擇 [**版面** 配置  >  **電子郵件簽名**]。
4. 在文字方塊中，輸入您的簽章並使用可用的格式設定選項，以變更其外觀。

    > [!TIP]
    > 不支援插入圖像檔案 (例如，在 .gif 或 .jpg 格式) 進行簽章，但您可以複製網頁中的影像，然後將它貼到 [簽名] 方塊中。

5. 如果您想要在所有待發電子郵件的底部顯示簽章（包括回復和轉寄），請選取 [ **在我撰寫的新郵件上自動包含我的簽名**]。
6. 按一下 **[儲存]**。
