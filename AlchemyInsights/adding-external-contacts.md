# <a name="enable-external-communications-and-add-contacts"></a>啟用外部通訊，並新增連絡人

在商務用 Skype，可以將人員新增至連絡人清單，他們就可以您只要按一下。 您新增的每一個連絡人都會獲得一或多個連絡人群組的成員資格。 如需詳細資訊檢閱[新增商務用 Skype 中的連絡人](https://support.office.com/article/add-a-contact-in-skype-for-business-89338023-2adf-4f5c-90b6-f8b6f72fadd1)。 

請確定您的租用戶系統管理員已允許[外部通訊](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users)後面的步驟。

您只想讓使用者連絡 Skype 消費者使用者嗎？ 請參閱[讓儲存商務版使用者的商務用 Skype 加入 Skype 連絡人](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts)。
