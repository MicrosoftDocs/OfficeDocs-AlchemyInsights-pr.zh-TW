# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>Proxy 位址時發生錯誤建立共用的信箱

如果您嘗試建立共用的信箱並收到這項錯誤 proxy address"SMTP:alias@domain.com"已在使用中...]，您所選擇的電子郵件地址已會採取由您組織中的另一個信箱。
  
您要尋找使用者、 群組或擁有此電子郵件地址的共用的信箱並將其刪除或變更其電子郵件地址。然後您可以建立共用的信箱搭配釋出的電子郵件地址。使用 [首頁] 頁面上的搜尋來尋找其。
  
如果您不想要刪除現有的電子郵件地址，選擇您共用信箱的新電子郵件地址。
