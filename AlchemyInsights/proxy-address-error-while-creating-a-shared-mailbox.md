# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>建立共用信箱時的 Proxy 位址錯誤

如果您嘗試建立共用信箱, 且收到此錯誤「proxy 位址 "SMTP:alias@domain.com" 已在使用 ...」, 您所選擇的電子郵件地址已由您組織中的另一個信箱所使用。
  
您需要尋找具有此電子郵件地址的使用者、群組或共用信箱, 並將它刪除或變更其電子郵件地址。 然後, 您可以使用已釋放的電子郵件地址來建立共用信箱。 使用首頁上的搜尋來尋找。
  
如果您不想要刪除現有的電子郵件地址, 請為您的共用信箱選擇新的電子郵件地址。
