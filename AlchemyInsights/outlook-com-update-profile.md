# <a name="change-my-profile-picture-or-name-in-outlookcom"></a>在 Outlook.com 中變更我的設定檔圖片或名稱

1. 移至[您的 Microsoft 帳戶資訊](https://go.microsoft.com/fwlink/p/?linkid=860841)。
1. 選取 [**編輯名稱**] 或 [**變更圖片**]。

請允許長達24小時的時間讓您的設定檔照片或名稱變更生效。
