# <a name="change-my-profile-picture-or-name"></a>變更我的設定檔圖片或名稱

1. 移至[您的 Microsoft 帳戶的資訊](https://go.microsoft.com/fwlink/p/?linkid=860841)。
1. 選取 [**編輯名稱**] 或 [**變更圖片**。

請讓 24 小時的時間，變更您的設定檔相片或移入效果的名稱。
