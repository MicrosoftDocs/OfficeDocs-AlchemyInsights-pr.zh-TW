# <a name="how-to-stop-signing-in-automatically"></a>如何停止自動登入

若要停止自動登入 Outlook.com，請在您輸入密碼的頁面上取消勾選 [**讓我保持登入**] 核取方塊。
  
若要深入瞭解，請參閱 how [to 登入或 Outlook.com](https://support.office.com/article/e08eb8ac-ac27-49f4-a400-a47311e1ee7e?wt.mc_id=Office_Outlook_com_Alchemy)。
