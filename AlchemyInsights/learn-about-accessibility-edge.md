# <a name="learn-about-accessibility-in-microsoft-edge"></a>了解 Microsoft Edge 中的協助工具

在 Microsoft Edge 中，我們引進了 [UI 自動化 API (UIA API)](https://go.microsoft.com/fwlink/?linkid=2153423)。 UIA 的變更是瀏覽器協助工具的重大投資，為依賴 Windows 10 輔助技術的使用者提供更具包容性的 Web 體驗打下基礎。 

如需深入了解，請參閱 [Microsoft Edge 中的協助工具](https://go.microsoft.com/fwlink/?linkid=2153512)。
