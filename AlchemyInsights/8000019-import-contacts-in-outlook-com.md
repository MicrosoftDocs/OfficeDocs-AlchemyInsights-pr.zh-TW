# <a name="import-contacts"></a>匯入連絡人]

之前將連絡人匯入 Outlook.com，您需要匯出成.csv 檔案在其他電子郵件供應商。然後，遵循下列指示。
  
1. 然後在頁面的底端，選取 [**人員**。 
    
2. 功能表列上，選取 [**管理** \> **匯入連絡人]**。 
    
3. 選取 [ **Gmail** （即使您.csv 檔案不可能會有來自 Gmail） 因為 Gmail 使用.csv 檔案類型。 
    
4. 選取 [**瀏覽**、 瀏覽至您儲存您匯出的連絡人資料夾並選取 [**開啟**]。 
    
若要深入了解，請參閱 ＜[給 Outlook.com 帳戶匯入連絡人](https://go.microsoft.com/fwlink/p/?linkid=873136)。
