# <a name="create-a-folder"></a>建立一個資料夾

1. 在左窗格中，選取 [旁**資料夾****建立新的資料夾**。 
    
2. 在新的 [資料夾] 方塊中，輸入資料夾名稱，然後按 Enter 鍵。
    
若要深入了解，請參閱 ＜ [Create Outlook.com 中的資料夾](https://go.microsoft.com/fwlink/p/?linkid=873114)。
