# <a name="intune-app-deployment"></a>Intune 應用程式部署

部署使用 Intune 應用程式是一個三步驟程序：
  
1. **新增應用程式**-您可以新增應用程式市集應用程式或上傳的行的商務應用程式。 
    
2. **指派應用程式**-判斷部署的應用程式的範圍。 
    
3. **監視應用程式**-檢視的應用程式的安裝狀態、 列出所找到的應用程式]，和管理是從應用程式商店購買的應用程式。 
    
閱讀 < <b0>What's Intune 應用程式管理</b0>的逐步解說及設定和保護應用程式的相關資訊。
