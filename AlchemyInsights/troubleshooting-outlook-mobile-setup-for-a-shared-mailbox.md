# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>疑難排解 Outlook 行動裝置設定的共用信箱

您無法透過行動裝置上安裝的 Outlook App 存取共用信箱。 您想要這項功能嗎？ 可讓知道 Outlook 工程團隊。 [在這裡投票](https://go.microsoft.com/fwlink/?linked=862116)！
  
您可以僅使用共用的信箱所開啟的行動裝置瀏覽器登入 Office 365，然後再移網頁型 Outlook。 如需相關指示，請參閱 < <b0>Add 網頁型 Outlook 中的共用信箱</b0>。 從 outlook 網頁版您將能夠存取共用的信箱。
  
了解更多： [outlook for iOS 和 Android 的應用程式內協助](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
