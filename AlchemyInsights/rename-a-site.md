# <a name="rename-a-site"></a>重新命名網站

若要變更網站的名稱, 例如從https://contoso.sharepoint.com/sites/sample1 https://contoso.sharepoint.com/sites/sample2, 您必須建立新的網站, 然後將舊網站的所有內容都移至新網站。 如需在文件庫中移動專案的相關資訊, 請參閱[移動或複製 SharePoint 文件庫中的專案](https://go.microsoft.com/fwlink/?Linkid=2018691)。
  
附注: 如果您嘗試在 SharePoint Online URL 中變更功能變數名稱 (也就是上述 URL 的 contoso 部分), 則這是不可能的。 請參閱[變更您的網域](https://go.microsoft.com/fwlink/?Linkid=2018696)。
