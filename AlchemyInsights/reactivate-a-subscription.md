# <a name="reactivate-a-subscription"></a>重新啟用訂閱

如果您的訂閱到期, 則是由 Microsoft 停用, 或是您在訂閱期限中間取消, 您可以在 Microsoft 365 系統管理中心中重新啟用它。
  
1. 在系統管理中心中，移至 [訂閱][](https://go.microsoft.com/fwlink/p/?linkid=842054) 頁面，或選擇 [計費]**** \> [訂閱]****。

2. 選擇您要重新啟用的訂閱, 然後選擇 [**重新**啟用]。

    如果您沒有看到 [**重新**啟用] 做為可用的動作, 請使用「讓我們呼叫您」選項, 與支援工程師交談以重新啟用您的訂閱。

3. 輸入您的付款詳細資料。 您可以在這裡更新您的現有付款資訊。
