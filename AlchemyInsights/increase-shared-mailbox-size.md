# <a name="increase-the-size-of-a-shared-mailbox"></a>增加為共用信箱的大小

您可以藉由將共用的信箱指派 Exchange Online P2 授權 （也就是包含內 E3 和 E5 授權） 增加 50 GB 的預設大小為 100 GB 共用信箱的大小。 當您將 P2 授權指派給在[作用中使用者](https://portal.office.com/adminportal/home)的共用信箱時，會自動增加大小 15 分鐘內。
