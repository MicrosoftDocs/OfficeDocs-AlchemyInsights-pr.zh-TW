# <a name="cant-select-contacts-when-composing-email"></a>無法在撰寫電子郵件時選取連絡人

當撰寫新電子郵件並按一下 [**到**] 或 [**副本**] 按鈕，某些客戶可能看不到其連絡人清單。若要選取 [名稱]，開始輸入**收**件者或 **[副本]** 欄位中並將顯示在相關的連絡人。
