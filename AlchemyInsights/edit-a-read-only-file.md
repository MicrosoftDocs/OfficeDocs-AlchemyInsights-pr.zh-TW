# <a name="edit-a-read-only-file"></a>編輯唯讀檔案

文件庫中當您在文件的名稱上按一下 [Office 程式可能會意外開啟成唯讀檔案。若要開啟供編輯，按一下 [**開啟**（三個點），與然後按一下 [**編輯]。**
