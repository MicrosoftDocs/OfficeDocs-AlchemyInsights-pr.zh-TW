# <a name="edit-a-read-only-file"></a>編輯唯讀檔案

文件庫中時您可以按一下 [文件的名稱，Office 程式可能會意外開啟成唯讀檔案。 若要開啟它進行編輯，按一下 [**開啟**（三個點），，然後按一下 [**編輯]。**
