# <a name="remove-users-from-a-sharepoint-group"></a>從 SharePoint 群組中移除使用者

SharePoint 群組是一個使用者的集合，其中的每一組許可權都具有相同的網站和內容許可權。 您可以使用群組輕鬆地將相同的許可權等級指派給多位人員，而不是一次指派許可權。

- [從群組中移除使用者](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#remove-users-from-a-group)
- [使用系統管理中心新增或移除 Microsoft 365 群組中的成員](https://docs.microsoft.com/office365/admin/create-groups/add-or-remove-members-from-groups?view=o365-worldwide)
