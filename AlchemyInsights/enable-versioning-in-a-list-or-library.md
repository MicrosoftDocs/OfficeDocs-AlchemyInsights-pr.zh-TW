# <a name="enable-versioning-for-a-list-or-library"></a>啟用清單或文件庫的版本設定

1. 移至清單或文件庫。
    
2. 按一下 [ **設定**]，然後按一下 [文檔 **庫設定** ] 或 [ **清單設定**]。
    
3. 按一下 [ **版本設定**]。
    
4. 選取您想要的設定，然後按一下 **[確定]**。
