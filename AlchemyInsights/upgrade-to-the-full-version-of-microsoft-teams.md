# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>升級至 Microsoft 小組的完整版本

需要從免費升級為 Microsoft 團隊的完整版本嗎？ 請參閱[升級 Microsoft 團隊免費的 Office 365 訂閱版本](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

您必須符合下列需求:

- 您是註冊目前免費團隊版本的人員。
- 如果您有自己的網域, 則它無法與 Azure Active Directory 相關聯。
- 若要在升級時傳輸資料, 您必須在升級程式期間訂閱。 如果您在升級程式之外購買 Office 365, 您就無法傳輸資料。
