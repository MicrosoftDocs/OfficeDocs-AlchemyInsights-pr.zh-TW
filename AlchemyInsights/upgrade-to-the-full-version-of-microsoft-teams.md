# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>升級至 Microsoft 小組的完整版

需要從免費升級為 Microsoft 團隊的完整版嗎？ 請參閱 [升級 Microsoft 團隊免費的 microsoft 365 訂閱版本](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

您必須符合這些需求：

- 您是註冊目前自由版本小組的人員。
- 如果您有自己的網域，它就無法與 Azure Active Directory 相關聯。
- 若要在升級時傳輸資料，您必須在升級程式期間訂閱。 如果您已在升級程式外購買訂閱，您就無法傳輸資料。
