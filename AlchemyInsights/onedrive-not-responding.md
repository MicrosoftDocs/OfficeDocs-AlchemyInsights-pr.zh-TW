# <a name="onedrive-not-responding"></a>OneDrive 無回應

若為新建立的帳戶，使用者必須使用自己的帳戶登入 Microsoft 365 系統管理中心，然後按一下 [OneDrive] 以手動啟動 OneDrive 網站的提供。
  
不過，如果使用者已按一下 [OneDrive] 而且其個人網站未布建在24小時內，請記錄支援票證以取得進一步協助。
