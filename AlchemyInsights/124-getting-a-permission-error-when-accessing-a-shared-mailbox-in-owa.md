# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>在網路上的 Outlook 中開啟共用的信箱時取得權限錯誤？

如果您收到錯誤**發生錯誤。您不需要開啟此信箱的權限。** 當您嘗試在網路上的 Outlook 中開啟共用的信箱，您的帳戶可能沒有存取共用的信箱的權限。貴組織系統管理員可以指定帳戶權限的共用信箱[建立共用信箱](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9)的主題中的指示。
