# <a name="site-and-content-structure-removed"></a>網站與內容中移除的結構

從 2018 年 10 月，網站內容與結構] 頁面上將不再支援或接收功能更新。 SiteManager.aspx] 頁面上的 [設定] 連結將會移除從 SharePoint Online，直接存取才會限制在網站集合系統管理員年 3 月 2019，此時 [網站內容與結構] 頁面上會完全停用。 

附註： 因應措施完全淘汰的功能，直到網站集合管理員可以瀏覽至 Microsoft 365 系統管理中心、 選取 [SharePoint] 磚，並附加或編輯 URL 取代 '/ Sharepoint.aspx' 與' / Sitemanager.aspx'。 


使用檔案移動和副本來管理網站內容與結構。 如需詳細資訊，請參閱[複製檔案和資料夾之間 OneDrive 和 SharePoint 網站](https://support.office.com/article/copy-files-and-folders-between-onedrive-and-sharepoint-sites-67a6323e-7fd4-4254-99a8-35613492a82f)。 

如需詳細資訊，請參閱[使用網站內容與結構](https://support.office.com/article/Work-with-site-content-and-structure-30fcaad9-02b1-4347-8b03-e1ccc5a4c19f)。
