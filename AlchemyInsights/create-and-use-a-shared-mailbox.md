# <a name="create-and-use-a-shared-mailbox"></a>建立和使用共用信箱

共用信箱可讓一個以上的人傳送和接收電子郵件的別名 (如 sales@contoso.com), 而不需要指派其他授權。
  
- [請移至這裡開始](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

您也可以閱讀 [[建立共用信箱](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx)] 文章, 以告訴您需要瞭解的所有共用信箱的相關資訊, 包括如何加以使用。
