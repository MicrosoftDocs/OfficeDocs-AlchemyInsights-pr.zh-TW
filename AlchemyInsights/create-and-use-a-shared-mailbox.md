# <a name="create-and-use-a-shared-mailbox"></a>建立及使用共用的信箱

共用的信箱可讓多人傳送及接收電子郵件別名，類似 sales@contoso.com，而不指定其他的授權。
  
- [以下移至 [快速入門](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)
    
您也可以閱讀[建立共用信箱](https://support.office.com/article/https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx)文章會告訴您必須了解共用信箱，包括使用方式的每個項目。
