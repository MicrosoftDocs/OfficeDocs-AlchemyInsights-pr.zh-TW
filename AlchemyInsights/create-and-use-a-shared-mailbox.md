# <a name="create-and-use-a-shared-mailbox"></a>建立並使用共用的信箱

共用的信箱可讓多人傳送及接收電子郵件別名，例如 sales@contoso.com，卻沒有指定另一個授權。
  
- [請移至這裡開始](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

您也可以閱讀[建立共用信箱](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx)本文將告訴您必須了解共用信箱，包括如何使用它們的所有項目。
